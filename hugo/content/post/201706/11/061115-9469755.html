C 羅和梅西這一年：欲戴王冠，必承其重

西諺有云：欲戴王冠，必承其重。

足球世界裏，沒有天生王者，只有不斷鑄就功業、逐漸自我證明的勇士王者。

過去這個賽季，C 羅奉獻了卡卡之後——歐冠最具決定性和說服力的個人表現，並幾乎預定了 2017 年度金球獎。既去年歐洲盃奪魁後，再度征服世人，他已戴上“歐洲之王”的冠冕。

然而這是梅西失意的賽季，儘管和隊友締造了諾坎普奇蹟，但在 C 羅雙冠王和歐冠衛冕的映照下，區區一座國王杯顯得暗淡失色。

當我們回溯過往，可以發現，有四大因素左右着梅西和 C 羅之間的競爭抗衡。

（1） 體系支撐的強與弱

隨着現代足球的技戰術演進，陣容架構和戰術體系對於前場球星的支撐烘托效應愈發突出，使得頂級的鋒線射手在進攻端的效率輸出大幅提升。

梅西和 C 羅強絕天下的個人能力，配合所在豪門深厚的陣容配置與強勁的體系支撐，順應並引領了這個綿延十年的絕代雙驕時代！

期間，他們取得的成就也深合體系支撐的強弱運勢演變。

2007-08 賽季，C 羅和曼聯君臨歐洲，先拔頭籌。隨後瓜迪奧拉橫空出世，巴薩“夢三王朝”的極致傳控和高壓反搶戰術將梅西推爲足壇第一人，C 羅和皇馬被雙重壓制。

然而巴薩盛極而衰，由於陣容老化和引援不力，中場三角控制力銳減，整體跑動和中前場壓迫反搶能力退化，高壓穩控所能帶來的拱衛後場效應消減，繼而進一步加劇了中後場防守失穩的問題，直接導致球隊難以持續、高效地支撐戰術體系合理運轉。

  • 如今的克羅斯和莫德里奇，更加凸顯巔峯期哈白組合對於梅西和巴薩的重要性

梅西在鋒線獲得的進攻支持下降，當遭受多人聯防時，沒有過去隊友對他的有效分擔支援（需要多名隊友具有很強的無球跑動和有球處理能力牽扯防線），在打不開局面或攻勢不暢時更多需要他來後撤拿球創造新局面。

期間，C 羅力助皇馬拿到十冠王，自身也在與梅西的不利競爭中扳回一城。

夢三解體，比拉諾瓦和馬蒂諾先後接手球隊，然而都未能改善根植於中場的體系問題。而之後路易斯 - 恩裏克的三冠王賽季，恰是就此問題給出了切實可行的方案：

那便是通過引進蘇亞雷斯，成功組建 MSN 組合，依靠鋒線三人組出色的個人能力和團隊意識，回補中場，建立起中前場強大的技術優勢和傳跑配合優勢，繼而捏合成型新一代的巴薩體系，成就新輝煌。

  • 三冠王賽季 MSN 的輝煌掩蓋了巴薩嚴峻的體系問題

然而隨着哈維離去、小白老化，巴薩中場體系進一步惡化，繼而牽動全身。

本賽季在失去阿爾維斯後，巴薩右路持球推進支援太過有限（主要表現爲右後衛和右中場），梅西不得不過度頻繁地回到深位參與拿球組織和向前推進。

他在此過程耗費太多體能，到了比賽後程需要他叩關斬將時，已然力不從心。

在此期間，卡瓦哈爾於 2013 年夏天迴歸，這一步引援和皇馬揚帆起航的時間點高度吻合。

戰術層面上的意義在於——終結了過往阿韋羅亞主力時期，皇馬後場出球和進攻選擇過分倚仗左側馬塞洛的歷史，大幅改善邊中配合效果，真正實現了兩翼齊飛。

齊達內治下這兩年，克羅斯和莫德里奇的中場雙核地位穩若磐石，卡塞米羅深受信任成爲主力後腰，防守穩健之餘，傳控能力也有明顯進步。

伊斯科在貝爾傷缺期間強勢上位，爲前期快速反擊的衝擊型打法，向強調邊中結合、滲透進攻的傳控打法的過渡轉變，做出了突出貢獻。

他們都在皇馬本賽季的雙冠之路上立下赫赫戰功，再加上 J 羅、科瓦契奇、阿森西奧等人，強大的中後場和陣容儲備爲 C 羅提供了強力的體系支撐。

皇馬的中場架構逐漸增進攻守平衡和控制力，而巴薩的中場卻是逐漸老邁失衡，以致成爲戰術沉痾。皇馬以邊衛爲基點的邊路攻防體系得到完美的升級優化，而巴薩的邊路攻防體系卻是嚴重衰化，不得不依靠雙側邊鋒回補深位。

時移世易，球星的個人輝煌，既離不開個人的不懈奮鬥，也不能忽視歷史的進程。此消彼長之下，戰術體系對於梅羅二人的支撐烘托效應徹底扭轉——C 羅愈強，而梅西分外掙扎。

（2） 個人轉型的得與失

在曼聯成熟期以及初來皇馬時，C 羅強大的身體、速度、折返能力、無球跑位能力，以及依然出色而又化繁爲簡的盤帶突破能力，使他成爲一個兼具突破手和終結者特質的超級球星，比賽影響力的一大體現在於前場得球便是威脅。

但之後隨着年齡增大，身體柔韌性和靈活性不可避免的下降，再加上力量訓練的增強，這些變得更爲顯著。

C 羅的二次轉型順應了身體機能的變化趨勢和自身競技層面的訴求（追趕梅西並減少傷病），從一個基於控球持球推進再轉化爲進攻效率輸出的前場自由人，轉化爲一個減少持球突破但又相對依賴球權和空間，有着更高效進攻輸出的無球空切型內鋒終結者。

最近兩個賽季，C 羅的轉型進一步取得了成功，其關鍵在於——戰術角色更加自由，跑位更靈活也更趨近中路和禁區，並降低了對於球權和空間的依賴！

卡瓦哈爾的存在，大幅優化了兩翼攻防體系和邊中配合效率，在經過克羅斯和莫德里奇的傳控調度轉移後，持球側所面臨的拿球壓力得到分化消減，從而更好地釋放馬塞洛在進攻端的威脅。

  • 卡瓦哈爾助攻 C 羅攻入歐冠決賽首開記錄的進球

如此完備的中後場體系，再加上馬塞洛這一強點，使得通常在同側活動的 C 羅可以更輕鬆地參與進攻，而不必花費過多體能去參與深位拿球和前推進攻，很好地規避了這次轉型中“失”的一面——C 羅在這些環節的決定性是有所下滑的。

更專注於最後一擊的 C 羅，不亞於足壇核武器。

他左中右橫向遊弋的活動範圍更大，觸球拿球和完成最後一擊的區域更靠近中路和禁區，處理球更加簡練高效。

本賽季 12 粒歐冠進球中，3 粒爲一次調整射門，9 粒爲直接射門。他依然保持着強勁的射門慾望、跑空當意識和反覆衝擊能力，在合理的輪換之後，於強強對話的硬仗表現更加出彩。

  • 歐冠決賽第 3 球，C 羅的跑位意識和搶點意識體現的淋漓盡致

對比 C 羅成功的轉型，梅西的轉型卻是喜憂參半。

梅西職業生涯至今，經歷過兩次關鍵轉型：一次是瓜迪奧拉執教伊始，將之從右側內鋒改造爲中路“僞九號”；第二次便是 2014 年起，他逐漸增多後撤組織，從前場自由人發展爲中前場組織進攻雙料核心。

這般轉型，助梅西和巴薩一同獲得 2014-15 賽季三冠王，也讓他連續獲得了 2015、16 年的 IFFHS 年度最佳組織中場，受到資深評委的廣泛認可。

鋒線 MSN 三人組中，梅西擁有最好的傳球能力、分球視野和比賽閱讀能力，對於巴薩中場的回補支撐貢獻便更爲突出。

可以說，梅西的能力意識是轉型基礎，身體機能下滑而尋求改變是自身需求，戰術環境變化則是核心的外部促因，這些共同造就瞭如今更成熟的梅西。

  • 2016-17 賽季國家德比，梅西多次回撤到中圈附近拿球組織

如果梅西轉型得更加成熟穩定些，對他的讚譽和認可會更加廣泛，因爲兼具得分手、突破手和中前場組織核心特質的球星（尤其是組織特質），往往比相對較固定的鋒線球員更受認可，典型的比如克魯伊夫之於蓋德 - 穆勒。

這是轉型“得”的一面。

但三冠王賽季的成功是幸運而又難以複製的。走到本賽季，巴薩混亂的中後場愈發低迷，嚴重牽絆了梅西，放大了這次轉型“失”的一面。

伊涅斯塔老化嚴重，布斯克茨狀態不佳，拉基蒂奇跑動能力下滑，戈麥斯遲遲難以融入，其餘諸人也難挑大樑。

再加上兩側邊衛的弱勢表現（阿爾巴狀態也低迷），梅西和內馬爾不得不頻繁地回撤深位拿球，在長距離的推進和反覆拉鋸間，增加消耗，苦苦提升進攻效率和創造新局面。同時，與禁區距離增大，也增加了梅西完成最後一擊的難度。

  • 梅西從深位拿球，奔襲數十米後送出妙傳

如果說 C 羅的轉型是趨向“一專多能”、專注最後一擊的話，那麼梅西的轉型就是要“多專多能”、既當爹又當媽。這對於身體機能已過黃金階段的梅西來說，負荷太大，責任過重了。

然而，梅西是非凡的。即便如此，他依然在本賽季奉獻了出色的傳射效率，52 場 54 球 19 次助攻。只是到了歐冠淘汰賽，轉型的得失對比放大，C 羅依然能夠維持高效率，而梅西在淘汰賽階段的效率卻是下滑的。

  • 二者賽季傳射效率對比，梅西收穫歐洲金靴

（3） 團隊榮譽的成與敗

陣容儲備、戰術體系的支撐，以及個人轉型的得失，直接影響到雙方團隊榮譽的競逐。

截止當前，梅西和 C 羅的足壇歷史地位，已然足以比肩貝肯鮑爾、克魯伊夫和迪 - 斯蒂法諾。在二者綜合能力、個人榮譽相近的情況下，他們未來的成就高低更多取決於最高團隊榮譽的比拼和自身的核心作用。

  • 國內盃賽特指西班牙國王杯和英格蘭足總盃、聯賽盃；2008/09 賽季，兩人分別獲得聯賽冠軍和國內盃賽冠軍

如上所示，最近十年，梅羅二人的金球獎評選與團隊榮譽呈現高度相關性，其中歐冠成績的影響比重尤爲突出——此前的 6 次歐冠掄元爲他們各自收穫 3 座金球獎。

由此推定，當前手握雙冠王的 C 羅，幾乎已經預定本年度的金球獎。如果成行，那麼 C 羅也將追平梅西的金球獎數量，在個人榮譽上全然不落下風。

如果再對比梅西在國家隊的失意歷程，C 羅 2016 年獲得的歐洲盃冠軍可謂意義重大。再加上最近 4 年 3 座歐冠獎盃，歐冠冠軍數量也追平梅西，那麼在團隊榮譽上，C 羅甚至可以說略勝一線。

  • 個人榮譽、紀錄對比，梅西將於 6 月 24 日年滿 30 週歲

只是當前來說，一切都還太早。梅西畢竟年輕 2 歲，巴薩終究是巴薩，他們隨時都可能王者歸來。不要忘記，巴薩依然是皇馬最強勁的對手，梅西依然是國家德比進球紀錄保持者！

（4） 核心作用的進與退

三月中旬，巴薩在歐冠 1/8 決賽次回合驚天逆轉巴黎聖日耳曼，梅西和內馬爾是那場奇蹟之戰的主要締造者。

梅西登上球迷看臺高呼的瞬間場景，被巴薩俱樂部安放在了諾坎普的廣場之上，用以紀念這場偉大逆轉，並附言：歡迎來到諾坎普，奇蹟在這裏誕生。

但很遺憾——這是梅西本賽季最具核心作用的表現，卻也是整個歐冠賽季的最後一次展現。

如下所示，梅西個人的競技巔峯是 2011-12 賽季，不僅上演個人最高的歐冠淘汰賽傳射效率（6 場 8 球 3 助攻），還創造了賽季 73 球的紀錄，其後製造年度 91 球的神蹟。

但之後幾個賽季，梅西在歐冠淘汰賽的核心作用卻是呈現下行趨勢。最近 5 個賽季的歐冠淘汰賽，他的進球數最多隻有 3 個（本賽季 1 球），其中 4 個賽季未能奉獻一次助攻！同時連續兩年隨隊止步歐冠 8 強！

反觀 C 羅，效率卻穩定得可怕。從加盟皇馬的第 2 個賽季開始，他已率領白衣騎士連續 7 個賽季至少闖入歐冠半決賽，並且連續 6 個賽季在淘汰賽階段場均至少製造 1 球。

最驚人的是本賽季！C 羅在小組賽階段僅僅攻入 2 球，但邁入八強後，5 戰 10 球，接連斬落拜仁、馬競、尤文三大豪門！一舉攬下金靴獎！

強強對話的決定性和領袖能力，震懾無數球迷！

  • C 羅進入歐冠八強後，5 場比賽打進 10 球

二者之間，體系支撐的效應和賽場上的決定性，都是此消彼長。而 C 羅在 30 歲之後依然取得如此輝煌的成就，更加令人歎服。

【結語】

正如“光榮屬於希臘，偉大屬於羅馬”，我們對於事物的評價不應限於非黑即白的二元論。梅西和 C 羅很好地詮釋了偉大的不同維度。

當我們見證他們——從少俠成長爲球星，從強者成長爲王者，從王者再度邁進之時，我們自身也在成長。

而當你被他們的偉大人生征服而拋去立場偏見，由衷地欣賞他們非凡的技藝時，或許能更好地感受到足球的豐美與壯闊！

C 羅這兩年的表現，爲他贏得無數尊重和敬佩，也打消了無數質疑與偏見。他慷慨激昂的鬥志、自礪拼搏的進取心、永不止步的職業精神，更是令其成爲了時代偶像！

梅西同樣是偉大的。可惜偏內向的性格阻礙他成爲更好的領袖，而難以在最需要他時展現出捨我其誰的王者之風。三十而立，等待梅西的——不是超越 C 羅，而是超越自我！

欲戴王冠，必承其重！前方還有兩尊豐碑巨擘！未來還有更高的王者冠冕！

“我會試試看，梅西還是一如既往地在戰鬥，當然我也是一樣，和他同時代是我的幸運也是動力。”

是啊！我們球迷也是如此幸運！
