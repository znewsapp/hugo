1. 心態

作爲業餘愛好，最主要的是什麼呢？最主要的當然是開心了。

但是木工比較特殊，花錢花時間，需要不小的場地，外加會產生大量的粉塵和噪音，所以還是建議在心態上做好準備，避免真的搞起來之後才發現原來這玩意兒這麼麻煩，結果葉公好龍，不了了之。

木工其實是個很寬泛的概念，種類有很多種，既可以是做傢俱，也可以是車珠子，大到木頭房子，小到廚房菜板，這些都是木工。木工的技術也有很多種，中國、日本的傳統手藝，歐美的傳統手藝，現代的電動工具，以及數控加工，這些都是木工。自己喜歡哪種，有沒有合適的條件，這些都需要考慮清楚。

另外，家人是不是支持，能不能忍受粉塵和噪音，能不能容許買工具買木料買耗材的花銷，這些也都需要考慮清楚。

還有，自己有沒有額外的計劃，比如將來想不想靠業餘木工掙錢，這些雖然不一定要現在考慮清楚，但也是一個需要問自己的問題。

2. 場地

做好了心態的準備，接下來就是場地了。作爲剛入門的業餘愛好者，顯然很少會有人去租用專門的場地或者車間，所以更多的業餘木匠都是在自己家裏做木工。當然，粉塵和噪音污染會不會影響到家人和鄰居，這是必須要考慮的問題。

在美帝，一般最常見的就是車庫，比如最普通的兩車位車庫剛好可以作爲一個合適的木工小作坊。但是車庫當了木工作坊，壞處就是車只能永遠停在外面了。社區允許不允許，以及自己老婆或者老公願意不願意，這都是需要考慮的問題。當然，如果房子夠大，有三車位車庫，或者有兩個獨立的車庫，那麼一般問題就不大了。

比如下面就是一個典型的剛開張的兩車位車庫的木工作坊。

缺點就是車庫不能停車了，車只能停在外邊風吹日曬了，比如下面這樣。

另一個選擇就是地下室，主要見於比較老的房子，比如六七十年代建的，大平層 ranch 或者 split-level 的那種房子，地下室沒有裝修，是一個開敞的 unfinished 的空間，裏面也沒有牆體，上面的房子多大，這個地下室就有多大，只是有一些鋼柱子和一些熱水器、空調主機之類的大型設備。這個也是個不錯的選擇，缺點主要就是可能搬運比較大的木料有難度，以及部分地下室的層高可能過低。

還有一種就是在後院徹底修一個獨立的木工工作室，一般適用於土地 lot 比較大的房子，比如土地產權至少 0.5 英畝以上的房子。獨立工棚的好處是專爲木工打造，隔音、粉塵處理這些問題一開始就能安裝好相應的專業設備，也不會吵到家人或者鄰居，但是缺點就是比較貴，第一要有這麼大的地盤，第二要花錢建這麼一個工作室。當然，另一個優點就是建這個木工房本身就是做木工。

比如下面這個就是建造中的後院獨立木工房。

另外，如果受限於場地限制，比如離鄰居比較近，或者後院太小等等，而不得不退而求其次，但是又擔心噪音的話，可以購買隔音掛毯掛在牆上，再給車庫門內側貼上額外的保溫層，然後幹活的時候關上車庫門，一般可以比較好的解決噪音問題。

開工之前，或者買了新工具之後，建議試用一下工具，同時讓家人或者朋友站在你的木工作坊外邊不同的距離，用專用儀器或者智能手機 app 測量一下噪音水平，避免擾民。

3.2 噪音防護

要認準隔音耳罩的標準，對照自己電動工具的噪音水平，購買具有足夠保護能力的耳罩。比如下面這種耳罩的 NRR 也就是 Noise Reduction Rating 是 30 分貝，換算一下也就是可以減少環境噪音 11.5 分貝。換言之，如果電鋸的噪音是 100 分貝，那麼帶上耳罩之後，你聽到的噪音就是 88.5 分貝。可以根據你所使用的不同工具，來選購不同 NRR 的隔音耳罩。

3.3 粉塵和有毒氣體防護

操作電鋸、打磨、開槽等等的時候，防塵口罩是一定要戴的。因爲祖國的甜美空氣，所以大家對各種防塵口罩已經不陌生了，這裏就不介紹了。3M 的那種就可以。

然後是化學保護，主要是刷漆的時候，部分清漆是有劇毒的，所以必須戴防毒面具，就是下面這種。

如果真的喜歡做木工，工作時間很長的話，可以考慮像下面這樣安裝全套的粉塵收集系統，這樣每個工具產生的粉塵都會自動吸走，降低了健康隱患。注意，這個並不代表就可以不戴口罩了。

如果剛開始起步，工具比較少，可以考慮臨時的軟管粉塵回收加一個小功率的專用抽風機。

有條件的話，建議安裝一個專用的空氣淨化器。不要直接用家用的空氣淨化器，因爲家用的那種應付不了鋸末和木屑。有了專門的淨化器，這樣工作的時候以及停工之後空氣中的粉塵很快就會被吸濾，避免粉塵一直飄蕩在空氣中，或者最終落到地板和工具上，變成污染源。

最最基本的，剛起步階段，如果上面的粉塵回收和空氣淨化裝置都沒有，至少至少要有一個吸塵器。不是那種家用的，而是工業用乾溼吸塵器，可以吸鋸末和粉塵的。大多數電動工具都有粉塵口，可以直接把吸塵器的軟管插到工具的粉塵口上。

4. 工具

我們前面說過，木工的種類有很多種，蓋房子、打傢俱、做玩具、車珠子，這都是木工，各自需要不同的工具。這裏我介紹一下基本的工具，掛一漏萬，只起一個簡單的介紹作用。取向主要是傢俱和家庭裝修，流派是電動流。另外，有些工具國內並不多見，沒有統一的中文名稱，這裏以英文爲準。

另外，工具的品牌和價位也是多種多樣，常見的品牌裏從高到低，有 Festool 這種木工界的勞斯萊斯，有博世（Bosch）德偉（DEWALT）牧田（Makita）日立（Hitachi）這些中等輕奢專業品牌，有良明（Ryobi）這種針對業餘愛好者的平價品牌，還有 BLACK+DECKER 這種廉價超市品牌。

比如說同樣都是 miter saw，Festool 和 Ryobi 的差價竟然如此之大。真是木工界也有真土豪！

總的來說，一分錢一分貨，但也沒有必要貪大求全，貴的好的並不一定適合自己，尤其是對於業餘愛好者來說，滿足自己需求的工具就是最好的工具。

4.1 電鋸類

4.1.1 Table saw

家庭木工最最常用的電鋸應該是 table saw，很多人認爲 table saw 是一個業餘木工作坊的核心。table saw 非常實用，可以完成各種木工操作，熟練的話，木板開槽、榫卯結構的榫頭等等都可以用它完成。如果沒有其它專業設備，還可以用它來拋光木料側邊。不過需要注意的是，table saw 也是一種非常危險的工具，非常非常危險，因爲它是圓形旋轉鋸裏唯一一個鋸片向上的，使用的時候一定要注意安全。根據不同的操作，必要的防護罩和防護措施一定要有。市面上有很多專用的夾具，比如 miter gauge，可以用這些夾具來推動木料，這樣自己的手指可以遠離飛速旋轉的鋸片。另外，高端的 table saw 有自動安全功能，一旦檢測到鋸片接觸到了比較軟的東西，立刻自動停止，並且鋸片立刻自動收縮，保護使用者的手指。如果不放心，可以拿火腿腸試驗一下。

比如下面就是一臺專業級別的 table saw，售價大約 2000 美元。

如果預算或者場地有限，可移動的 table saw 是不錯的選擇，這也是大多數業餘木工的選擇。自己打造一個工作臺，然後把這種小型 table saw 安裝進去，土法煉鋼，差不多就可以獲得專業級別的性能。比如這種博世的四五百到六七百美金不等。其它平價品牌比如 Ryobi 的這種 table saw，甚至只需要一兩百美金。

table saw 非常的多才多藝，分割大面積的木板是它最常見的用途，此外還能用它幹很多別的事情，甚至可以部分代替下面要說的 router 和 jointer，比如下面就是用它來開槽。

4.1.2 Circular saw

如果你覺得 table saw 太貴或者太佔地方或者太危險，那麼可以用 circular saw 代替。這倆其實是一對，table saw 是電鋸固定，手推動木料運動，而 circular saw 反過來，是木料固定，手推動電鋸運動。

circular saw 的價錢一般幾十元到一兩百美元不等。因爲 circular saw 可以移動，低功率的還可以是無電線的電池版本，所以使用起來非常便捷。對於建房子的木工來說，circular saw 是最核心的工具。比如下圖就是美帝大量的木結構房屋工地上常見的鋸木料操作。

比如這就是用 circular saw 鋸木版，木板不動，電鋸走直線。

很多廠家還有各種配套的導軌和夾具配合 circular saw，來獲得更好的效果，這種也叫 track saw。比如下面就是不同品牌的 track saw，這些一般要四五百美元以上。

4.1.3 Miter saw

另一種常用的電鋸是 miter saw，也叫 chop saw，就是有點像鍘刀的那種。價位一般在兩百到五百美元左右。這種電鋸特別適合切割長條狀的木料，非常非常方便。

絕大多數的 miter saw 可以左右旋轉，有些鋸片也可以左右傾斜，這樣可以做不同切面角度的切割。帶有這種功能的一般叫 compound miter saw。把木條 45 度切開，四根一拼，很容易就能做成一個畫框或者鏡框。

還有一些 miter saw 可以前後伸縮，這樣大大加大了它能切割的木料寬度，這種一般叫 sliding miter saw。大多數高端 miter saw，都是 sliding compound miter saw，兩種功能都有。sliding miter saw 一般都有明顯的前後伸縮用的導軌，比如下圖這種，後面有兩根平行的圓管鋼導軌。

4.1.4 Vertical band saw

Band saw 有兩種，一種是 vertical 的，就是鋸條豎直的，另一種是 horizontal 的，就是鋸條水平的。水平的一般常見於金屬加工，木工領域用的一般是豎直的。比如下面就是專業級別的 vertical band saw。

前面的 table saw、circular saw、miter saw 都只能切直線，而用 band saw 可以切出各種不規則圖案。

另外，band saw 做榫卯節點也非常方便。

像 Ryobi 這樣的平價品牌還特別針對業餘愛好者推出了袖珍 band saw，個頭很小，雖然性能不強，但一般業餘愛好者應該也足夠了，售價是一百多美元。

4.1.5 Jig saw

jig saw 和 band saw 也是一對兒。band saw 是鋸固定，木料動，而 jig saw 是木料固定，手拿着鋸動。同樣，jig saw 也能切各種不規則圖案。

另外，jig saw 的另一個好處是它可以在不損壞木板其它部分的前提下在木板中間切一個不規則形狀的洞，而 band saw 就做不到。先用電鑽在需要鋸掉的部分鑽一個洞，然後把 jig saw 的鋸條伸進去，然後按照預先畫好的圖案走位就可以了。

比如這個自制的小裝置，調節 jig saw 螺桿的位置，就可以像圓規一樣在木板上切出不同直徑的大圓。

4.1.6 Reciprocating saw

這種鋸一般主要用於非常粗糙的切割，以及破壞式拆除舊有的櫥櫃、裝修等等。另外，它也可以用來切塑料管、金屬管這些東西。

4.2 刨平打磨類

4.2.1 Planer

除了鋸之外，木工另一個標誌性的工具應該就是刨子了，而 planer 其實就是一個強大的電動刨子。設置好你想要的厚度，把木料從這一頭推進去，那一頭出來的木料已經被刨到這個設定好的厚度了。

上面這種一般就是業餘愛好者使用的小型 planer，大型的專業 planer 可以處理非常大尺寸的木料。

4.2.2 Jointer

上面說的 planer 是把木板的上下兩個表面刨平，而 jointer 則可以把左右兩個側面刨平。

4.2.3 Belt sander

Belt sander 主要用於木料的粗糙打磨，或者讓木板局部打磨變薄。構造就像一個小坦克，履帶就是砂紙做成的。把它放在地上，打開開關的同時鬆開手，它真的會像小坦克一樣在地上跑…… 玩壞了別說是我教你的……

4.2.4 Orbit sander

Orbit sander 也叫 disc sander，它的砂紙是貼在下面的一個圓形。它在打磨的時候並不是朝一個方向旋轉，而是無規則的來回轉，這樣避免了朝一個方向的規則性花紋。所以它主要用於傢俱完成之後上漆之前的最後精細打磨，以及第一遍上漆和第二遍上漆以及此後的上漆之間的打磨。

4.2.5 Detail sander

這個主要用於邊邊角角的打磨，比如櫥櫃內部 orbit sander 夠不到的地方，或者因爲 orbit sander 直徑太大所遺留下的未打磨到的角落。它同樣也是無規則的來回動。另外，一些品牌的 multi-tool head 電動工具套頭也能起到類似的作用，可以替代 detailed sander。

4.3 開槽邊角類

4.3.1 Router

Router 是非常好用的木工工具，同樣也是多才多藝。可以用它做邊角的修角或者各種曲線藝術效果，可以用它開槽，甚至可以用它打眼。router 這個詞還有上網的那個路由器的意思，所以不要搞混了…… router 的構造也很簡單，就是一個電機，連接着下面一個刀頭，不同刀頭的形狀、大小不同，可以獲得不同的效果。

router 主要有兩種，一種是 fix based 的，就是刀頭的高度設置好之後，在運行過程中不能調節，只能切固定深度，另一種是 plunge，刀頭的高度在運行的過程中也可以上下移動。有些品牌也有二合一的 router，比如下面這款博世，電動機和刀頭的核心部分可以拿出來，安到兩個不同的底座上，左邊那個是 fix based，右邊這個帶上下伸縮的導軌的是 plunge。

比如可以用 router 開槽，然後把這個開槽後的木板豎起來，當作書櫥的側壁，書櫥的橫板就可以插進這個槽裏。

比如 router 可以裝上不同的邊角修飾的刀頭，做出不同的邊角效果，這個成果大家肯定都見過，櫥櫃、踢腳線、天花板裝飾邊角，都是如此。

比如 router 還可以用來做木牌標識，在木板中間切出雕刻的感覺。既可以藉助於現成的模板，也可以自己在木板上寫好，然後用 router 雕刻。因爲是在木板中間，所以必須是可以上下伸縮的 plunge router。對於沒有數控機牀的業餘愛好者，這種土法煉鋼的效果其實非常不錯。

4.3.2 Router table

router table 跟 router 的區別就像是我們前面說的 table saw 和 circular saw。router table 其實就是把一個 router 倒過來，固定在桌面下面，然後桌面上一個洞，把 router 的刀頭露出來。比如下面就是用 router table 在木板中間開一個倒 T 形的槽。router 的電機藏在桌面下面，只有刀頭倒着伸出來。這種異形的溝槽，是其它電動工具和手動工具很難做到的。

4.4 電鑽類

4.4.1 Power drill

這個就是我們俗稱的電鑽，不僅做木工需要，各行各業都需要。手持的電鑽一般都是 keyless chunk，也就是說更換鑽頭是不需要額外的工具的，徒手就可以做到。一般電鑽都是可以適配不同直徑的鑽頭。

4.4.2 Drill press

這個跟手持電鑽類似，只不過是固定的鑽牀。所以功率更大，扭矩更大，所能使用的鑽頭直徑也就更大。一般這種的 chunk 都是需要特殊的 key 來擰緊或者鬆開，從而更換鑽頭。drill press 的缺點當然是不能便攜，而且需要把木料固定到上面，如果木料過大，或者已經安裝成了傢俱，那麼就比較不方便。一般針對業餘愛好者的小型鑽牀大約一百多美元。

4.4.3 Drill bits

對於木工的電鑽和鑽牀來說，主要是有幾種特殊的鑽頭。

一種是這種 brad-point 鑽頭，跟普通鑽頭比，這種鑽頭頂端有特製的額外的尖，是專門針對木工的。如果用普通的鑽頭，在剛剛開始接觸木材表面的時候，很容易滑動，導致精度不好控制。而這種鑽頭頂端的尖頭可以戳進木材裏，解決了一開始接觸的時候來回滑動的問題。

一種是 countersink 鑽頭，它可以直接把螺釘的頭部佔據的那個空間也給鑽出來，這樣傢俱表面就是平的，不會有螺釘頭的突起。如果不用 countersink 鑽頭，拿着螺釘直接硬鑽也可以，但木材容易開裂。

一種是 spade bit，可以在木料上面鑽大直徑的圓形空洞。

還有一種是 hole saw 鑽頭，可以在木料上開更大的巨型圓形孔洞。

另一種是 Forstner bit。我們上面說的 spade bit 和 hole saw 只能切貫通的圓孔，如果我們需要一個不貫通的圓孔，比如凹進去的一個圓形的坑，這就需要 Forstner bit。因爲 hole saw 只是邊緣在切割，最後會切出來一個圓木塊，而 Forstner bit 是整個把木頭攪成碎屑，所以可以在任意深度隨時停止，留下一個坑。因爲 Forstner bit 的切割面很大，扭矩很大，所以不太適用於手持電鑽，安全起見，一般只用在固定式的 drill press 上。

4.4.4 Impact drive

這個看上去跟手持電鑽類似，但區別在於，電鑽可以容納不同直徑的各種各樣的鑽頭，而 impact drive 只能容納一種規格的六邊形的鑽頭，主要用於擰螺絲。由於它內部的衝擊機制可以提供更大的瞬間扭矩，所以它擰螺絲比電鑽更強勁，可以把更大直徑的螺釘擰進更硬的木材裏。

配合着電鑽和 impact driver，裝配速度可以大幅提升。比如先用安裝了 countersink 鑽頭的電鑽鑽一個孔，然後立即用 impact driver 往這個孔裏擰一個螺絲，裝配就完成了，方便快捷。

4.5 其它

4.5.1 捲尺

木工界的名言之一，就是 Measure twice, cut once。一個好的捲尺是非常必要的。小小的捲尺其實也有不少學問。

比如捲尺頭上鉤子上的這個洞，剛好可以鉤在一根釘子上。再比如這個黑色的鉤片，其實是可以沿尺子方向前後滑動的，而能滑動的距離，則剛好等於這個鉤片自身的厚度，這樣保證勾着邊沿拉伸測量和抵着邊沿壓縮測量，得到的結果相近。

再比如美國常見的木結構房屋，木格柵牆的格柵與格柵之間，也就是一根根豎向的木料之間，中心間距一般是 16 英寸，所以尺子上 16 英寸的整倍數都用紅色標識。

4.5.2 各種曲尺

比如這種 speed square，可以很方便的放樣各種斜率和角度，也可以當量角器用，還可以當 circular saw 或者 jig saw 的軌道。

再比如這種 combination square，配合不同的配件，可以完成各種放樣。

4.5.3 各種夾具

由於膠水的廣泛使用，各種夾具也必然跟着不可或缺。比如說這種 pipe clamp，可以把好幾塊窄木板粘合成一整塊寬木板。大餐桌的桌面就是這麼做出來的。

比如這種直角夾具，保證做出來的櫃子是直角。

再比如 corner band clamp，可以很方便的粘合櫃子、箱子、鏡框等等。

配合不同角度的配件，還可以做不同角度的粘合。

4.5.4 錘子、鉗子、螺絲刀、木工鑿子等

這些很常見的工具就不再贅述了，需要說的是木工還有一種特殊的錘子，carpenter's mallet，一般是木頭做成的，用來敲擊鑿子的。

5. 木料的準備

作爲一個業餘木工，首先應該知道的是木料是一種天然材料，每一塊都不盡相同。而且更重要的是，木材是各項異性的材料，也就是說，因爲木材其實是由木質纖維組成的，可以想象成一捆稻草或者一捆飲料吸管，不同方向的受力性能是不同的。比如一捆吸管，沿着吸管的方向很難拉斷，但是垂直於吸管的方向一拉，這一捆吸管就散架了。理解了這一點，是理解很多木材節點設計的關鍵。

木材的兩大類別是 soft wood 和 hard wood，也就是軟木和硬木。這兩者的分類與實際的物理性能無關，只跟樹種有關係。針葉林的木材就是軟木，闊葉林的木材就是硬木。一般來說，硬木更加緻密，相對難以加工，一般用於傢俱，軟木比較輕，產量很大，一般用於木結構建築。

對於進階的業餘木工，如果對於木材有特殊需求，可以直接去當地的木材供應商。對於入門愛好者，一般來說，Home Depot 和 Lowe's 這樣的大賣場就足夠了。比如下面就是我們鎮的 Home Depot 木材區。

5.1 Dimensional lumber

一根砍伐的大樹，並不能直接用來做傢俱，需要經過木材廠的加工之後，製造成一定規格的木料。這些一定規格的木料，一般叫做 dimensional lumber。

根據不同的需求，木材廠會把樹幹切成不同的形狀。比如下面這樣。

或者這樣。

這些樹幹切出來的木料的尺寸並不是隨意的，而是有工業標準的。一般美帝常見的爲 2x4、2x6、4x4、1x4、1x6、1x10 等等。注意這些只是名義尺寸，實際的取直刨平之後的尺寸要小於這個名義尺寸，比如最常見的 2x4，實際的尺寸爲 1.5 英寸乘以 3.5 英寸。這一點要特別注意，比如選擇購買 router 不同直徑的刀頭來配合木料做榫卯節點的時候，一定要按實際尺寸買，不要按名義尺寸買。

另外，實木傢俱的桌面、衣櫃門板等等大面積的木板，多數是由這些比較小的木料拼接成的一塊大板。自然界很少有那麼粗的大樹可以產出一整塊餐桌板那麼大的木板，如果有，把這樣的樹砍了做成餐桌，那也是暴殄天物。

所有的 dimensional lumber 裏面，最爲常見的就是軟木的 2x4，所有的美帝房子，幾乎全部是由 2x4 做成的。

一根 8 英尺長的 2x4，一般售價爲兩到五美元。由於它實在是物美價廉，所以有很多業餘愛好者做的各種只用 2x4 的傢俱。

5.2 Plywood

另一種比較經濟適用的板材就是 plywood，也就是膠合板，由多層薄木片疊在一起膠合而成，這些薄木片的纖維方向互相交替，保證兩個方向的受力性能相同。用來建房子的 plywood，一般就是沒有飾面的，用來做傢俱的 plywood，最外面會貼上比較漂亮的木材的薄木片，比如橡樹、楓樹等等，起到裝飾的效果，這樣你做出的傢俱，看上去也像是橡木或者楓木做成的。一般的 plywood，尺寸爲 4 英尺乘以 8 英尺，厚度一般爲名義尺寸 1/4、1/2、3/4 英寸等等。

由於跟實木相比， plywood 的售價非常便宜，一整塊 4x8 的 plywood 售價一般爲十到三十美元不等，很適合業餘愛好者，做壞了也不心疼，進料也沒有太大的經濟壓力。所以也有很多業餘愛好者的只用 plywood 的傢俱。

plywood 的一個特點就是側邊會顯露層疊的花紋，哪怕上下表面有實木貼片，側邊仍然不可避免的露怯。一種做法是做實木包邊，另一種更方便的做法是用 iron-on edging，直接用熨斗把木皮貼邊燙上去。

當然，也可以大大方方的顯露層疊的木紋，作爲一種設計元素，比如下面這個傢俱的側邊。

另外，plywood 在美帝是極其常見的貨物，所以美帝的皮卡貨鬥都是以這個尺寸來設計的。全尺寸皮卡應該能至少平放一張 4x8 的 plywood，比如下面就是拉着好多張 plywood 的豐田坦途。

再比如本田的第二代 Ridgeline 大力宣傳的賣點之一，就是它作爲中型皮卡，也能像全尺寸皮卡那樣平放一張 plywood，而它的同級別競爭對手豐田 Tacoma、尼桑 Frontier 和雪佛蘭 Colorado 目前就做不到這一點。

5.3 MDF

MDF 是 Medium-density fibreboard 的縮寫，一般是木屑混合了一些化工產品在高溫高壓下聚合而成。這種板材看上去已經不太像木材了。比如大家熟悉的宜家，很多產品都是 MDF 做的，比如這種茶几的臺板。

需要指出的是，MDF 在加工的時候，比如鋸或者鑽孔，產生的粉塵顆粒非常小，對健康的危害遠遠大於實木和 plywood，這一點要額外注意。

6. 耗材

6.1 膠水

膠水是特別常見的連接方式，也是主要耗材之一。事實上，wood glue 專用的木工膠水的粘合能力非常非常好。絕大多數情況下，膠水連接處的強度要大於木材本身的強度。

比如下面這個小實驗，最終被掰斷的時候，膠水連接的部位沒有壞，是木料本身被掰斷了。

比如這種拼花切菜板，沒有木工膠水，很難做出這樣的東西。

6.2 金屬連接件

金屬連接件也是很常見的木工耗材。最最常用的就是木工螺釘。相比於傳統的十字形和一字形螺釘，現在的星形螺釘不容易破壞螺釘頭，可以承受更大的扭矩。表面處理過的螺釘不容易腐蝕，即使有水分存在也依然可以保證耐久性。

另外，單純的無螺紋的釘子同樣也很常用，不過相較於傳統的錘子敲釘子的方式，現在的木工已經大量的採用射釘槍。如果使用射釘槍的話，很多時候用膠水甚至可以不用夾具，抹好膠水對齊之後，立即拿射釘槍啪啪訂上，直接用永久的釘子來代替夾具。

此外，還有一些公司開發的特種連接件，很多是用於房屋結構，但也有一些可以用於傢俱。比如這種 Simpson 的連接件。

6.3 木質連接件

總體來說，各種常見的木質連接件都是卯榫結構的近似替代，比卯榫更容易加工，方便快捷。

比如 dowel joint 裏的小木棍兒，可能大家在宜家傢俱上也見過這些小木棍。

比如 biscuit joint 裏的木頭餅乾兒，作用跟 dowel 類似，也是起到固定相連木料的位置的作用。

再比如 Festool 有專利的 Domino joint，可以看成是 dowel 的加強版。

6.4 油漆清漆塗料

木材的最後上色上漆，一般稱之爲 finish。大體來說分爲兩種，一種叫 paint，一種叫 stain。

paint 就是下面這樣上油漆顏色，掩蓋了木紋的做法。好處是可以做成各種各樣的顏色，活潑可愛，而且非常耐操，壞處就是把木紋給蓋住了，沒有了木材的特殊美感。

stain 就是塗清漆或者某些油漆，凸顯木材紋理的做法。這也是傢俱最常用的方法，比如下面這種。

stain 的做法也分好多種，最常見的有 polyurethane、lacquer、tung oil 、Denish oil 等等，不同的漆有不同的保護效果和視覺效果，也有各自的最合適的木材，各自的上漆工藝也不相同。這裏我們就不深入了。特別注意的是 lacquer 的毒性很大，必須戴防毒面具才能操作。另外 lacquer 由於它的物理性質容易適用於壓力罐噴塗，所以它也是目前工業界流水線機器人上漆最普遍的做法。
