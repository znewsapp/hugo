短視頻大行其道，對內容平臺和內容消費有什麼負面影響嗎？

首先我們要站在唯物主義歷史觀上，明確一個概念，任何新事物催生的根因，是生產力的提升。

在電視機制造業與影視行業開山以來，專業級內容生產者的 PGC（Professionally-generated Content ）的代表者是一系列電影、電視劇、戲劇、廣告等，在拍攝技術與設備還沒有普及之前，是脫離民衆（或者說，民衆沒有生產這類內容的能力）的，僅僅在專業影視領域存在的產物。

但隨着拍攝知識的普及，以及科技生產力的提高，拍攝成本在一系列技術爆炸的影響下，硬件（DV、智能手機等）與軟件（各種輔助剪輯的 PC 端軟件或 APP，如 APP Store 裏的 QUIK）的支持使拍攝成本迅速降低，人人都可以製作短視頻，動態素材市場的重心可能會逐漸從 PGC 開始轉移成爲用戶生產內容的 UGC（ User Generated Content ）。前些年微電影的浪潮可以看做是這場趨勢的拐點，象徵着影視，或者說象徵着動態級素材製作技術將逐步向底層沉澱。

不僅僅是影視相關生產力的提升，技術爆炸還讓人類經歷了社交信息爆炸，首先是網絡傳輸速度的不斷提速，從電纜到光纖，移動端則是從 1G 時代到 4G 時代，且即將邁進 5G 時代（據說 2019 年可以廣泛普及），其次是互聯網技術，FB、Tencent 等巨頭提供的社交工具的普及，使得以前只能出現在電視或熒幕的視頻，可以得以在移動端及 PC 端廣泛傳播，短視頻作爲一種新的社交元素，可以逐漸改變用戶的瀏覽慣性，改變原有的圖文靜態網絡社交模式，接受靜態 + 動態社交模式的人將逐步遷移進這種複合類型的平臺。

以上可以簡單當做是短視頻火爆現象的時代基礎。下面我用民科數學來講解一下，短視頻爲什麼能提升 App 使用時長，以及其中的壞處是啥。

有人拍視頻，自然是有人想看視頻，或者說正是因爲有人拍了視頻，纔會有人想去看；內容的生產者與內容的消費者必然是共演的關係。如今短則 4 秒，長則 4、5 分鐘的五花八門的短視頻層出不窮，每一個內容生產者都可以在一個用戶基數足夠大的平臺中靠 UGC 甚至 PGC 來吸引精準用戶。

以上這個模型來自真實的商業案例，是我運營過的公衆號，主攻一個低頻消費領域，在該領域下，我主張逐漸提高商品文案動態素材的應用頻率，以促進這個低頻消費市場的用戶作出消費決策（更加合理地表達該類型商品，解決用戶決策鏈條中信息不對稱等問題）。也就是說，在內容消費領域中，某些商品的信息展露上，短視頻有着得天獨厚的優勢（如廣告一樣，可以煽動情緒，激發情感訴求，可以突出創意，可以突出實景，可以突出運動過程等）。

所以短視頻大行其道，其負面影響也並不複雜，無非就是如下幾點中的辯證的另一面：

1. 內容爲王（知名 Up 主、大 V、網紅、電競主播、公知製作的短視頻更有可能成爲爆款內容，也有民間高手製作的爆款短視頻，但基本上可以默認，好的內容一定是由優秀的內容提供者提供，所以一個短視頻平臺早期要儘可能迅速培養起這種共享優質知識產權的愉快氛圍，平臺不斷髮展過程中，短視頻的數量與質量的均衡要處理好）

2. 平臺的正外部性（保證不會有短視頻造成用戶流失，這種情況要區分純短視頻平臺與複合型平臺，純視頻平臺做好對視頻內容的審查就好，只要不引起用戶反感，不違反和諧社會主旋律，基本上都可以不管，反正看的人大於等於 0 就好。複合型平臺需要留意哪些內容適合用短視頻呈現，哪些內容不合適，社會正常運作的潛規則是效率優先，如果使用了短視頻的內容，在觀衆老爺眼前的信息吸收效率反而沒有靜態素材好，那就造成了負外部性，降低用戶粘性，即會造成用戶流失）

3. 平臺自流量的調度（用戶老爺的觀看激勵、各個內容的引流功能設計。這個就是做品類的事了，每個平臺定位不同，品類設計也不盡相同，如今各種視頻流門戶網站都有奇門遁術，在此，鄙人就不再多言）

4. 消費欺詐與誘導嫌疑（以 PGC 爲首的，逐漸轉移成爲 UGC 的，經過精心包裝後呈現給世人的糖衣陷阱，難免會讓社會的有識之士認爲這裏面有道德上的問題，但這個問題並不只針對短視頻，任何涉及營銷的內容都無法擺脫嫌疑）

另外補充一下，但其實即使是這樣，有關視頻技術的生產力問題上，真要涉及視頻領域生產力升級的，應當是 VR，也就是說，短視頻其實只是這個時代的生產資料逐漸向下層沉澱帶來的現象級產物，而 VR 纔是真正的技術更迭結果產物，畢竟平面視頻的存在已經至少有 92 年曆史了，在互聯網概念過度開發的環境下，開拓更多的視頻載體、接口，並沒有觸及時代的核心訴求，那就是，對視頻內容維度增加的訴求，對視頻信息量增加的訴求。所以最大的負面影響，莫過於誤將現象級產物當作是時代的真實訴求，以不切實際的期望運營，過度開發這個概念，我覺得這個應該纔是最大的負面影響。

回答來自機構帳號：微鯨視頻

從內容平臺角度來說，這裏分兩種。

一種是已經有視頻業務的平臺發展短視頻，這類內容平臺很可能會因爲接入大量短視頻而導致之前積累的平臺內容氛圍失去平衡，用戶粘性下降。體驗上來看，本來用戶打開這個產品是有固定的期待和預期的，而短視頻內容是一種去中心化的內容形式，用戶會陷入茫茫多的預覽圖和標題文字中，很難再找到之前歸屬感和便捷體驗。

另一種是還沒有視頻業務的平臺發展短視頻，這類平臺會面臨一些基礎性的問題，初期的內容存量從哪來？是否開放 UGC 生產？開放的話，生產工具如何簡單易用，後臺審查系統如何建立，這些工作的難度對於從圖文內容轉變到視頻內容的產品來說，是指數級的提升。

從內容消費的角度來說，可以從現有的內容商業化方式來看短視頻會面臨的問題。

貼片廣告：短視頻的觀衆大概只能忍受你貼個 3 秒的廣告，而實際情況是目前短視頻內容幾乎沒有貼片廣告的商業化形式。

會員：短視頻的內容難以進行會員收費，以第一梯隊的知名網紅、主播這些來看，還都是全網發行的方式，沒有對視頻內容進行會員收費的跡象。

粉絲經濟：這個可以，前提是你的短視頻 IP 能擠進第一梯隊，然後千方百計通過全網發行的內容把觀衆導入你的微信公衆號，然後做用戶運營，培養粉絲，做商業化。

品牌植入：這個比較多，以片頭創意口播和內容植入爲主。不過不會長久，廣告主都講究個精準和轉化。視頻裏做品牌植入，IP 類的內容還好，會形成對品牌的捆綁認知，有助於殺入消費者決策腦海，而短視頻做品牌植入，多是曇花一現，品牌和內容 CP 嗨一下就沒了下文。論廣告效果，不及 IP 鉅製也不及 TVC 廣告片。

平臺嫁接：這要求平臺同時擁有 to B 和 to C 業務，B 端的短視頻拉動 C 端用戶流量，C 端用戶的短視頻 UGC 爲 B 端做品牌背書，B 端返點給 C 端，平臺從中搭橋賺抽成。因爲是在一個平臺內，所以轉化的潛力上限比較高，意見領袖型 C 端用戶和品牌意識強的 B 端商戶都會有生產動機，當然這需要平臺在其中努力的引導，建立平臺內的短視頻內容生態。那麼，國內有幾家平臺同時具有 to B 和 to C 業務，並且有足夠用戶量來發掘培養平臺網紅的呢~

歡迎交流。

「知乎機構帳號」是機構用戶專用的知乎帳號，與知乎社區內原有的個人帳號獨立並行，其使用者爲有正規資質的組織機構，包括但不限於科研院所、公益組織、政府機關、媒體、企業等。這不僅是知乎對機構的「身份認證」，更是涵蓋了內容流通機制、帳號規範等全套帳號體系。和個人帳號一樣，機構帳號開通不需要任何費用，同時也受社區規範的監督管理，並要遵守相關協議。目前機構帳號入駐採用邀請制。您可以通過  什麼是「知乎機構帳號」 來了解更多機構帳號信息。
