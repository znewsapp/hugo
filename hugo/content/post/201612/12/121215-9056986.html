感冒的病因是感染，多數是病毒感染。

冬天跑步比較容易成爲感冒的誘因是因爲，一個冬季是感冒高發季節，室內爲了保溫空氣可能流通不佳，接觸的病毒量、特別是流感病毒的可能性要大一些；

一個冬季跑步容易着涼，呼吸道局部防禦功能降低，增加了感冒的可能性。相比夏天，冬天跑步最直觀的區別就是“冷”，運動過程又會持續產生熱量，前後這麼一動一靜一冷一熱，感冒就很容易乘虛而入了。

冬天跑步如何避免感冒，可以從這 2 點去準備：選擇適合冬季運動的裝備、養成儘快轉變運動狀態的習慣。

合適冬季運動的裝備：

冬季跑步裝備，尤其是衣物，一定要滿足兩個條件：儘可能快速的吸收並排掉汗水、讓身體保持乾爽（怎麼聽着像衛生巾廣告），維持體溫避免失溫及皮膚暴露。

溫度越低，溼噠噠的衣服就越容易變冷造成失溫，貼身上內熱外冷，體感會非常難受。另外如何在保溫的同時能做到保持透氣排汗性是所有運動廠商在設計冬季裝備時的技術攻關，想做到排汗透氣 + 保溫是很難的。

排汗透氣和保溫是兩個相對的詞，兩者都能做到的衣服那一定很貴（真話），比如羊毛含量較高的運動服裝，或者有高科技噱頭的（一般是使用的材質比較保暖透氣，同時在編織上做文章），比如 odlo、X-Bionic、FALKE 等，都有一些冬季服裝，在某些溫度下只穿一件就能排汗透氣同時又保暖，價格比較昂貴。

與夏季一樣，棉質是不適合冬季運動穿着的。同時定位戶外的、比如近年流行的軟殼，以及衝鋒衣，雖然都有一定的透氣性，但更偏重戶外低強度（徒步之類）時的性能，也更着重保溫和保護，跑步時產生熱量和汗水比較大，它們的排汗是跟不上的（當然也有一些運動品牌有類似輕薄軟殼的設計，用以低溫環境下的慢跑沒有問題）。

那一般冬季運動比較推薦的是【複合着裝】：

貼身層，比較貼合皮膚（不要與身體有太多空間）的速幹服裝，比如一般的速乾材質短 / 長袖；

外層，皮膚風衣或者跑步風衣之類，透氣但可以保證運動狀態時身體提問的。

這些衣服一定透氣性較好（反例就是一次性雨衣，同樣很輕薄但是穿着不透氣，捂着很難受，排出的溼氣冷卻後身體會感覺非常的冷），輕便適合運動、有一些巧妙的運動設計（如衣服下襬的抽繩、彈性袖口、後背 / 腋下的排氣口設計、防潑水、可摺疊，等等）。

哪怕運動時不需要，也很建議摺疊後塞跑步腰包隨身帶，運動完穿上保暖。

如果更冷地區就可以考慮那類帶絨的運動長袖 / 外套。

配件，比如魔術頭巾之類的萬能小物，當帽子當圍脖當擦汗護腕當口罩都很合適，特別適合冬季跑步時使用。

另外手和頭部是散熱大戶，也最容易感到寒冷，手套和帽子在低溫地區是很必要的。

襪子除了選擇較厚款外，儘量中高幫設計，可以包裹住腳踝。

冬季跑步有一點法則，越冷地區越試用：儘可能避免任何皮膚的外露。

就我自己的經驗，一般 5℃ 左右長袖 + 長褲，0-5℃ 長袖長褲外，再套一件速幹短袖，0℃ 左右長袖 + 皮膚風衣，0℃ 以下長袖 + 短袖 + 皮膚風衣 + 手套 + 帽子，長袖和長褲換成帶絨款。

總之就是有錢就買羊毛買那些高大上的運動貼身層，沒錢就買帶絨的冬季款。排汗性能高於一切，跑步時身體大量產熱，只要一直在動不會覺得多冷，但排汗跟不上很要命！越跑越冷，甚至汗水迅速冷卻後結冰的感覺可不好受，也大大增加了感冒風險。

另外在選擇冬季裝備上，比如鞋子襪子都有很多講究，這裏就不展開說了。

養成儘快轉變運動狀態的習慣：

1、要熱身。將身體從正常狀態調節到可運動狀態，再出門跑步，或者一開始跑步時慢點，幾公里後再正式開始訓練。熱身的關鍵是，讓你的心率緩慢提上來，有點像汽車熱車，冬天這個熱身的時間也許要比夏天更久一些，熱身的時間推薦儘量不少於 12 分鐘。

另外熱身建議採用慢跑 / 快走、動態拉伸、全身活動等方式，千萬不要用靜態拉伸，也就是公園老大爺那種一上來就壓腿死命虐關節拉韌帶。切記熱身的目的是讓心率過渡到運動狀態。

2、善用跑步小配件，皮膚風衣、魔術頭巾、跑步腰包這些在冬季跑步都非常管用，魔術頭巾可以當帽子 / 口罩，我經常一開始跑是圍住臉只剩眼睛，後面熱了再拉下來當圍脖套着。皮膚風衣掛腰包上，前面熱身時 / 跑完拉伸走回家時穿着，避免溼熱的身體迅速降溫。

3、環境越冷，越注意身體皮膚不要外露，防止凍傷和乾燥，也是爲了保證皮膚的呼吸性能。後面跑開了感覺熱，不要隨便就把手套 / 帽子給摘了，這樣一冷一熱，感冒就找來了。

我自己的經驗是，跑步帽不分前後，我一個人跑步也沒人看，跑到後面會把帽子調個角度戴，因爲額頭那邊出汗大，有時候帽子溼噠噠跑久了已經變冷了，調整一下會舒服些，也讓帽子那塊汗水可以乾的快些。

4、跑步上有些小技巧，比如呼吸上，帶個口罩 / 魔術頭巾，或者舌頭抵住上顎，儘可能不讓冷空氣直接進入口鼻，還能避免岔氣的發生。

5、避開高人流 / 居民區，我沒去過北方跑步，不知道是否存在這種現象，冬季室外排放的污染更嚴重？如果有就遠離它們。

在南方的話，人流量高一個是感覺周圍打噴嚏 / 吐痰 / 咳嗽的人更多，都是感冒潛在傳染源啊，另一個馬路邊如果是社區的商鋪或者食品店，它們有時候在馬路旁倒水，更容易變成暗冰，相比那些公園或者馬路，一些社區旁的小路明顯要溼滑一些，這個危險性在冬季是放大的。

6、完成跑步後第一件事不是脫，是讓自己儘可能維持在一個溫度，慢慢“冷身”，要麼加件衣服要麼快速進室內。跑完身體很熱覺得沒事，還在室外拉伸或者做一些無謂的事，本身就溼熱的身體會很快降溫，就很容易感冒。

這點同理運動後儘快擦乾汗水，洗澡更衣，如果要拉伸放鬆一定要穿件外套，或者在室內進行。跑步 1 小時再冷都沒問題，很多人感冒就是出在跑完那一會會時間體感溫度的驟變。

總結一下——

裝備要合適，排汗性高於一切，切勿讓汗水大量長時間在身體接觸的衣服上停留，一冷下來就完蛋；保暖的意義在讓身體可以維持在一個溫度，不驟冷驟熱，在熱身冷身階段能維持體溫，預防失溫。

善用一些冬季跑步的小配件；

避免高人流量或者潛在安全性缺失的地方，減少接觸病毒可能性；

跑步習慣要養成，別在非運動狀態時過多室外停留，跑前跑後要熱身冷身，跑完儘快加衣且保持身體乾爽；

實在過冷的低溫，或者雨雪天氣，不一定就得挑戰嚴寒去室外跑步，進行別的運動，或者在家吃吃火鍋養養肥也是不錯的。
