不，就目前來說，紙幣是我們最後也是最重要的堡壘，或許有一天我們可以採用類似比特幣的這種匿名電子貨幣來取代紙幣，在此之前，紙幣因爲其匿名支付的重要性而無可替代。

我們從最基本的消費說起。爲什麼商家不會對每個人收取不同的價格？因爲商家不知道每個顧客到底願意對這件商品最高出多少錢。所以商家只能大概的估算出來一個分佈，然後收取一個統一的價格。比如一件商品，商家標價 200， A 沒有買， B 和 C 都買了。我們假設 A 願意出 100，B 願意出 200，C 願意出 300。對於 C 而言，C 願意出 300，但是隻花了 200 就買到了，所以 C 獲得了 100 的淨“剩餘”。這個剩餘其實就是因爲 C 的私人信息“最高出價 300”沒有被商家知道得來的，可以稱之爲 C 的“信息租金”，也就是因爲 C 擁有自己的私人信息而獲得的剩餘。

而假如商家通過某種心靈探測器，瞭解到 ABC 三個人的最高出價，那麼可以把同一件商品分別賣給 A 100， 賣給 B 200，賣給 C 300. 這個叫做“一級價格歧視”，也叫做完全價格歧視。在完全的價格歧視下，消費者是沒有任何剩餘的。

如果我們取消了紙幣，在目前的環境下，無論你是用支付寶還是用微信，你的消費記錄和偏好都被原原本本的記錄在案，這就是事實上的一個“心靈探測器”，當商家知道越來越多你的消費偏好和習慣的時候，你會享受到越來越多的“個性化服務”，但是在此同時，你的消費者剩餘也會越來越少。

這也是機制設計的內涵所決定的：機制設計者爲獲得參與者的私人信息而支付信息租金。而沒有私人信息的時候，機制設計者可以讓參與者做一切滿足參與約束的事情。

到時候你就只有兩個選擇：

  • 老老實實的消費，然後忍受越來越少的消費者剩餘。
  • “欺騙”心靈探測器。就是你明知道你想要商品 a，你想買，但是你知道你買了之後，你的購買信息就會被記錄在案，所以會顯著的提高你接下來購買商品 b 的價格，所以你會忍住自己不購買商品 a，然後去購買商品 b。這種情況下，你確實會有多一點消費者剩餘，但是因爲你扭曲了自己的購買行爲，依然造成了很大的損失——在匿名支付的情況下，你本可以同時擁有 a 和 b 的。

生產力在發展，社會創造的財富是越來越多的，但是如何保證消費者也能享受到越來越多的剩餘呢，一個重要的方面就是要限制發掘消費者的私有信息和偏好。而紙幣在其中扮演着重要的角色，只要紙幣還在流通，只要還有相當多的人都在使用紙幣，支付寶和微信就永遠不敢說掌握了大部分消費者的消費習慣和偏好，所以也就無法有效的對消費者進行價格歧視了。

上面的例子中，我們有兩個隱含假設：

1. 平臺具有很強的壟斷力量，

2. 人是 identifiable 的，也就是說平臺可以很精確的區分消費者；同時商品無法隨意轉售，類似現在的數字遊戲。比如說賣給你的電視機，只有你才能用，別人壓根就無法使用，可以通過虹膜識別等技術手段實現；
