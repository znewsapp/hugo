2016 金球獎揭曉，C 羅生涯第 4 次獲獎

C 羅上賽季的表現配得上一切個人榮譽，儘管在我心裏他之前拿到歐洲最佳球員比金球獎含量更高。

要說 2016 年遺憾也有，比如最後時刻金靴獎被蘇亞雷斯反超；歐洲盃決賽傷退；因傷錯過歐洲超級盃，差點成爲同一年拿到歐冠 + 歐洲盃 + 歐洲超級盃大滿貫第一人。除此之外，堪稱完美。

C 羅上賽季表現最佳的三場比賽：西甲 2:1 逆轉巴薩，絕殺諾坎普。歐冠八分之一決賽 3:0 淘汰狼堡，上演帽子戲法帶領皇馬絕地逃生。歐洲盃半決賽 2:0 淘汰威爾士一球一助攻帶領葡萄牙打進歐洲盃決賽。

以上瞬間，再次回憶仍然熱血澎湃。本以爲步入而立之年的他很難再次回到世界之巔，本以爲功名成就的他對於勝利已不是那麼渴望。但這就是 C 羅，任何比賽都要爭做最好的那個人，一直都是。

賽季初 C 羅因歐洲盃大傷錯過幾場比賽，錯過夏季拉練迴歸之後狀態起伏不定，每年“C 羅老了，C 羅開始走下坡路”的論調再次被炒作起來，那段時間即使平常訓練踢飛一個球也會被認爲在表達着不滿。結果他用兩個帽子戲法完成了打臉和迴歸。（阿拉維斯以及馬德里競技）

皇馬各項賽事已經保持了 35 場不敗記錄，之前僅輸過的兩場比賽（狼堡以及馬德里競技）後來都由 C 羅用帽子戲法完成復仇。他的個人能力也許在現役球員裏不是最好的，他的身體機能也許正在無法避免的慢慢老化，但 C 羅的大心臟是越來越強，被黑的越厲害，噓聲再多，他更能進球。這顆大心臟的支撐就是自信——皇馬隊史以及葡萄牙隊史第一射手用每一個“粒粒皆辛苦”的進球建立起的自信。

足球是一項集體運動，任何個人榮譽都是整個團隊努力的沉澱。誠然，沒有拉莫斯的進球，皇馬歐冠決賽可能會被馬競打敗。沒有埃德的靈光一現，葡萄牙歐洲會敗在東道主腳下。但沒有 C 羅，可能連決賽都打不進。C 羅目前對皇馬的意義已經不是簡單的射手，更是一個精神領袖，他這七年爲這支球隊輸入的信念，多次關鍵時刻的挺身而出是無法被抹去的。

要說對 C 羅今後的職業生涯還有什麼展望？真的很難了。因爲大部分重要的紀錄都是他在保持，幾乎每場比賽過後都是一次刷新。這個賽季或許他不能打進 50 個進球了（C 羅已經連續 5 個賽季打進 50 球，歷史第一人），但能看到一個轉型期的 C 羅，一個爲冠軍服務的 C 羅，一個仍然打拼的 C 羅也何嘗不是一種幸福呢。

最後再談一談梅西。金球獎被梅西和 C 羅壟斷的這麼些年，背後更是兩傢俱樂部，兩個人，兩隊球迷之間的愛恨情仇。如果沒有梅西，肯定不會有更好的 C 羅。反之也是如此。不久前的西班牙國家德比 C 羅特意在賽前通道等待梅西與他擊掌，比賽時二人的親密合照也讓人動容。他們二人佔據了足壇半壁江山，但終有一天會老去，而且這一天正越來越快的到來。寫到此，心裏頗有酸楚，梅羅二人前無古人的時代多少年纔有來者追上？這不重要了。因爲即使有我們也沒有激情去追了。作爲 C 羅球迷，對於梅西我只能獻出真誠的尊重，我肯定他毋庸置疑的實力，肯定他低調沉穩的性格，珍惜二人競爭的時代，別無他言。

祝賀 C 羅。對於這個賽季願景就是別再受傷。在這個年紀受傷也許對剩餘職業生涯的影響是致命的。剩下的榮譽不必多說，敬請期待吧。
