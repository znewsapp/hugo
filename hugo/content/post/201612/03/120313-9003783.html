大部分人覺得衣服收納是爲了讓房間看起來乾淨整潔，但我認爲衣服收納的意義是在於能夠最大限度地保持衣服的色澤、版型、質感，從而增加衣服的壽命和價值，最終達到省錢，哦不，提高生活品質的目的。

衣服收納這件事可以追溯到洗衣→晾衣→熨燙 / 摺疊→存放等這些步驟，每一個環節都會影響到衣服的品相。試想一下，不認真的洗滌和晾曬，導致衣服發皺變形，最後熨燙和摺疊時已經很難將衣服捋順了，更別說還要將這樣的衣服穿在身上。今天跟大家分享我在衣服收納這方面的一點經驗，希望能爲你帶來一些收納靈感。

洗

洗衣服很簡單，先把衣服丟進洗衣機，加入洗衣液，摁下開關鍵就好了。不過如何讓洗過的衣服保持最初鮮豔整齊的模樣，這當中有許多值得注意的小細節。

/ 一定要用洗衣袋

在對衣物進行分類後（深色 / 淺色、褪色 / 不褪色、家居服 / 外出服、貼身 / 非貼身），將分類好的衣服放入洗衣袋中可以防止衣服在洗衣機中拉扯變形，根據衣服的數量準備多個使用。

/ 反個面再洗

鮮豔顏色的衣物，特別是牛仔褲長期洗滌後，色彩褪去就失去了當初的美感，將衣服反個面再洗滌，可以減輕褪色的程度。

/ 褲子洗之前疊一下

褲類衣物由於褲管較長，在洗滌過程中容易扭轉產生皺紋，可以先摺疊一下再放入洗衣袋中，這樣洗好的衣服就不會皺巴巴的，不易變形也減少熨燙的成本。

晾

晾衣服的時候，潮溼的衣服由於重力增加，很容易因爲晾衣架的拉扯而產生變形，所以晾衣架的選擇是特別重要的。

/ 上下裝衣架要分開

市面上有的衣架會在兩側做出細微的弧度，方便晾曬褲子，但這種造型的衣架拿來晾上衣時，很容易會導致上衣肩部被撐出一個謎之弧形，穿上身後肩膀處總鼓出一個小包，凹好的造型分分鐘打個五折。所以，千萬記得上裝和下裝的晾衣架要分開，或者統一使用最基礎款的衣架。

/ 根據材質選擇晾曬手法

晾上衣的時候，很多人會採用領口懸掛法，即利用衣架將領口和肩膀撐起的方式，這種方法領口和肩部很容易被衣架拉扯變形，個人推薦用腰部摺疊法晾曬~

腰部摺疊法就像掛毛巾一樣將上衣攔腰摺疊，耷拉在衣架橫杆處晾曬，這種方法適用於所有的套衫、POLO 衫和吊帶衫，特別是純棉等柔軟易變形的材質，避免領口、肩部和揹帶變形拉長。

針織衫、薄毛衣等衣服自重大又容易變形，清洗的時候需要放進獨立的網袋中，也不適合掛着晾曬，平攤在籃狀的晾衣網上，像鹹魚一樣等待風乾即可~

一般晾褲子都是將褲耳（固定皮帶的小環）穿過衣架的凹處或者掛在衣架的掛鉤上，這種方式能保持褲子的筆挺也不容易產生變形，但是遇到沒有褲耳的褲子時就沒轍了，我曾經試過將褲腰卡在衣架上，最終導致褲腰兩側被撐出兩個真·耳朵，腰圍也被撐大沒法穿了。

血一般的教訓讓我走上研究晾曬大法這條不歸路，後來發現褲子也適用摺疊晾曬法，也就是將褲子從膝蓋處摺疊耷拉在衣架上，不過這種方式很容易在曬的過程中被風吹落，只要將褲管分開從兩側掛到衣架上，再也不怕風吹了，哈哈~

褲子採用摺疊晾曬法比較難晾乾，梅雨天可以多用幾個夾子夾住褲腰，增加褲子的空氣流通量加速風乾的過程。

/ 反個面再晾

衣服經過長時間的清洗和晾曬，容易泛白顯得舊舊的，在晾衣服的時候可以將衣服反面朝外（特別是牛仔製品），儘量保持衣服原有的色彩，延長衣服的壽命，我的衣服都是這麼晾的，親測有效。

疊

如果你做完上面的步驟，將會收到一堆品相尚好（平整不起皺）的乾淨衣服，將襯衫和西褲熨一下就可以開始疊衣服咯~

說到疊衣服，可以分爲三個流派，一個是不疊全部掛起來的懶癌派；一個是以服裝店爲教科書模板的平疊派；最後是新晉的以近藤麻理惠爲教主的豎疊派。到底哪種好呢？我們來看看。

/ 懸掛

大部分的衣服都能採用懸掛的方式收納，但棉質的 T 恤、背心和毛衣掛久了容易變形，最好還是摺疊起來。商務襯衫、雪紡類的衣服比較挺括，穿的時候不能有太多摺痕，適合用懸掛的方式。

/ 豎疊

豎向摺疊的方法就是將衣服多疊一下，能夠立起來存放，這樣收納的衣服一目瞭然，在拿取的時候也不用擔心會弄亂其他衣服。

不過豎向摺疊會讓硬挺的衣服出現較多摺痕，衣服太少的時候還需要立板擋住，讓衣服不至於像多諾骨牌一樣倒下。

總的來說適合豎向摺疊的衣服有 T 恤衫、棉質襯衫薄的衛衣和毛衣，這些衣服不容易也不用擔心有皺紋，疊成差不多 1/3 大小就可以豎着收納。

/ 平疊

有些衣服很輕薄（絲質品、莫代爾棉），摺疊後依舊難以站立，懸掛又擔心變形，那就只好採用平疊的方式收納。另外比較厚的毛衣或者加絨的衛衣等，豎着疊反而容易佔地方，也可以採用平疊法收納。

/ 隔夜衣

還有關於隔夜衣的收納問題，如果擔心衣服放在外面顯得凌亂，可以在衣櫃中設置一個籃子專門存放。

我曾經喜歡頻繁更換衣服，換下來的“隔夜衣”就這麼存在籃子中，越積越多直到溢出籃子。有一天看着凌亂的衣櫃，痛定思痛，爲自己定下規矩，就算隔夜衣，也必須摺疊得很漂亮放起來！就這樣，內心中懶惰的小人戰勝了愛美的小人，隔夜衣氾濫的情況終於有所改善。

就算是隔夜衣，也要認真對待▼

衣服收納的方式基本上決定了衣櫃的構造，通過今天的文章跟大家分享我的衣服收納理念，希望能讓大家開始思考和建立適合自己的衣服保養&收納體系，從而規劃出適合自己的收納空間。關於衣櫃的問題，也可以翻看我的文章衣帽間：除了顏值，更要有實力。如果你有了不起的私藏收納技巧，歡迎在底下留言，讓我們共同學習，共同進步~
