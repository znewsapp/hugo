如何評價美劇《西部世界》（Westworld）？

第一季完結撒花，輕度劇透預警。

作爲「西部世界（美劇）」話題索引的編輯者，無論是劇集本身還是知乎的討論，我也算是和大家緊緊追了一季。雖然每個週一，都早早在更新索引的時候被劇透個爽，終於到了第十集的時候，好心的 HBO 鼎級劇場邀請我去參加看片會，第一時間看到了終季結局，拯救我於劇透地獄。看完之後，站在剛剛吹散北京霧霾的大風裏，氣流裏攜裹着北半球深冬的陽光和冷，東三環高樓矗立，車流擁堵，無論是剛剛吃過的杯裝提拉米蘇，還是路過人們平淡無奇的羽絨服，都散發着存在的味道。被這現實的大風不帶憐憫的吹着，突然就想起來了片尾曲的名字， Radiohead 的 Exit Music（For a Film），today we escape，we escape。

《西部世界》第一集結尾, Teddy 下意識的撫摸着自己上次中彈的胸口在火車上迷惘的醒來; 黑衣人帶着印第安荷官的頭皮，一騎黑馬獨行於尋找 Maze 的路上，而 Delores 在自己「我知道一切皆有定數」的旁白中，淡定拍死一隻蒼蠅。片尾曲這時響起，一首經典福音音樂，Ain’t No Grave, 據傳由當時身患肺結核的靈歌歌手 Claude Ely 即興創作於 12 歲，帶着一點冥冥之中神啓的意味。「There ain't no grave can hold my body down」這一句，伴隨鈴鼓被吟唱多次。歌詞裏描摹天使，遠方和不知前路的靈魂，大地之上沒有墳墓能夠留住死亡或者生命，這個故事纔剛剛開始。作爲觀衆，我們懷疑自己被編劇用平行世界或者多維空間愚弄，我們懷疑黑衣人是真實存在的人類還是編碼而成的 AI，也問自己，如果真的有一個「西部世界」，你手持門票，跳下火車，是會參與一場殘暴歡愉，還是依然遵循規矩與美德？所謂「真正的自我意識」，對於 Host 來說是跳脫程序編排，對於人類來說，難道處於爲所欲爲的世界裏，才能發掘它的存在？

就是從這一首歌開始，一個故事在溯回，交疊，跨越中漸漸展開。劇中 Host 們追逐迷宮，我們則在一集一集的劇情演進中驗證假設和結局。戲裏戲外，都在尋求一種真相，無論它是被早已寫在程序或者劇本里，或者隨着時間和記憶的更迭，有着更加隨機的演繹。

追這樣一部劇，交流就成爲了不可少的一部分。從第一集到第十集，每個劇情討論的問題都聚集了幾百個答案和上千關注者，「西部世界（美劇）」這個話題，也在短短兩個多月的時間裏有了 43000 多的關注。它有時候是觀劇指南，有時候是辯論場。之前我雖然沒有爲這部劇寫過任何評論或者分析，但爲了編輯更新索引，也爲了看大家的紛繁答案，每週都會爲這個話題貢獻不少 PV。有太多的猜想和假設需要驗證和分享，看到相同的觀點按下贊同，遇見未想到的細節丟給朋友一起討論，當然也會有與自己的猜想相左的觀點出現，嗯，算你是另外的平行宇宙啦，不然我們打個賭，諾蘭會引領情節走向哪一邊？雖然不知道自己這一點微小的工作是否有助於他人（可能更有效的是用索引頁面隔離更直接的劇透），不過對於我而言，從第一天開始持續的關注話題和討論，是看劇之外更加有趣的拓展閱讀。也許不小心看到這篇答案的你，也有一部真愛的劇集或者電影，要不要也來試試編輯它的話題索引？整合更有效的信息之餘，也能享受一場更完整的劇透。當然，重點是前者嗯。

花了很多筆墨給第一集，除了因爲它是一個開始，也同樣由於這是和最後一集一樣，我唯二沒有被劇透的兩集，看的時候更多了點原生的震撼。70 分鐘，已經足夠讓許多個相互關聯而又看似獨立的故事展開它的一角。我沉迷於 Hector 蒙面騎白馬而來，血洗酒館的一段。背景音樂是滾石的Paint it Black，經過重新編曲配器，配合刻意放慢的鏡頭，如同酒吧裏自動鋼琴的表演，帶着非天然的美麗無情。女強盜從容擡起雙管獵槍，瞄準，射擊，彈無虛發，日光之下無新事，千百次循環中平淡無奇的一次重複。那時候並不知道她的名字，後來才知道是 Armistice，我天然對字母「A」開頭的英文名字抱有好感，又何況是這樣一個看起來遠離她的暴戾，帶點性冷淡意味的單詞。第十集裏，她和 Hector 果然光芒萬丈，最後的彩蛋，也是她的主場。這兩個人似乎從不糾結於自我意識或者迷宮，說起模糊存在於記憶中來自異世界的回收人員，也並未抱着對待神明般的敬畏。無論是在甜水鎮還是 Delos 公司，有一把槍，就有血與火。他們不糾結於自己的存在，我也同樣不糾結他們的行動是否由自由意志所指引。簡單粗暴的一對暴力搭檔，實在可愛。

當年看到這裏，獨坐於電腦面前的我，只想隨便抓住一個人，近距離直視此人的眼睛，連說三遍「這他媽實在太酷了」。苦於無人即時交流，所以經過互相強喂安利，我們由看電影分隊發展出來的魔獸世界五人刷刷團，又成爲了西部世界討論組。當年一橙不染，裝等喜人的我，友人 ABC 以及隨機出現的 D，一邊苦戰於蘇拉瑪，一邊交流關於西部世界的各種迷思。首先困擾我們的是，作爲人類，在樂園裏如何分辨誰是 host，誰是真正的同類，接下來，腦洞由此就延伸到了每個或者每隊 guest，他們是參與了單獨爲之開放的一個副本，還是大家一起進入了 MMORPG 混戰。討論了一輪「故事線多久會循環一次」，「每隊 guest 是否都是單獨出現」，「黑衣人是否遇到過其他 guest」，乃至「黑衣人是否是人」（是的，我曾在第五集之前堅定的相信過，黑衣人也是一個有着既定故事線與職責的 host ），並且羅列推敲了許多細節之後，我們開始在大祕境裏談論尚且模糊的幾條時間線。Logan 與 William 登場之後，友人 ABC 中最勤快的那一位整理了幾位角色的主視角，並試圖把被編劇刻意打碎模糊的時間節點拼接起來。Teddy 與年輕時代的 William 並不處於統一時間線，以及幫助 Dolores 拾起牛奶罐的 Teddy 和 William 之間必然有一些微妙的穿越時間的聯繫，大概算是謎底揭開之前，這個討論組達成共識的研究成果。其間，友人 B 如同神助，在第二集剛剛結束之時，就預言到與 Ford 在沙丘上對話的男孩，大概就是少年 Ford 的 host 版再現；我也覺得 Bernard 的舉止總有一種細微的違和感，他以及更多在控制總部工作的人，應該也只是比起園區 NPC 具有更高權限的 host。

起初幾集，我的樂趣在於辨認猜測時間線和 host，隨着劇情的慢慢推進，假設的過程也漸漸傾向於驗證。對於劇情的思考，也擴散到更加形而上的層面。 這種關於存在以及自我意識的思考和恐慌，很多來自 Maeve 的故事。她，也是我把《仿生人會夢見電子羊嗎》放進書籍推薦的原因之一。小說中的未來都市陰霾冰冷，人類執着於排除異己，仿生人拼命證明自己活着。在 Maeve 身上，兩種矛盾大概是共存的。一方面要打破這個世界的既有界限，一方面想從內在的情感和意識中喚醒自我的存在。雖然她並未吟誦引用任何莎士比亞的臺詞，卻如同莎翁悲劇中的典型角色，強大而多情，有過抗爭與彷徨，最終還是臣服於不可知的宿命。有些朋友認爲，最後一幕中 Maeve 終於坐在駛離樂園的火車中，拿着女兒座標的紙條躊躇很久，終於在最後一刻選擇迴歸西部世界去找尋女兒，是一種覺醒後人性光輝的反射。對此，我持完全相反的意見。 Maeve 曾經明白，自己任何語言其實只是早已寫好的臺本重現，關於「女兒」的記憶只是因爲曾經作爲 host 扮演母親的角色，甚至連反抗逃離也是編排好的戲碼，但最後，她依然遵從於早期故事線的設定，以一個母親的原始設定身份迴歸西部世界。大型廣告牌上，「Live Beyond Limit」Slogan 之下，依然播放着 Maeve 和女兒白衣白裙其樂融融的影像，這是樂園造物主爲她安排的最深刻的願望與夢。地是空虛混沌，淵面黑暗。神的靈運行在水面上。神說要有光，就有了光。至於爲什麼，這時的她不想。

看完這一幕後，我的時間線就到了一開始提到的東三環。深冬的陽光冰涼刺眼，我對自己的存在，也有了些許疑惑動搖。有時候我是手持薩格拉斯權杖的毀滅術士，這時的世界有着明確的邊界，分辨 NPC 和玩家輕而易舉。然而當我登出帳號，迴歸所謂現實，真實和虛假就從簡單的二元對立模糊成了抽象的概念。真實可能隨着概念的轉化，符號的替代和信息的流動慢慢被消磨，一個世界最真實的面貌，往往無法是感官認知所能觸及表達的。

除了宏大諸如世界觀，一些細節也一樣迷人。 @馬元翔@尼克桑@尚鹽 等知友提過的西部世界官方網站 Westworld: A Delos Destination ，真的是一個非常有趣的地方（他們的回答也很精彩）。只是因爲地域 IP，總是會被強制跳轉到 HBO 的亞洲官網。當時費了一番力氣才登入，如同各種旅行預定網站一般定好房間以及入園時間之後，就會有一位名叫 Aeden 的 host 來介紹關於西部世界的種種。留下郵箱，還會每週收到來自 host 或者 Delos 公司的郵件。隨故事進展，郵件裏也會寫到，隨着 Ford 博士的新故事線即將完成，Delos 董事會的成員也會到訪，這場慶典活動，我們邀請您一同參與。每當看到一封來自「Aeden at Westworld」 的來信，都有突然從現實世界掉進愛麗絲兔子洞的錯覺。在網站上可以和 host 對話，我說了一句 「I love you」，沒想到卻被回贈了一句經典臺詞。

嗯，好吧。當時大概播到第三集，William 與 Dolores 之間的感情尚未鋪展開來，現在回頭想想，好在 William 沒有向 Dolores 說出這三個單詞，否則他的黑化大概要提前不少。

另外一個想要提提的，是《西部世界》的 Tagline，「Every hero has a code」。我之前在知乎也提過一個關於它的問題： 《西部世界》的 Tagline 「every hero has a code」有什麼特殊含義？ ，雖然回答數量不多，但大家對此都有一些自我的解讀。HBO 看片會之後，剛好有一個短暫的 James Marsden （Teddy 扮演者）見面會，我就把這個問題拋給了 James，順便請他聊聊 Teddy 的 code 在他心中應該是怎樣的。James 回答說：

我認爲 code 應該是有兩重含義。從客觀層面來說，host 是被人工製造，程序控制的。另一方面，從更主觀的角度來看，每個角色都有他的核心特質。至於 Teddy，如果用幾個詞來形容他，那麼應該是榮譽感，高尚和善良（honor，dignity，goodness）。 他與很多其他 host 一樣，是「西部世界」的受害者。他死了成百上千次，內心也依然是堅毅善良的。他是一個很好的人類（good human being）。

James 在回答裏特意用了「human being」這個詞來形容 Teddy，大概在他眼中，Teddy 是配得上「人類」這個身份，他有血有肉，也有一個美好的人格。回顧整季，Teddy 確實是最深重的受難者。當他被程序驅使，協助 Dolores 屠殺全鎮時，即使不能控制自己停止舉槍射擊手無寸鐵的居民，也絕望的喊着「我都做了什麼？這不對勁」。他知道自己的行爲不受自己控制。我相信，這是某種自由意識。

《西部世界》真的是一部巧妙的劇。追着它的這兩個半月，因爲有了每週一的期待，時間過得飛快而輕易。完結之後的本週二三四，我都處於一種惆悵的賢者狀態，於是決定記錄一下這個被虐以及被劇透觀衆的心路歷程。磕磕絆絆寫這篇回答的時候，大概循環播放了十幾遍 Radiohead 的 Paranoid Android ，雖然這首歌不曾在劇集中出現，但隨機往復的旋律和歌詞，和主題也是情理之中的合拍。

Please could you stop the noise'
I'm trying get some rest.
From all the unborn chicken'
Voices in my head.
What's that'

What's that'

Westworld，我們 2018 年再見。
