收購過程中，收購公司支付的價格是目標公司的股權價值還是總價值？

從法律上來說，支付對價收購的是企業股權，當然是 Equity Value，但是你肯定在做併購時聽投行 blabla 跟你說了半天 Enterprise Value（EV），所以有點懵？中國人一般習慣了用市盈率來給企業估值，有多少利潤算算乘以一個倍數就給企業估值了，聽起來也蠻合理，Equity Value = Earnings * PE。

個人認爲這更多是從參股或者買股票的角度看的，因爲你是個小投資者嘛，唯一的盼頭就是分紅或者股價上行，那重點看看市盈率也是講道理。但如果做控股型收購呢，就不能這麼簡單了。

這個時候，Equity Value = Enterprise Value + Cash -Debt 也許更合適。Enterprise Value 考察企業本身的有機價值，可以理解爲能產生利潤，或者更嚴謹些，產生自由現金流的資產包，而這個資產包之外的現金和負債是不影響企業價值的，相應調整後得出企業的股權價值。

那 Enterprise Value 怎麼估呢？複雜一點就用 DCF 模型計算企業自由現金流折現後的金額，簡單點看看 EV/EBITDA 也行。計算完 EV，然後再看看企業賬上的現金和負債情況，如果企業賬上沒有負債但是一大堆現金，那麼股權價值應該等於 EV+Net Cash，即你花的股權投資款不但包含企業本身，還包含了可以立即分紅的現金；如果相反，企業負債很多，那股權價值就應該小於 EV。而如果只看市盈率，你很可能得出有淨現金和淨負債的兩個公司估值差不多的錯誤結論。

好比說，同樣是買二手房，都是相同小區相同戶型，都賣 5 萬一平米。但一個是沒裝修的清水房（還得花上 20 萬裝修才能住），一個是宜家風格簡約裝修拎包入住，另外一個豪華裝修牆上還有一幅價值 100 萬的名畫，你覺得估值能一樣嗎？能只看每平米多少錢嗎？Enterprise Value 就是房子每平米值多少錢 * 面積，好比 EV=EBITDA * Multiple, Equity Value 則是在 EV 基礎上根據房子的其他情況（追加裝修才能住 or 賣掉名畫變現也不耽誤住）做調整後的估值。
