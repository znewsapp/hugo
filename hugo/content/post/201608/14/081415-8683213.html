感冒的常見誤解

感冒是人類最常見的疾病，每個人每年都可能會得上幾次。美國和加拿大衛生部門的數據統計顯示，成年人每年會得 2~3 次感冒，而兒童可能多達 6~7 次。然而感冒這個病，說一萬次也不嫌多，因爲大多數人（包括醫生）對感冒都存在一些誤解。

我們平常說的感冒一般指普通感冒，指的是以流鼻涕咳嗽等症狀爲主的一種病。在講普通感冒之前，我們得先把普通感冒和另外一種名字類似的疾病——流行性感冒區分開。這是兩種不同的疾病，我們首先要清楚兩者的病原體是什麼。

普通感冒俗稱“傷風”，屬於急性上呼吸道感染的一種臨牀表現。就目前所知，有大約兩百種病毒（不包括流感病毒）可能引起普通感冒。

而流行性感冒是由流感病毒引起的，在新聞上常見的 H5N1、H7N9 都是屬於流感病毒的一種亞型（順便說說這個代碼的意義：流感病毒的包膜上有兩種類型的糖蛋白，一種是血凝素，即 H，有 15 個亞型；另一種是爲神經氨酸酶，即 N，有 9 個亞型。）

關於普通感冒的常見問題通常有以下幾個，這裏一一做詳細回答。爲方便閱讀，以下部分用“感冒”代替“普通感冒”。

1. 着涼會引起感冒嗎？

不會，如前所說，感冒是病毒引起的，着涼不會導致感冒。只有感冒病毒通過鼻腔進入人體後，人才會可能出現感冒症狀。

可能有人要說，我被冷風一吹 / 洗冷水澡就打噴嚏，這不是感冒麼？當然不是，打噴嚏雖然是感冒的症狀之一，但也是正常人在受到外界刺激時的生理反應之一。病毒進入人體到出現症狀的這段時間被稱爲潛伏期，對於感冒來說這個潛伏期大約是 1~3 天，因此一着涼就打噴嚏，顯然不是感冒。

至於冬季時感冒患者增多，同樣不是寒冷所致。冬天時大多數人會減少室外活動，留在室內的時間增加，而長時間處於人羣擁擠的封閉環境對於病毒的傳播非常有利，因此冬天時感冒的人會增加。

很多中國人都對着涼有一種莫名的畏懼，雖然寒冷的確會導致一些疾病，例如凍瘡，另外寒冷時期腦血管疾病的發病率也會顯著增加，但是寒冷並不會導致感冒，也不會導致關節炎。關於寒冷和關節炎的關係是一個話題，以後再說。

2. 感冒需要吃抗生素麼？

抗生素其實就是大家平時常說的“消炎藥”，比如先鋒四號、阿莫西林等等，但其實抗生素和消炎藥是完全不同類別的兩種東西，大家可以上網查詢一下兩者區別，這裏暫不討論消炎藥。

抗生素，顧名思義就是抵抗細菌生長或者直接殺死細菌的藥物（實際上還包括抗真菌藥物）。細菌和病毒是兩種不同的生命體，兩者的一些重要區別請見下表：

感冒是由病毒引起的，自然不需要使用抗生素治療，用了抗生素不但不能縮短感冒病程，也不能減輕感冒症狀，反而可能會引起其他副作用。目前抗生素濫用是一個嚴峻的問題，抗生素濫用的成本並不在於抗生素本身，而在於超級耐藥菌感染之後的治療成本。因此，除非在感冒期間繼發出現了細菌感染，否則一般不需要使用抗生素。

3. 維生素 C 能防治感冒嗎？

維生素 C 是一種水溶性維生素，最早因爲可以治療壞血病而廣爲人知。根據不同國家的膳食指南，一個成年人的維生素 C 的日需求量大約爲 45~100 毫克，而每天一個橙子即可提供足量的維生素 C。維生素 C 在人體內有很多作用，比如輔助膠原蛋白合成、預防牙齦出血、參與免疫細胞的產生等等。

幾十年以來，維生素 C 對於預防和治療感冒的作用一直是個備受爭議的話題。曾有研究顯示，感冒後服用大劑量的維生素 C（200~1000 毫克）可以縮短感冒病程，然後這一結論與後來的其他研究結果並不一致。一些研究表明，感冒後再服用維生素 C 既無法縮短病程，也不能減輕症狀；而如果是每天服用的話，大約可以縮短 8% 的病程——換句話說，每個成年人每年大約會感冒 12 天，而如果你一年吃了 365 天維生素 C 的話，你可能只會感冒 11 天。綜合目前大多數研究的結果來看，維生素 C 既不能預防感冒，也不能減輕感冒症狀，而只對於極端條件下的運動員來說可能會有幫助。

說了這麼多結果不一的研究，那麼維生素 C 對於感冒到底有沒有幫助呢？用一句話總結就是：目前來看，沒太多幫助。

4. 小朋友生病了要吃什麼感冒藥？

如果家裏有小朋友出現了感冒症狀，請諮詢兒科醫師的意見後再購買感冒藥，切勿自行購買成人感冒藥給小朋友服用，尤其是兩歲以下嬰幼兒出現感冒症狀的時候。2008 年 1 月 17 日起，爲了防止感冒藥副作用對嬰幼兒的傷害，美國 FDA 已經全面禁止銷售兩歲以下兒童感冒藥。另外，FDA 建議 6~11 歲的兒童不要使用成人感冒藥，以免出現嚴重的不良反應。美國兒科學會同時建議兒童以及青少年在感冒時不要服用阿司匹林或者含有阿司匹林的感冒藥，因爲可能會引起瑞氏綜合徵（Reye`s syndrome，一種罕見的藥物不良反應，但是後果非常嚴重，可能會導致死亡）。

5. 感冒了應該怎麼辦？

根據美國國立衛生研究院（NIH）的建議，感冒雖然是不治之症（因爲目前市面上的感冒藥都無法直接殺死感冒病毒），但可以採取一些措施來減輕症狀。NIH 給了我們以下一些建議：

·多躺牀上休息；

·喝足夠的水；

·使用溫鹽水漱口或冰片以減輕喉嚨痛的症狀；

·服用減充血藥物（編者注：如僞麻黃鹼）或鹽水鼻腔噴霧以緩解鼻部症狀；

·服用阿司匹林或對乙酰氨基酚——比如泰諾——來治療頭痛或發熱。

這些措施都能讓感冒患者感覺更舒服一些。感冒是一種自限性疾病，對於大多數人來說，即使不吃藥物，感冒也會在 7~10 天左右痊癒。

以上就是關於感冒的一些常見問題。即便是像感冒這樣最常見的疾病，大家仍然會有種種誤解，更何況其他不常見的疾病。因此，在健康方面遇到問題的時候，最靠譜的方法便是尋求專業人士的幫助，而不是輕易嘗試微博或朋友圈裏面那些天方夜譚一般的偏方。
