從零開始的日本動畫製作工業流程解讀之畫面製作篇

無論是日本、歐美還是中國動畫，片尾工作人員名彈裏會寫上每個參與動畫製作人員的名字和相應的職務，但不同派別動畫類似日本動畫和迪士尼這種歐美動畫對同一功能的職位可能有着不同的稱呼，而同一稱呼的職位也有可能有着不同的職能，於是今天開始就和大家聊聊日本動畫片尾中的那些職位和任務，以及簡單說明日本動畫在畫面製作方面的流程

作畫監督 Animation Director

作畫監督的職能就是要盯着自己負責的那一話內容的作畫質量和畫風，保證每話的動畫畫面都能保持一致性並且每話的質量都要有保證，作畫監督主要注意的是關鍵情節的原畫，而並非是全部的原畫，而在歐美的動畫製作中 Animation Director 就是作畫監督，肯定有很多人誤以爲 Animation Director 和日本動畫的監督的地位一樣，有時候他們也被稱爲 Animation Supervisor ，這個描述可能更貼切一些，事實上在 Sunrise 和萬代的一些動畫中，作畫監督也經常以 Animation Director 片假名的形式寫爲 アニメーションディレクター

這個職位的最重要任務就是確保所有的動畫畫面達到設定的質量要求，而且還要將不同製作者製作的動畫畫面連接起來，成爲一個具有連貫性的畫面，通常這些基本畫面是由幾十位製作者創作出來的，雖然作畫監督需要管理者衆多原畫製作者，但這並不意味着作畫監督會強制要求所有的原畫師必須符合自己的畫風，很多著名的作畫監督是允許原畫師在合理範圍內進行一定的自我發揮。

作畫監督可以根據需要對這些原畫師的創作進行一些修改，以保證前後連貫。作畫監督一般自身也是一名比較有個人特色的創作者，作畫監督對於動畫最大的價值就是讓自己所負責的動畫在畫面有所創新跳出傳統和常規侷限，因此作畫監督更像是原畫師們的領路人而非一個看門者。

一些作畫監督有時候會和動畫導演據理力爭，改變之前導演既定的畫面要求，進行相關的調整讓畫面對觀衆能夠更加有吸引力。

但是作畫監督其實是個苦差事，對於動畫製作的整體團隊來說，作畫監督是畫面質量的第一負責人，換句話說畫面出了崩壞第一個背鍋的就是分集作畫監督，對他們來說自己的工作天天都很忙碌，他們需要用最少的時間串聯一話的畫面，並且要讓畫面瑕疵儘量少，有時候動畫監督還會要求作畫監督的畫面能夠不走尋常路，開創新的畫面表現方式。

雖然作畫監督職位平均工資比原畫師們高，但是這種巨大的直接責任和耗費腦力的畫面表現方式創新，讓作畫監督這職位並不是太有吸引力，很多優秀的原畫師其實對於擔任作畫監督是拒絕的。

作畫監督對於畫面的修正並不是直接打回重練，而是根據提交上來的關鍵卡畫面進行角色的臉部表情以及人物線條的修正，保證每一卡的畫面都能穩定

總作畫監督 Chief Animation Director

總作監是整部動畫作品的畫面負責人，總作監的工作是畫龍點睛式的，他要管理自己麾下的分集作畫監督，並進行最終的畫面審覈和修正，日本動畫的產出在不斷增加，但是可用的動畫製作者數量並沒有隨之大幅增長，因此動畫製作行業實際上人手是比較緊缺的。

緊缺的人手導致各個動畫的製作進度或多或少的都存在一些問題，而這些數量有限的人又要馬不停蹄的儘快做完一部動畫，當然是保質保量，日本動畫目前的做環監督系統誕生於 60 年代，對於動畫的單獨一話來說一個作畫監督是足夠的，總作畫監督當時也沒這個職務。

但是在動畫製作數量大幅增加後，總作畫監督這一職位就出現了，因爲快速增加的作品讓作畫監督之間也出現了配合的問題，總作畫監督就是掌握所有的作畫監督，保證每話作畫監督之間最終修正完的畫面不會產生過大的差異，簡單來說作畫監督負責的是每話裏面各個部分的畫面鏈接，而總作畫監督就是負責每話之間的畫面鏈接，相較於作畫監督，總作畫監督操的心更多。

動作作畫監督 Action Animation Director

《幸腹塗鴉》的進食畫面

這些具體的作畫監督職位的誕生，原因很簡單：減少本就已經很忙的作畫監督的工作量以及增加畫面的專業性，理論上所有的動畫製作工作者應該是萬能的，上一個片場畫完萌妹子下一個片場就能畫機甲，不過動畫製作者們總會有自己擅長的領域，這些特殊領域的作畫監督作爲某個領域的大師，可以讓動畫的表現力更上一層樓，像機械作畫監督這種職位應該說是做一部好的機甲動畫必備的職位。

《百合熊嵐》的熊作畫監督而除了這些常見領域的作畫監督外，還有很多臨時起用的某一領域的作畫監督，例如《百合熊嵐》的第 7 話裏面有一位熊作畫監督，《幸腹塗鴉》裏有一位專門的美食作畫監督。

助理作畫監督 Assistant Animation Directors

這個職位就是作畫監督的助手，助手們在作畫監督底下工作，以確保作畫監督完成工作，在電視動畫中這類助理職位並不少見，作畫監督助理是最常見的一種，作畫監督助理的工作永遠是處理動畫作品中最麻煩的部分，誰叫你是助理呢，雖然是助理但成爲一名作畫監督助理是適應作畫監督這一職位最好的方式，說白了就是一個 80 級的大號帶着 1 級的小號練級，大號並不會替你殺怪，反而還會給你引怪，當你克服了所有最麻煩的怪物時，你也就成了 80 級大號了

這類大號帶小號的助理職位除了作畫監督外其他職位也有，而最令人費解的職位大概就是《魔法科高中的劣等生》中出現的總總作畫監督。

佈局 / 佈景 Layouts Animation Director

Layout 的任務是給動畫繪製一個畫面佈局藍圖，Layout 表示的是一個整體空間，其中會標註畫面上的角色或者物品所處的空間相對位置，以及背景和大致構圖。

通常除了負責 Layoutu 的人以外基本上觀衆是不會看到 Layout ，Layout 在分鏡和原畫之間是一個承上啓下的作用，Layout 讓分鏡信息變得更具體讓原畫師能夠正式繪製。

但有的時候作畫監督和動畫監督會親自繪製 Layout ，因爲 Layoutu 直接影響畫面最終的表現結果，監督們自己跑來畫 Layout 也是完全可以理解的，因爲他們希望最終的畫面能夠後自己一開始想象的一樣。

原畫（第一原畫）Key Animation

原畫的職責就是根據 Layout 在角色的運動週期中繪製關鍵幀原畫，通常來說第一原畫負責的是關鍵動作，他們畫出來的是原畫的一個大致框架，能夠讓你意識到最後原畫畫出來的效果會是怎樣的。

原畫師們並不是什麼畫面都親歷親爲，原畫本身是分層的，而不像是 Layout 把所有畫面要素彙集在一起，第一原畫往往處理的是對於自己最重要的角色行動和動畫效果，而複雜的動畫畫面原畫則是要分很多層，而按照計件付費的動畫製作組通常也將每層原畫視爲一個計數單位。理論上一話動畫使用的原畫師數量越多代表這部動畫在畫面上所投入的成本就越高。

數字作畫時代依舊離不開的硬彩鉛

從 2000 年開始日本動畫製作轉入了數字時代，很多人指責離開了紙張離開了手繪基礎的日本動畫逐漸沉淪千篇一律，然而這個指責是跑偏了，因爲動畫的絕大部分工作依舊還是在紙上進行，不過後期是要掃描進電腦處理，這就是爲什麼當三菱鉛筆宣佈紅色的繪圖鉛筆要停產時日本動畫業界如此驚慌，不過脫離紙張倒是大勢所趨。

龍之子製作的《夜櫻四重奏》全數字原畫陣容

很動畫原畫師們已經開始在平板上進行作畫，但還是依靠手繪，年輕的原畫師更傾向於使用平板繪製原畫，因爲可以大大提升生產力，例如龍之子工作室已經完全接受原畫媒介平板化還希望以此來提升技術和製作效率。

即使是從平板上繪製原畫，也沒有影響到原畫師們的想象力和創作力，當然也會有一些動畫製作者們堅持傳統的賽璐珞動畫製作方法，但新動畫製作者和老動畫製作者和諧共存，讓日本動畫在批量的工業化生產和精緻難以複製的手工生產兩方面都能有所提升

第二原畫 Second Key Animation

第二原畫的角色類似清道夫，當第一原畫製作的原畫線條太過粗糙時第二原畫要進行一些修正和清理工作，根據第一原畫留下來的原畫進行更進一步的原畫補完，在過去十年間第二原畫已經從一個不常見的職位變成了如今動畫製作團隊中不可或缺的職位，成爲了行業標配。

很多動畫製作公司的每週播出的新番中都會配備第二原畫，當然也有行業例外一種是長期播放的子供番因爲製作週期和深夜新番不同，一個就是京都動畫的作品，作爲深夜新番的製作公司京都動畫永遠是行業的異類，在 2000 年到 2005 年京都動畫的職位中還有第二原畫，但現在第二原畫的職位漸漸消失在京都動畫的製作序列中。

觀察京都動畫作品的片尾名單，你依舊能看到很多年輕的原畫師負責類似第二原畫的清道夫工作（那些名字列在旁邊的畫師），京阿尼取消了第二原畫這個稱呼但是依舊讓自己的年輕原畫師從事類似的工作

動畫（Douga） -In-betweens

這個職位和我們所說的動畫作品的動畫不是一個意思，這個職位的名字應該說是一個使意動詞「讓畫面動起來」，說白了這就是爲靜態的原畫填充細微的動作幀讓原畫能夠連貫的動起來，動畫的工作就是在原畫的指導下完成工作。

這個職位難以有什麼個人發揮的空間，通常動畫師拿到的原畫上會充滿了各種註釋，確保最後動畫師產出的動畫畫面能夠符合監督的要求。

但是動畫師的工作和行業地位現在也岌岌可危，目前已經出現了能夠自動填充過度幀的動畫製作軟件，雖然這類軟件距離行業大範圍使用還有段距離，但是在測試中已經表現出了不輸於專業動畫師的水準。

這種自動化有好有壞，雖然動畫師的工資一直是行業最低的水準，如果動畫自動化的話還是能削減一些動畫製作成本，但另一方面動畫師的工作由於大量接觸動畫原畫以及按照監督要求製作動畫畫面，所以對於新人來說這個職位也是迅速積累原畫知識和動畫表現手法的好地方，如果未來動畫師這一行業真的被自動化的軟件所取代，會發生什麼難以預料，但肯定會引起動畫行業的地震

動畫檢查 -In-betweens Check

動畫師的工作要嚴格按照原畫上的備註進行，但並不意味着動畫師出現錯誤不會影響最終播出的動畫質量，動畫師們毀掉一部動畫的方式倒不如說是最多的，例如隨手一滑缺幀、少幀、原畫錯位，動畫的工作要是做爛了比原畫畫崩了還可怕，因爲整部動畫的畫面敘事性和流暢性全部被破壞掉。

這種潛在的恐怖破壞力也讓動畫檢查這個看似中規中矩的職位也需要人來進行相關的檢查防患於未然。

最後讓我們回顧一下一個分鏡從分鏡變成動畫畫面的過程

分鏡繪製完成 分達到原畫

按照分鏡製作者的思路進行畫面佈局確定原畫上的角色和背景位置關係（空間透視）

第一原畫在 Layouts 上再繪製關鍵畫面的原畫此時還是草稿

第二原畫接手第一畫師的原畫後如果線條太粗糙要進行清理和美化工作，之後將繪製完成的原畫送到作畫監督手裏，作畫監督將會審查較上來了每一卡原畫對於不滿意的地方親手修正

同時對於關鍵的原畫作畫監督會對畫面進行一些嚴格的註釋，在交由動畫師負責製作成動畫畫面，最後動畫檢查對製作出的動畫畫面進行審覈。
