爲什麼中國的無神論教育能夠取得成功？

這個問題下有不少關於“中國的無神論人口比例究竟有多少”的爭論，但似乎沒有知友注意到楊鳳崗教授 2007 年做的“中國居民精神生活調查（Spiritual Life Study of Chinese Residents），特地在此補充。總體來說，在 2007 年時，立場接近無神論者的中國人，應該是佔多數的。

首先簡單介紹一下楊鳳崗教授和這個調查。楊鳳崗教授是中國宗教方面很有名的學者[1]，目前任教於普渡大學。調查在 2007 年進行，覆蓋全國 49 個城市，受訪人數是 7000 略多一點。同時預先說明：這個調查包含數百個問題，數據完全公開[2]。感興趣的知友可以自行下載來做更多分析。

表 1 受訪人自己的看法，表格來自宗教數據協會官網（Associations of Religion Data Archives，縮寫 ARDA），下同。由表中可見，不相信上帝 / 神靈 / 鬼魂 / 佛等存在的受訪者比例達 75.2%，所謂的不可知論者只有 12.1%。即使假設缺失的 1609 個樣本全部不是無神論者，有神論者 / 拒絕回答 / 不可知論也佔不到多數

表 2 受訪人父親的看法。不相信上帝 / 神靈 / 鬼魂 / 佛等存在的比例仍然達到 72.3%

表 3 受訪人母親的看法。不相信上帝 / 神靈 / 鬼魂 / 佛等存在的比例仍然達到 70.1%

表 4 受訪人配偶（包含夫妻及同居的男女朋友）的看法。不相信上帝 / 神靈 / 鬼魂 / 佛等存在的比例仍然達到 75.2%

從以上幾個數據可以看出些什麼呢？首先，在 2007 年時，大部分受訪者都不相信有上帝 / 神靈 / 鬼魂 / 佛等存在。同時，我們可以確知她們也不是所謂的不可知論者；其次，大部分受訪者的父母和配偶也不相信有上帝 / 神靈 / 鬼魂 / 佛等存在，比例均超過 70%。幾個問題可以相互驗證。

表 5 受訪人對“鬼魂是否存在”的回答，認爲不存在的比例達到 86.2%。數據無缺失

表 6 受訪人對“祖先靈魂是否存在”的回答，認爲不存在的比例達到 72.1%。數據無缺失

你可能有疑問：雖然很多中國人嘴上說自己是無神論者，但他們不還是經常去掃墓、燒紙錢，擺放牌位？這說明他們肯定覺得祖先的靈魂存在，死後還有來世。作者開了幾十個單獨問題，專門詢問受訪者：上帝 / 神靈 / 天堂 / 地獄 / 靈魂 / 祖先靈魂 / 來世是否存在？絕大部分都回答不存在。

表 7 受訪者對“你信仰什麼”的回答。77.1% 的人什麼也不信

前面問的都是針對神啊鬼啊一類的信念。那麼，是否有可能許多中國人不相信這些，但仍然相信某個宗教，或者其他存在呢？調查直接發問：“你信仰什麼”？此時，接近 80% 的人回答什麼也不信。不可知者只有 0.9%。另外，非常有趣的一點：有 4 人選擇信仰毛主席。

總之，雖然有很多答案說“中國真正的無神論者很少”，但數據不支持這個結論。起碼在 2007 年的受訪者裏面，什麼也不信的佔到了大多數。此外，大家對宗教的利弊，以及儒家 / 教是否宗教的回答非常有意思[4]。比方說，認爲沒有任何好處的有 23.6%，認爲沒有任何壞處的也有 17.9%。

最後，簡單說一下這個數據的優缺點。優點：問卷由專業的宗教學家設計，內容細，有交叉驗證。此外，這幾年可能很難再做類似的調查；缺點，抽樣只覆蓋了 49 個城市，主要集中在東部和中部，西部城市比較少。此外，受訪者中城市人口比例比較高，解讀時應該保持謹慎[3]。
