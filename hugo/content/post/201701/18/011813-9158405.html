最近幾天有不少人表示，自己的 iPhone 在使用過程中突然出現 Apple ID 被鎖的現象。

如果 Apple ID 被鎖，蘋果官方建議到官網（iforgot.apple.com/cn/password/verify/appleid）使用現有密碼解鎖或者重設密碼。如果無法解決，那麼就要聯繫人工服務了。

懂了這些，能省錢省時間

如果撥打蘋果 4006668800 的客服電話解決：

  1. 客服一般會在收到問題後將用戶轉接給高級客服。
  2. 高級客服會引導用戶使用自己的 Apple ID 登錄官網，然後將產生的 PIN 碼告訴客服。
  3. 客服會手動將 Apple ID 進行解鎖。

雖然整套客服流程並不複雜，但最爲關鍵的問題是，由於不明原因，最近這幾天 Apple ID 被鎖定的用戶特別多，撥打 400 電話可能需要等待一個多小時，才能夠等到高級客服爲自己解鎖。

而蘋果官方的 400 客服並不是免費的，會根據電話時長收取話費。等待一個小時所需的話費錢也着實不少。

所以如果你的 Apple ID 被鎖定，愛範兒（微信號：ifanr）建議你直接在蘋果的客服網站（網站鏈接點這裏）上進行登記，選擇自己合適的時間讓蘋果客服直接撥打你的電話。

這樣的話即使仍然需要等待高級客服很長時間，自己的話費是省下了。

另外，AppSo（微信號 appsolution）提醒，要留意自己的案例編號（Case Number）以及郵件裏的信息。

這樣即使中途掛斷電話，再次通話時只要提供相關信息，就不用從頭開始描述問題了。因爲，重新撥打電話能遇上同一個客服大概相當於中頭彩的機率吧……

如何讓賬號更安全？

目前消息顯示，用戶的 Apple ID 被鎖，可能是因爲使用了第三方助手導致 Apple ID 郵箱泄露，被惡意試密碼而造成賬戶被鎖。

另外，如果你使用 QQ 郵箱作爲 Apple ID，那麼賬號也會有較高風險。因爲 QQ 郵箱的用戶名組成都是數字，容易批量生成有效郵箱。而且，QQ 郵箱的覆蓋面廣，使用頻次高，被用作 Apple ID 郵箱的可能性大，因此也更可能被不法分子盯上。

由於目前蘋果官方並沒有對此次 Apple ID 大量被鎖定做出迴應，愛範兒建議讀者爲了保險起見，最好更改自己的 Apple ID 郵箱，並開啓二步驗證。

1. 更改 Apple ID 郵箱

根據蘋果官方建議，更改 Apple ID 需要先在官網（https://appleid.apple.com/#!&page=signin）登錄 Apple ID，點擊「編輯」-「編輯電子郵件地址」就可以輸入你想要的 Apple ID 郵箱。

AppSo（微信號 appsolution）提醒：在更改之前先在設備上註銷 Apple ID，否則更改後可能會不斷彈窗出來要求你驗證。

蘋果會給你輸入的新郵箱發送驗證碼，驗證之後你就能使用新的 Apple ID 郵箱了。

2. 二步驗證

二次驗證指的是在輸入密碼之後，你還需要通過其他方式（如身份驗證器、手機短信、語音電話等）來確認登錄人的身份。

幾乎每個賬號通過你的郵箱註冊的。這意味着一旦主郵箱失守，那麼基本上所有的賬戶都得失守，畢竟很多網站找回密碼只需要一封郵件。所以，一定要給重要賬號開啓二步驗證。

Apple ID 的開啓此功能，需要前往 Apple ID 管理頁面（https://appleid.apple.com/），登入 Apple ID。在「安全」選卡中找到兩步驗證，點擊「開始使用」，回答安全提示問題，然後按照相應步驟完成設置。

開啓兩步驗證，只需驗證一個可信的電話號碼和一個可信的設備，根據引導完成設置即可，流程十分簡單。

最後你還會收到用作安全保障的恢復密鑰，可以在忘記密碼或丟失設備時用密鑰訪問帳戶。Apple 建議把這個恢復祕鑰寫下來放在不同地方。（沒錯，就是寫下來……

如果這還不能滿足你，那麼可以試試雙重認證。

3. 二步驗證的升級版：雙重驗證

iOS 9 以上的 iPhone、iPad 或 iPod touch，還可以開啓「兩步驗證」的升級版——雙重認證。需要指出的是，開啓雙重認證需要先關閉「兩步驗證」，你可以在 Apple ID 賬戶頁面的「安全」部分關掉它。

相較於「兩步驗證」，「雙重認證」：

  • 驗證碼從 4 位變成 6 位
  • 用動態更新的密鑰取代了長期的祕鑰鑰匙串
  • 每次登錄都會給所有信任設備推送消息和登錄位置

你只需在手機上操作 2 步即可完成設置，還能離線生成驗證碼，更加方便。

在手機裏打開「設置」- 「iCloud」，點擊自己的賬號，輸入密碼進入 Apple ID 的設置頁面。之後選擇「密碼與安全性」- 「設置雙重認證」，按照步驟驗證電話號碼即可。

更多關於 iOS 設備的安全設置，可以看 AppSo 的這篇文章。

但是除了這些關鍵設置，我們日常還要養成良好的「密碼習慣」，這樣才能杜絕風險。

日常做好這些，保護賬號安全

拖庫和撞庫是黑客常用的手段。簡單來說，拖庫就是指黑客在拿到網站的數據之後獲取了其中保存的密碼，而撞庫則是使用已有的賬號密碼來批量登錄其他網站。

所以，千萬不要在很多個網站使用同一個郵箱的賬戶和密碼。那麼如何優雅地記住密碼？

比如 QqJingzhe5 和 IfanrJingzhe5 這樣，每次大寫的字母不同即可。

2. 使用密碼管理軟件

使用市場上的一些密碼管理軟件，如 1Password 、Dashlane 之類的，這類軟件就是勝在方便。

一方面它們可以通過瀏覽器插件在瀏覽的時候快速登錄，另一方面它們還可以自動生成無序列的強密碼，從根本上杜絕被撞庫的可能。密碼管理軟件介紹可以看 AppSo 的這篇文章和這篇文章。

另外，還有這些基本工作：

  • 不要在公共計算機登錄敏感賬戶的密碼，因爲你永遠不知道有多少人碰過那臺電腦
  • 記住自己的賬號恢復代碼或者密碼驗證問題
  • 把重要的賬號綁定手機並開啓登錄通知，這樣會在異地登陸的時候獲得提醒

希望能幫助大家更好地保護賬號，不用折騰。
