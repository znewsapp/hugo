看朋友圈，覺得所有人都在霓虹看櫻花。

九爺的老孃說，乖兒子，快帶我去看櫻花。

看櫻花是小事，九爺更擔心的，是老孃的溫飽。

刺身是生的，不吃；

壽司是冷的，不吃；

擼串量太少，吃不飽；

豬排飯吃一次就夠了；

鰻魚季節沒到；

……

這時候，東京 / 大版 / 京都 / 札幌街頭的拉麪店，就會成爲最受大家歡迎的餐館！！

所以，在帶老孃去看櫻花之前，九爺決定，先爲大家寫一篇——

一個完全看不懂日文但想吃口熱的美食愛好者，如何在紛繁複雜的拉麪店派系中選到一碗最稱心如意的拉麪？

選擇一家合適的拉麪店

日本拉麪的種類，較真說起來，那簡直太複雜了！

但九爺有一個比較方便的判斷標準——看門臉兒！

比如，看見招牌上有博多兩個字（或者九州 / 福岡 / 久留米 / 鹿兒島）

你就能知道，這家拉麪的湯底是豬骨的。

看見“東京”字樣

那就是醬油湯底拉麪

如果有“函館”倆字兒

啊，鹽味拉麪歡迎你！

如果是“北海道”呢？

那就是味噌口味。

湯底是拉麪的靈魂。選擇拉麪，最重要的是湯底。

拉麪的湯底由タレ（醬料）和だし（出汁，即俗稱湯頭）兩部分組成。

湯頭，粗略來分，可以分爲三大類：雞骨湯、豬骨湯和海鮮湯，用最簡單的公式表示就是：

雞骨=雞骨

豬骨=豬骨 + 雞骨

海鮮=豬骨 + 雞骨 + 魚乾

雞骨湯最“單純”，也被稱爲“清爽系”湯底。顏色淡且薄，稍濃於人們日常煲制的雞湯。

豬骨湯又叫“白湯”，通常會有濃厚的豬骨香味，口味濃厚。

海鮮湯底常見於關東地區，顏色爲紅褐色，值得一提的是，大多數使用海鮮湯底的拉麪，其實都是在豬骨或者雞骨中混入了一定量的魚乾，並非純粹使用海味作爲湯底材料。

有了好湯頭，還要加入“醬料”，用日本人的話來說，就是“着味”。

所以，你看到的“博多”招牌，就意味湯底是：豬骨湯 +nothing

“東京”招牌，就意味湯底是：豬骨 / 雞骨 / 海鮮 + 醬油

“北海道”招牌，則是：豬骨 / 雞骨 / 海鮮 + 味噌

“函館”，應該是：豬骨 / 雞骨 / 海鮮 + 鹽

是不是已經暈了，我們也可以按照口味來理解：

鹽味：口味最爲清淡，能夠凸顯湯底原味

骨湯：比鹽味稍重

味噌：加了味增的拉麪通常油脂豐富、香濃順口，適合重口味。你可以像日本人民一樣，配米飯食用。

醬油： 醬油系是所有選擇恐懼症的噩夢，因爲日本的醬油也太多了！濃厚醬油、薄味醬油、溜醬油、白醬油、生醬油……所以，如果選擇醬油味那就等着坐享彩蛋吧！

你好，拉麪自助販賣機

當你選定了店面走入店裏，通常會看到這麼一個“龐然大物”，這就是日本拉麪的一大特色——自助販賣機。此時是不是頓時心底發慌？別怕，有了下面這張圖，點餐就方便許多了！

既然進了店，面的口味算是選定了，接下來該說“澆頭”了。在日本拉麪裏，它們叫做配菜。因爲各地區物產的不同以及店主各自的巧思，拉麪的配菜通常沒有嚴格的限定。一般情況下，叉燒肉、筍乾（有的地方也把筍乾叫面碼）、青蔥、味玉（醃溏心雞蛋）、海苔算是拉麪的標配，但是從加強版開始，畫風就開始突變了！木耳絲、玉米、豆芽、蔬菜、魚餅，甚至是豬板油……

▲ 一碗拉麪的標準配菜 © http://takumen.com

當然配菜裏也是有一些規矩的，如果你吃到的符合這樣的標準那是比較正宗的：

叉燒每片肉都要厚度超過 3 毫米；味玉的蛋黃不能熟，也不能流出來，要如同在眼眶裏打轉的眼淚；小蔥是澀味少的九條蔥……

但這些都不是重點，重點是這些在機器上就可以全部實現追加，通過你自己的組合，就可以完成從基本版到加強版的進階！

拉麪配主食纔是當地吃法

大多數人在看到米飯後，都會繼而在販賣機上尋找“菜”的身影，結果當然會是失望的。這種搭配其實經常會作爲貧窮的象徵出現在日劇、漫畫之類的作品裏，所以好面子的人是絕對不會選擇這種吃法的。不過主食配主食，雙重碳水化合物的神奇配搭，在日本的確很常見，什麼餃子配飯，烏冬 / 蕎麥麪配飯……

至於放在拉麪中就更合理了，因爲日本拉麪實在是太鹹啦！偏鹹的拉麪湯跟飯非常搭，等麪湯剩一個底的時候，把白飯扣進去，再追加一點叉燒和菜，又是一條好漢。

除了米飯，有些拉麪店的菜單還會出現煎餃。至於煎餃，日本人是把餃子當菜吃的。如果吃不下米飯和餃子，又怕鹹的妹子可以在一開始跟老闆說一聲“薄味”，這樣也就可吃到口味合適的拉麪了。

替玉，大胃王的專屬

由於麪條被稱之爲“玉”，所以替玉指的其實就是指再加一份麪條。如果沒在販賣機上看到也不用擔心，進店之後一定可以加到，而且也可以先視面量大小稍後決定是否需要添加。

▲ 替玉碗底的回汁 © http://pixnet.net

說替玉就不得不提回汁，回汁是博多拉麪的特色調味品，通常以醬油爲基底，再加入其它海鮮熬製。回汁是替玉的好基友，因爲麪條經過水煮勢必會給已有面湯帶來一些多餘的水分，湯汁本身的味道也會因此變淡，所以回汁就成了替玉環節中必不可少的調味品。但千萬別一上來就澆這個汁，那一定會讓湯變的太鹹。

寫到這，你可能覺得又加面又加菜的夠豐盛了，是不是終於可以放心坐等吃拉麪了？事情當然沒那麼簡單。因爲講究的店還會遞上這樣一份“私人訂製”菜單。口味九爺就不細說了，九爺來說說面！

選擇合適的麪條軟硬度

在大家以爲只有牛排可以煎到不同熟度的時候，日本拉麪（多指博多拉麪）纔是真正的玄幻，因爲它居然有足足七種熟度的選擇！（一般拉麪店只會放 3-5 種）

按照從軟到硬排列，博多面條可以分爲：超軟面→軟面→普通→硬麪→超硬麪→金針→粉落。

最受歡迎前三甲：

軟面やわ（Yawa）：軟面是把拉麪煮到比一般拉麪還要軟一點點的程度，比較像煮到剛剛好的方便麪口感，適合老人、孩子或者本身牙口不好或者食慾不佳的人。

普通（Futsu）： 這是默認的麪條軟硬等級，但你還是會嚐到相對其他麪條依舊偏硬的口感。不過這纔是日本拉麪的精華之所在，如果是第一次嘗試，可以從普通開始。

硬麪かた（Kata）：是比較符合中國人口味的軟硬程度，因爲麪條處於比較硬朗的口感，泡在碗裏也不會糊掉，適合慢慢品嚐。

以上是一般拉麪店都會在菜單上標註 3 個等級的軟硬程度，但其實拉麪店也有如同“星巴克隱藏菜單”一般的存在，那就是更多喪心病狂的軟硬度選擇了……

超軟面やわやわ（Yawayawa）：基本可以參考在家不小心煮過頭的拉麪狀況，雖然還是可以用筷子夾起來，但是除非是牙疼到打滾或者牙齒掉光，還是不建議選擇比較沒有口感的這款。

超硬麪バリカタ（Bari Kata）：超硬麪通常會讓很多女性食客望而卻步，因爲吃的時候明顯會感覺到面在口中碎開的感覺！不過想想感覺也是很刺激呢。

金針ハリガネ（Harigane）：當你以爲超硬麪就是頂峯的時候，金針出現了，別看它是“針”，英文裏把這個等級的拉麪稱爲“steel beam”，也就是鋼筋的意思（這跟針也差太多了）！下鍋 7 秒就撈出，它的硬度可想而知……

粉落とし（Kona Otoshi）：粉落，這個看似最軟妹子的名字，纔是最反差萌的存在。下鍋 5 秒撈出，所以這裏粉落就是麪條生到咀嚼時，可以明顯感覺到麪條上的麪粉在口中撒開的感覺（跟生吃有什麼分別……）。然而更讓人想不到的是它在博多拉麪中被很多高級食客所鍾愛，果然是老饕的冷門口味！

桌上的瓶瓶罐罐怎麼用？

除了販賣機上的收費配菜，博多拉麪的店裏通常還有一些“免費”配菜。這裏有兩個固定搭配，一個是辛子高菜，一份是紅姜。辛子高菜是福岡本地的特產之一，將醃過的高菜細細剁碎後，用芝麻和辣椒碎翻炒，加入調味油就成了這具備了辣、鹹、香口味的小菜。而紅生薑最初被放入拉麪完全就是因爲顏值喜慶，但隨着骨湯越來越濃厚，紅生薑也可以起到解膩的作用。

但注意：辛子高菜和紅姜在拉麪上來之前吃是違反規矩的（還會被鄙視），畢竟拉麪纔是主角嘛。而且大家還有一個相同的默契，就是吃拉麪的時候之前一定要先喝勺湯，倒並不是什麼規矩，因爲日式拉麪的精華就在湯頭，一定要在放入其他調料之前嘗一口“原味”。

除了這兩員大將，大蒜也是不能替代的存在，而且食用方法也是講究的。通常拉麪店會直接擺放壓蒜器，由食客自行將大蒜壓成蒜泥。除此之外辣油、醋、芝麻（多用於鹽系拉麪，可以增香）、黑蒜油等更多調味料也是任君挑選了。

至此，拉麪才終於算是“點”完了。

好好享用拉麪吧！

日本人吃拉麪的正確姿勢是什麼？

答案：他們會邊吃麪邊發出吸面的聲音！

先彆着急說餐桌禮儀，這種吃法可是有科學依據的。日本人這麼吃是爲了面和湯能同時進到嘴裏，這樣就感受到湯汁和麪條的共鳴，進而品嚐到高湯富含的細緻風味。曾經也有吃麪大聲等於對廚師廚藝的肯定的說法，但是照九爺說，最好的方法其實就是：隨便！

吃東西這麼開心的事爲什麼要被條條框框鎖住呢。聽九爺的，快去用你最舒服的姿勢吃拉麪吧！
