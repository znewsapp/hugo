五彩斑斕之喵星人：貓咪花色有哪些？

文章來自機構帳號：萌爪醫生

（閱讀提示：由於遺傳在貓咪的花色中起決定性作用，因此本文會涉及一些遺傳學的解釋，如果你覺得某些地方讀起來像咒文，那麼跳過看圖即可_(:3 )∠)_噠噠噠~）

貓咪的毛色都有哪些，如何分門別類，又是如何稱呼這些顏色的呢~？聽小編爲您娓娓道來~~

首先，和許多其它動物一樣，貓咪的色素細胞能且僅能分泌兩種色素，即真黑素（eumelanin）和褐黑素（phaeomelanin），任何貓咪的任何毛色，都是由這兩種色素及它們的變體所展現出來的~

真黑素主打黑色，其變體還能表現出藍色（灰色）、巧克力色（棕色）、淡紫色、栗色和淺黃褐色；

褐黑素主打紅色，其變體能表現出從奶油色、橘黃色、橘紅色到紅色等不同深度的紅。

通常，除了三花姑娘以外，一隻貓咪只能表達一種色素。

下面我們就來看看貓咪都有哪些豐富多彩的毛色吧~

一、斑貓 / 紋貓（Tabby）

斑貓，也就是身上有條紋或斑點的貓咪~所有的斑貓都有一些共同的特徵，TA 們的面部都會有細細的條紋，眼周一圈淡淡的毛色，額頭位置會出現一個“M”形狀的斑紋~如果你仔細去看斑貓的毛髮，就會發現 TA 們的斑紋是由兩種不同顏色（一深一淺），在毛髮上相間分佈而形成的（agouti hair）~

斑貓是貓咪最原始的野生型態，在人類馴化 TA 們之前，貓咪大多都是這種花色~這樣的毛色能爲 TA 們提供很好的掩護，藏在草叢之中不容易被獵物發現~

然而同是斑貓，在不同貓咪身上表現出來的斑紋分佈和顏色也會有非常大的差異。從【斑紋分佈】上看，斑貓的斑紋可分爲 4 種類型：鯖魚斑、古典斑、點斑、細紋斑。

1、鯖魚斑 -mackerel tabby

顧名思義就是像鯖魚花紋一樣的斑紋啦，也是我們平時常說的虎斑 / 狸花（前方密恐患者請繞行）~

鯖魚斑，圖片來源：http://petinfoclub.com

2、古典斑 -classic tabby

又可稱爲“blotched tabby（寬紋）”，這種斑紋非常寬，在貓咪的身體兩側盤曲而行，是中世紀剛剛興起養貓風潮的時候，西方貴族們最爲推崇的一種花色。

古典斑，圖片來源：http://en.wikipedia.org

3、點斑 -spotted tabby

也很好理解，就是條紋斷裂成了一個個的斑點~如果原型是鯖魚斑，點狀花紋就會更細碎一些~如果是古典斑，則會形成較大的斑點。點斑的一個變體形式能讓貓咪展現出如金錢豹一般的皮毛，也就是我們熟知的孟加拉豹貓（bengal）的花色~

點斑（孟加拉豹貓，bengal），圖片來源：http://animalsadda.com

4、細紋斑 -ticked tabby

是非常特殊的一種斑貓，他們身上的兩種顏色相互混雜，幾乎形成了一種純色，分辨不出斑紋。只有在頭部、四肢才能看出少量斑貓所共同具有的特徵，如額頭的“M”狀斑紋。這種斑貓的典型品種就是阿比西尼亞貓，所以有的時候培育者們也會稱其爲“阿比西尼亞斑”。長毛細紋斑貓的代表是索馬里貓（somali）。

細紋斑（阿比西尼亞，abyssinian），圖片來源：http://vetstreet.com

其實以上四種斑紋並沒有嚴格的劃分，而是逐漸過渡的。在貓咪個體上所表現出來的形狀可能並不會那麼典型，而往往是介於某兩者之間的~

以上是斑紋的分佈，那從【斑紋顏色】上看，又可分爲很多類型~花紋是由底色（較淺的顏色）和條紋（較深的顏色）構成的。底色可以是褐色、灰色、橙黃色、奶油色等；條紋可以是黑色、藍色（灰色）、巧克力色、淡紫色、栗色、淡黃褐色或橙色。那麼常見的組合有以下這些：

  • 棕斑貓（brown tabby）：底色爲褐色或灰色，條紋可以是黑色、巧克力色或栗色。
  • 藍斑貓（blue tabby）：底色爲淺黃色或淺灰色，條紋可以是藍色（灰色）、淡紫色或淺黃褐色。

（以上二者在遺傳學上是以真黑素表達爲基礎的。通常，即使黑色斑紋的貓咪底色呈現出橙黃色，也不認爲是發生了真黑素 - 褐黑素的切換，其依然是真黑素，只是分泌量和排列方式，以及可能存在的紅化修飾基因 -rufousing 使其看上去呈現出黃色。）

  • 紅斑貓（red tabby）：底色爲奶油色，橙黃色條紋。
  • 奶油斑貓（cream tabby）：底色爲較淺的奶油色，條紋顏色略深於底色。

（以上二者在遺傳學上是以褐黑素表達爲基礎的，褐黑素表達相對真黑素是顯性的，並且位於 X 染色體，爲伴性遺傳）

那麼不同的【斑紋顏色】又可以和不同的【斑紋分佈】進行排列組合，舉幾個栗子：

奶油古典斑，圖片來源：http://scottishfoldcatsaustralia.com

紅色古典斑（緬因貓，Maine coon），圖片來源：http://mainecoonspalace.com

藍色點斑加白，圖片來源：http://felinest.com

藍色細紋斑（藍阿比西尼亞，blue abyssinian），圖片來源：http://zazzle.com

棕色鯖魚斑，圖片來源：http://pages.total.net

學會了嗎~？這纔剛剛介紹完一個花色哦~是不是感覺貓咪的色彩真的很斑斕呢~\（≧▽≦）/~~~未來我們會介紹更多花色的貓咪哦！敬請期待~

二、純色貓（solid/self）

如果一隻貓咪全身都是一種顏色，那我們就可以稱其爲“純色貓”。包括：黑色（black）、藍色（blue）、巧克力色（chocotale）、淡紫色（lilac）、栗色 / 肉桂色（chestnut/cinnamon）、淺黃褐色（fawn），以及白色（white）。在這些純色貓咪中，除了白色以外，都是真黑素表現出的顏色，而白色則是全身色素缺失造成的~

你可能要問，同樣是真黑素，爲什麼顏色不一樣呢~這實際上是基因突變的結果~“棕色化基因”位點（B locus）的不同基因型，能讓真黑素呈現出黑色（black）、巧克力色（chocolate）和栗色（chestnut），此三者依次爲顯性→隱性→最隱性的關係；而“淡化基因”位點（D locus）的突變（爲隱性突變，等位基因都是小 d 時，毛色纔會被淡化）又能將以上三種顏色分別淡化成藍色（blue）、淡紫色（lilac）和淺黃褐色（fawn）。因此在上圖中，越靠右下角的毛色，越爲罕見和珍貴呢~

純色只會發生在以真黑素表達爲基礎的貓咪，所以世界上不存在純紅色或純橙黃色的貓咪哦~紅色貓咪（以褐黑素表達爲基礎）都多少會有一些可見的斑紋~

純白色的貓咪和其它純色貓的原理不同，後面會單獨拿出來講述~

三、銀色 / 金色底毛——銀斑（silver tabby）、金吉拉（chinchilla）、陰影色（shaded）和煙燻色（smoke）

之前已經講過，斑貓的斑紋是由兩種顏色在毛髮上相間表達形成的（底色和條紋），而有時貓咪的底色能夠呈現出白色（銀色，silver）或黃色（金色，golden），並且白色 / 黃色延伸到所有毛髮的毛根位置。這種淺色底毛可以發生於任何花色的貓咪，我們可以稱這些貓咪爲“銀色 / 金色的貓咪”。下面以“銀色”爲例來講解，金色同理~

當“銀化”發生於鯖魚斑、古典斑或點斑貓時，可稱爲銀斑貓（silver tabby）：貓咪底色爲純白色，條紋 / 斑點可以是黑色、灰色、奶油色或紅色~比如廣受大衆喜愛的銀虎斑美國短毛貓，從花色的角度描述，就是“銀色古典斑”哦~

銀色古典斑貓 / 銀虎斑（美國短毛貓，american shorthair），圖片來源：http://silvershorthairs.com

左：銀色點斑，右：棕色點斑（歐西貓，ocicat），圖片來源：http://catoninetail.com

當“銀化”發生於細紋斑貓（ticked tabby）時，就出現了金吉拉 / 栗鼠色（chinchilla）或陰影（shaded）色：即所有毛髮的尖端是深色的，下方及毛根都是白色的。金吉拉色與陰影色實際上沒有本質區別，只是相比之下，金吉拉色毛髮尖端的深色部分更少，使貓咪看起來幾乎是純白的，只有星星點點的深色~

銀 - 金吉拉（品種“金吉拉”，實際上是指擁有金吉拉花色的波斯貓，persian cat），圖片來源：vetwest.com.au

金 - 金吉拉，圖片來源：http://ermespersan.unblog.fr

陰影色，圖片來源：http://catnipsum.com

當“銀化”發生在純色貓咪身上的時候，就會出現煙燻色（smoke）。煙燻色的貓咪幾乎是純色的，但在 TA 們運動的時候，卻能若隱若現地看到 TA 們銀色的底毛。

黑煙薰色，圖片來源：http://colorgenetics.info

藍煙燻色，圖片來源：http://imgarcade.com

“銀化”的毛髮

四、白色（white）

貓咪的“白”不能算是一種顏色，實際上是毛髮缺乏色素的結果。在貓咪的基因中，有三種不同的突變都能讓貓咪呈現出白色。

其一，白化基因（Albino white），也是決定重點色（pointed color）的基因（C locus：cc）。

其二，白色基因（Spotting），通常在貓咪身上表現出一定面積的白色斑塊，也就是我們通常說的“加白（with white）”。極端的時候也能讓貓咪全身都變成白色。

其三，顯性白基因（White），可以掩蓋所有色素表型，使貓咪擁有純白色的被毛和藍色 / 橙色 / 異色瞳的眼睛。

當貓咪是純白色的時候，如果不做基因鑑定，很難區分到底是哪種白。不過值得一提的是，後兩種白色有一定機率會導致貓咪耳聾，而白化基因和耳聾則沒有關係。

當貓咪不是純白，只是身上有白花的時候，可以斷定是第二種白。對於不同程度的“加白”，貓咪愛好者們還根據白色面積的多少，給這些貓咪起了許多有趣的名字，比如：

“手套 / 雪鞋（mitted/snow shoe）”，指的是擁有白色爪子的貓咪。

“吊墜（locket）”，指的是胸前有一小簇白色的貓咪。

“鈕釦（button）”，指的是肚子上有數個白色小點的貓咪。

“雙色（bi-color）”，指的是身上約有一半面積是白色的貓咪。

“花斑（harlequin）”，指的是身上大部分爲白色，只有幾個花色斑塊的貓咪。

“梵色（van）”指的是幾乎全白，只有頭上和尾巴是其他顏色的貓咪。

“晚禮服（tuxedo）”，指的是擁有白色的爪子、胸脯和肚子，身體其餘部分都是純黑色的貓咪~

玳瑁梵色（土耳其梵貓，turkish van），圖片來源：http://vetstreet.com

黑白晚禮服（襤褸貓，RagaMuffin Cat），圖片來源: Community For Cat Lovers

五、重點色（pointed color）、貂皮棕（sepia）、貂皮重點色（mink）

終於輪到挖煤的暹羅（siamese）啦\(≧▽≦)/~~重點色實際上是白化基因位點的一種突變形式（Cs），這種突變使得色素的產生對溫度很敏感，低溫產生多，高溫產生少。貓咪的肢體末端，包括面部、耳朵、四肢和尾巴都是體溫比較低的地方，這些部位產生色素就會比較多，因而形成了獨特的重點色。重點色幾乎只存在於暹羅貓，以及與暹羅貓有血緣關係的貓咪之中~

重點色同樣可以和其他花色相互組合，形成各式各樣、豐富多彩的喵星人~比如和純色貓咪結合，就會有黑、藍、巧克力 / 棕（棕重點色又叫“海豹重點色 -seal point”）、淡紫、慄、淡黃褐的重點色貓咪。

淡紫重點色（暹羅貓，siamese），圖片來源：http://catsnkitten.blogspot.com

重點色與斑貓（tabby）結合就會形成帶斑紋的重點色，又稱爲“猞猁重點色 -lynx point”。

紅猞猁重點色（喜馬拉雅貓，himalayan），圖片來源：http://niul.org

之前也提到過，紅貓（以褐黑素表達爲基礎）沒有純色的，一定是有斑紋的，因此紅色重點色的貓咪一定也是猞猁重點色的貓咪，能夠找到 TA 們額頭上似“M”的斑紋~

當然，重點色也可以加白，布偶貓就是典型的例子~

藍猞猁重點色加白“手套 -mitted”（布偶貓，ragdoll），圖片來源：http://wpclipart.com

在決定重點色的同一個基因位點（Color），還存在另一種基因型（Cb），能使貓咪呈現出一種獨特的棕色，我們稱爲“貂皮棕（sepia）”，最典型的品種之一就是緬甸貓（burmese）。這種棕色和真黑素棕化形成的巧克力色用肉眼是很難區分的，此二者原理不同且可以平行發生。

貂皮棕（緬甸貓，burmese），圖片來源：http://burmeseinfo.com

當這個基因位點是貂皮棕和重點色的雜合子時（CbCs），貓咪會表現出“貂皮重點色（mink）” ，典型品種是東奇尼貓（tonkinese）~

貂皮重點色（東奇尼貓，tonkinese），圖片來源：http://animal.memozee.com

當然同樣也可以和其他花色相互組合~

貂皮重點色點斑，圖片來源：charcoalbengalcats.com

六、玳瑁 / 三花（Tortoiseshell/calico）

學霸們堅持住~最後一個啦\(≧▽≦)/~

玳瑁色和本文一開始提到的真黑素（eumelanin）與褐黑素（phaeomelanin）的表達息息相關。表達褐黑素（主打紅色、橙黃色）的基因（O locus）位於 X 染色體上，是一個伴性遺傳的基因，對於母貓，有 2 條 X 染色體，也就是 2 個 O 基因，會有以下 3 種情況：

OO——表達紅色，貓咪全身以褐黑素爲基調，也就是一隻紅 / 橙色的貓咪。

oo——不表達紅色（允許黑色表達），貓咪全身以真黑素爲基調，包括純黑、藍、棕虎斑、淡紫重點色等一些列真黑素的變體形式。

Oo——紅色和黑色的隨機表達，貓咪全身隨機表現出真黑素基調斑塊和褐黑素基調斑塊。也就是我們說的“玳瑁 - Tortoiseshell”啦~

玳瑁，圖片來源：http://www.ebayclassifieds.com

而對於公貓，只有 1 條 X 染色體，也就只存在 2 種情況：

O——褐黑素基調，紅貓

o——真黑素基調，黑貓

這就是爲什麼玳瑁都是小姑娘的原因啦~除非小公貓偶然多獲得了一條 X 染色體（XXY），但這個機率非常非常低，在玳瑁貓中大約只有 1/3000，而且這樣的小公貓是沒有生育能力的~

那麼所謂的“三花”，實際上是玳瑁加白的結果~

玳瑁加白 / 三花，圖片來源：http://it.forwallpaper.com

玳瑁色還可以和其它花色相互組合~比如說，玳瑁色的黑基調部分可以被棕化或淡化，紅基調部分也可以表現出不同的深度，所以原本的【黑 - 橙】可以變成【巧克力 - 奶油】、【藍 - 奶油】等玳瑁花色哦~

藍 - 奶油玳瑁（英國短毛貓，british shorthair），圖片來源：http://pinterest.com

當玳瑁和斑貓組合起來的時候，可稱爲“玳瑁斑 - torbie”~黑色基調與紅色基調的斑紋會相互連續起來~

玳瑁點斑加白（薩凡納貓，Savannah），圖片來源：http://lequoiacats.com

當玳瑁和重點色同時發生的時候，就可以稱爲“玳瑁重點色”~

玳瑁重點色加白“手套”（伯曼貓，birman），圖片來源：http://imgarcade.com
