今天你不知道麥肯錫是怎麼回事，來知乎發帖，等待有人給你答案，你給人家點個贊。

明天你面臨畢業選 offer 的幸福煩惱，你來知乎發帖之餘，可能也會找行業裏的過來人給你意見，請人家吃頓飯。

二十年後，你飛黃騰達，做到跨國集團某事業部一把手時，一樣會有各種不知道，各種想不通。

你手下幹活的人，要麼能力有限，要麼視野有限，還有一些你根本就信不過。

這時候，你願意出幾百上千萬的價格，僱傭金錢能夠僱傭到的最優秀的腦力，來帶你想通、爲你規劃，甚至帶你手下的人做出點初步的樣子來嗎？

取決於你是誰，你面對的事情是什麼，這個問題偶爾會有肯定答案。

這便是我們諮詢婊的飯碗來源了。

（至於這個事情具體是戰略問題還是戰術問題還是什麼奇葩問題真的不重要，重要的是客戶願意以這個價錢來解決該問題）

（落在我手裏面試的候選人，但凡說出“專注於戰略諮詢的 MBB / 專注運營與 IT 諮詢的埃森哲” 這種話來的，無一例外地全部拒了）

其實我覺得 @豆撲 的回答特別好，非常到位，我狗尾續貂，展開解釋一下。

管理諮詢服務的一般是企業的 CXO，或者是 N-1 的領導。這些領導人經常會有很多重要的問題需要回答，但是手下人（因爲能力、知識、時間或是利益衝突的原因）回答不了、回答不好、回答不全面、回答不嚴謹。

這些問題，要麼動輒涉及上億上十億的投資，要麼影響着現有的幾億幾十億的生意 / 利潤，花幾百上千萬來爲重大決策上個保險一般來說都是很值得的投資。

比較常見的，頂層管理者需要回答的戰略性問題包括但不限於：

  • 某某產品的市場到底有多大？增長率是怎樣的？未來增長會集中在哪些客戶羣體 / 地理區域 / 價格區間 / 銷售渠道？要想在這些快速增長的地方佔據優勢，需要哪些投資？需要企業建設哪些能力？
  • 需不需要收購上 / 下游的某個 / 某些企業？如果收購了，能有多大的協同效應？怎麼最大化協同效應？會有哪些潛在風險？怎麼最小化潛在的風險？
  • 現有的組織結構，工作流程，產品組合是不是適合當下和未來市場 / 技術 / 經濟形勢的變化？如果不適合，需要做哪些改變，爲什麼？
  • 我的市場份額大量流失給競爭對手，我的手下找了 100 條理由，到底哪些是真的原因，哪些是藉口？我需要做哪些改變來應對？
  • 市場上出現了新技術 / 新趨勢（e.g.區塊鏈，數字化），我的手下對這些新技術 / 新趨勢也一知半解，這些新技術 / 新趨勢能不能應用在我的企業裏？能夠帶來多大的價值？在未來 3-5 年裏怎麼實施，能最大化這些技術給我的價值？
  • 公司的利潤率相比競爭對手的財報一直偏低，但我手下的每個部門老總都說自己做的比競爭對手好，到底是價值鏈的哪個環節輸給了對手？怎麼能趕上 / 超越對手？

這些問題，都是頂層管理者會經常思考，但是一般在現有組織中不能被很好回答的。

一是沒有足夠的知識 / 能力和人力去做足夠深入和嚴謹的研究，二是由於各個部門 / 環節都不夠中立，作出的回答會有失偏頗，大多數時候兩種情況都存在。

這也是爲什麼，諮詢公司總會說，能夠給即使是最初級的員工很好的職業起點的原因——你從第一天開始需要解決的就是頂層管理者在思考的問題。
