「Farewell, Flash.」感謝你，但這一次是真正的永別。

一年半前，我曾和 Flash 作過一次告別。那一次，Adobe Flash Professional CC 被重新命名爲了 Adobe Animate CC，宣告着 Flash 作爲一個創作工具走到了盡頭。

而最近，通過 Chromium 博客 So long, and thanks for all the Flash 我才得知，Adobe 官博在 Flash & The Future of Interactive Content 一文中，宣佈將在 2020 年底時停止發佈與更新 Flash Player。這一次，意味着 Flash 作爲一個平臺走到了盡頭。

在不少人眼裏，Flash 與 HTML5 是純粹的競爭關係，我們應該爲 HTML5 與 Open Web 標準的勝利歡呼，而將 Flash 狠狠的咒罵在黃泉之下。但其實，大多數人都忘記了，或是從不曾知道：HTML5（嚴謹的來說，其 marketing 含義中所涵蓋的那些 Web APIs），有很大一部分正是 Flash 平臺、Flash 社區對 web 標準做出的貢獻。

正如 Flash & The Future of Interactive Content 所說：

Adobe has long played a leadership role in advancing interactivity and creative content – from video, to games and more – on the web. Where we’ve seen a need to push content and interactivity forward, we’ve innovated to meet those needs. Where a format didn’t exist, we invented one – such as with Flash and Shockwave. And over time, as the web evolved, these new formats were adopted by the community, in some cases formed the basis for open standards, and became an essential part of the web.

當我們（企業、用戶）需要 web 平臺承載包括視頻、遊戲在內的各種富交互內容而 web 平臺本身還不具備這樣的能力時，我們通過給予這個平臺一種新的格式，以滿足大家的需求，這就是 Flash Player，作爲一種私有平臺與瀏覽器插件，卻能一度成爲 web 事實標準的客觀原因。

而時至今日，這些 web 平臺所欠缺的能力，在得到市場與社區的認可之後，逐漸被從 Flash 中吸收與揚棄，成爲了諸如 HTML5 Video/Audio/Canvas、WebGL 這些真正的 Open Web 標準。這時候，這些在誕生之初頗爲創新的，作爲了一種「過渡手段」、「Shim」的私有平臺，便自然而然的，慢慢的不再被需要了。

這並不應該理解爲一種失敗，而應該說，它們「功成身退」了。

ActionScript 3.0，Flash 中的御用編程語言，作爲 ES4 的唯一實現，推動了 ECMAScript 標準的發展，深遠得影響着現代 JavaScript；

Adobe Flex，Flash 平臺的企業開發框架，在今年和 @徐飛 老師聊到時，還一起懷念並認可其相比現代 web 前端 / 客戶端開發在工具鏈、協作、兼容性、UI 組件等方面的先進與成熟；

Adobe AIR，作爲最早借鑑 JRT 將 web 相關技術的 Runtime 植入操作系統或捆綁在可執行文件內的跨平臺開發方案，或許可以視作 Cordova、Electron、NodeWebkit、ReactNative 這些方案的一個前身與成功先例；

Microsoft IE 私有技術 ActiveX 中的 XMLHTTP，作爲 XMLHTTPRequest 的前身，促進了 Ajax 的誕生與 Web 2.0 時代的來臨；

Google Gears 作爲 2008 年時爲了增強 web 應用的瀏覽器插件，其私有 API 分別是 App Cache、Web Worker、WebSQL 等標準或標準未遂的前身；

Cordova/Phonegap 作爲第一個面向移動端的 Hybrid 方案，成爲了 web 開發與移動設備的 polyfill 與橋樑，加速了 Web 平臺 Device APIs 的發展，並與 WebOS、FirefoxOS、Chrome Apps、Windows Runtime Apps 等一同影響了 Progressive Web App 的出現；

Google Extension 中 Background Page 與 Event Page 多年對 web 平臺後臺持續計算的嘗試，直接幫助了 Service Worker 的 API 設計；

Google 的 NativeClient、Mozilla 的 asm.js 對於 web 追逐 native 性能的極致追求，則奠定了 Web Assembly 的誕生……

你看，在這條道路上，Flash 與它的朋友們，其實並不孤單。

「看到你長大了，我也就可以心滿意足的離開了。」

就像是， web 技術發展的必然規律一樣，

而 Open Web 則因此不朽。

我很高興，Google Chrome、Mozilla Firefox、Microsoft Edge 都能這麼寫到：

Flash helped make the web a rich, dynamic experience, and shaped the modern set of web standards.
