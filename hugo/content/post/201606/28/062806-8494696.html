「愛或情借來填一晚 / 終須都歸還無謂多貪」——《曖昧》王菲

「就像蝴蝶飛不過滄海 / 沒有人忍心責怪」——《蝴蝶》王菲

把愛情寫成脆弱風景，雖然心中愁腸百轉，但嘴上不說，是這二位才子的拿手好戲。特別喜愛這些句子？從 QQ 狀態用到 MSN 簽名，再到微博、朋友圈的深夜感言？我想提醒你，你在戀愛中的依戀關係可能有些問題。

依戀理論（attachment theory）是一種心理學、演化、動物行爲學理論。依戀理論認爲，人類從嬰兒時期開始就有與最初的照顧者，也就是父母，保持親密的傾向。而每個人與父母之間建立的依戀關係，會直接影響到長大後的戀愛模式，因爲他們會下意識選擇自己最熟悉的那種依戀關係來與戀人相處。

目前的心理學界，廣泛採用一種成人依戀關係的測量工具——「親密關係經歷量表」（Experiences in Close Relationships Inventory）。其中以焦慮感和迴避感爲核心維度，區分出四種依戀類型人格。

疏離型（Dismissing-Avoidant），低焦慮感，高迴避感。

恐懼型（Fearful-Avoidant），高焦慮感，高迴避感。

迷戀型（Preoccupied），高焦慮感、低迴避感。

安全型（Secure），低焦慮感、低迴避感。

 

一、疏離型

會格外青睞開頭那些美麗歌詞的人，很大比例屬於疏離型。

 疏離型的人表面看起來十分 OK，冷靜，不輕易捲入情感的旋渦，迴避親密接觸，剋制與依戀有關的想法。所以連失戀的時候都聲東擊西，保持十足的美感與剋制。

疏離型的人表面看起來十分 OK，冷靜，不輕易捲入情感的旋渦，迴避親密接觸，剋制與依戀有關的想法。所以連失戀的時候都聲東擊西，保持十足的美感與剋制。

 這是由他們內心的防禦機制強行塑造的。在嬰兒時期，我們每個人都會自然地向父母發出「關注我」、「陪伴我」、「保護我」、「喜愛我」等依戀信號，疏離型的人，往往遭遇過忽視和拒絕。他們將焦慮轉化成一種冷淡的、不在意的態度，反覆訓練自己快速回到「安全」狀態的能力，迴避付出以避免傷害，漸漸陷入一種頑固的情感禁閉。

疏離型典型代表是喬治·克魯尼在「Up in the Air」中飾演的 Ryan Bingham

 

二、恐懼型

還是看我們可能特別中意的歌詞。

「害怕悲劇重演 / 我的命中命中 / 越美麗的東西我越不可碰」——《暗涌》黃耀明

「你的心和眼 / 口和耳亦沒緣份 / 我都捉不緊」——《暗涌》黃耀明

「沒關係你也不用對我慚愧 / 也許我根本喜歡被你浪費 / 就算我再去努力愛上誰 / 到頭來 / 也是白費」——《浪費》林宥嘉

「舊愛的誓言像極了一個巴掌 / 每當你記起一句就挨一個耳光」——《給自己的歌》

從歌詞中你大概也能感受到，恐懼型的人對戀愛關係基本得出的是災難性的評價。消極，認爲失敗是必然的，喜歡逃避問題，不敢主動與戀人發生親密的接觸。

更早陷入愛情，又過早主動離開的，往往是這類恐懼型戀人。在他們的成長期，父母很可能非常喜歡以「因爲你做錯了事，所以要得到懲罰」來施加冷暴力，長大之後，潛意識裏認爲自己不值得擁有無償的親密關係，缺乏爭取的動力和能力。

成年後糟糕親密關係也會造成恐懼型依戀類型，很多人都曾經短期進入過這種狀況，但它並非不可擺脫。

恐懼型典型代表是「春光乍泄」裏張國榮飾演的何寶榮

 

三、迷戀型

接下來我們看看迷戀型的畫風。

「我不要你的承諾不要你的永遠 / 只要你真真切切愛我一遍」——《原來你什麼都不要》張惠妹

「如果你知我苦衷 / 何以沒一點感動」——《如果你知我苦衷》周慧敏

「如果沒有你 / 我在哪裏 / 又有什麼可惜 / 反正一切來不及 / 反正沒有了自己」——《如果沒有你》莫文蔚

「找一個一個多情的 / 癡情的 / 絕情的 / 無情的人來給我傷痕」——《單身情歌》林志炫

迷戀型的對感情很投入，對戀人非常依賴，需要被照顧，充滿不安全感，常常表現出過度控制慾。

在沒有心理醫生專業指導的情況下，人們很難徹底改變自己的依戀類型，但對迷戀型的人來說，最簡單的自我治癒方法是不要太快投入一段感情，尤其是不要去尋找玩疏離的高手。不過回到我們講述過的那個邏輯，迷戀型的人因爲自己兒時與父母的依戀關係可能是不安全的，長大後，還是會下意識地尋找讓自己重複體驗不安全感的疏離型戀人。

迷戀型的人與疏離型的人相愛是一樁經典悲劇，足以創作幾個世紀的傷心情歌。但他們彼此之間最容易產生天然的互相吸引。只有儘可能地意識到自己屬於哪種類型的依戀型人格，保持克服自身依戀類型的警覺，互相坦誠交換需求，努力建立彼此的安全感和信任感，纔有可能創建健康的依戀關係。比如有意識約定彼此行爲的方式和邊界，並有效行動。

他們還要一起共同防禦另一種潛在可能，假性親密關係（irrelationship）。也就是當他們彼此之間的行爲約定並不出於本心、也無法滿足自己真正的需要時，他們可能在共同表演一種親密行爲。與各自迴避風險相比，假性親密關係只是兩個人一起精心構建了表面上安全的關係。

迷戀型典型代表是「被嫌棄的松子的一生」中的松子

 

四、安全型

這種類型的人最好描述：積極樂觀，對於親暱行爲感到舒服，享受與戀人的相處。不懼怕問題，善於自我表達，堅信戀愛的長久。

我們來看看這個類型的情歌歌詞，那真是恩愛秀一臉，小心被虐到…

「不需要別人來教 / 把愛緊緊抓牢 / 這一秒 / 決定擁抱你給的美好」——《你的微笑》F.I.R.飛兒樂團

「手牽手一步兩步三步四步望着天 / 看星星一顆兩顆三顆四顆連成線 / 背對背默默許下心願 / 看遠方的星如果聽的見 / 它一定實現」——《星晴》周杰倫

「我想帶你 / 回我的外婆家 / 一起看着日落 / 一直到我們都睡着」——《簡單愛》周杰倫

「你知道 / 就算大雨讓這座城市顛倒 / 我會給你懷抱」——《小情歌》蘇打綠

「天上風箏在天上飛 / 地上人兒在地上追 / 你若擔心你不能飛 / 你有我的蝴蝶」——《無與倫比的美麗》蘇打綠

「我決定不躲了 / 你決定不怕了 / 我們決定了讓愛像綠草原滋長着」——《有故事的人》周華健

「我能想到最浪漫的事 / 就是和你一起慢慢變老」——《最浪漫的事》趙詠華

除了心理學家之外，安全型戀人是最佳治癒者。無論是疏離型還是迷戀型的人，在與安全型的人交往過程中，都最有可能有效改變自己的依戀類型。

安全型典型代表是「一吻定情」中的相原琴子
