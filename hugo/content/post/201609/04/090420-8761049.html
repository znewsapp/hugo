一個優秀的財務總監應該能做什麼？做到什麼程度？

由於 title 在各個企業裏頭不太一致，如果題主說的財總是財務的一把手，那我爲了減少歧義把“財總”替換爲“CFO”吧。

像其他答案裏頭提到的，很多時候中型或者快速擴張的創業公司對 CFO 的能力要求會有不同的側重，具體看老闆具體需求及企業實際情況：如融資需求，業務複雜程度等。我下面就按照大型上市企業 CFO 的標準（算是最全面標準）說說我的看法：（都是總結圈子裏各位高手的分享，並非單純的個人經驗：））

我總結的是 4+1 的能力框架，也就是說頂尖 CFO 都必備的是這五項能力。先說“4”是什麼：

PS: 因爲每個詳細講都能成本書了，這裏篇幅有限，恕我不展開細述，只是簡單解釋，行內人應該能懂。

一）專業能力

  1. 統籌會計覈算及賬務處理：指導會計團隊把會計覈算搞清楚是一切的基礎（包括收入 / 成本 / 費用 / 資產 / 負債等等)
  2. 財務管控（現金流管理）：領導對內的應收應付、庫存、現金出納等的管理和對外的稅務合規
  3. 財務分析爲企業增加利潤（利潤表管理）：在實現全面預算管理的基礎上，主持財務和業務團隊合作爲企業價值鏈進行分析，從而通過定價策略調整 / 成本優化 / 供應鏈規劃等手段爲企業增加利潤。
  4. 投 / 融資分析爲企業增加市值（資產負債表管理）：主持企業對內的資產項目投資決策和對外的融資項目（債務 / 股權 / 融資租賃）。投融資往往聯繫在一起，如何打通就是這項能力的關鍵所在（每個項目都可以看作是孵化一個小公司）。
  5. 戰略及財務規劃爲企業可持續發展提供目標和路線：主持中長期產品 / 渠道 / 市場戰略的財務及投融資規劃。

二） 組織能力

  1. 溝通能力：這是企業領導層必備的對內（戰略部署 / 指導執行 / 跨部門合作）及對外（客戶 / 政府稅務部門 / 投資人 / 供應商）發揮影響力的基礎
  2. 組織建設能力：培養人才和領導團隊的基礎
  3. 流程 / 制度建設能力：建設高效流程，至少包括：i）以財務決策分析爲目標的 PDCA 流程（計劃 / 執行 / 檢測 / 調整）和 ii）以內控爲目標的財務管控流程。

三）企業家能力

  1. 風險管理：爲企業構建風險保護機制（戰略 / 制度）對抗不確定性（包括下面投資新機會的不確定性）。
  2. 發現商業規律 / 機會的能力：通過宏觀環境 / 產品 / 市場 / 渠道 / 競爭對手的數據分析找到生意機會。有人會說這個交給 CEO/COO/CMO 等就可以了，其實不然。在正規上市企業，如果 CEO 要投資一個新的生意機會 / 進入一個新市場是需要董事會表決的，表決就需要財務數據（中長期的 BP，盈利率，scenario analysis 等）。 所以雖然其他各種總監也能找到機會，但是隻有靠財總認可的財務分析模型（通常也是財務部的人做的）才能夠把各種機會轉化成量化的財務指標。

四） 學習能力

  1. 學習上述能力的能力：上述各種能力的積累九成靠個人，一成靠機遇。個人能否製造和抓住機會快速把經驗轉化爲能力，這本身就是一種能力。
  2. 學習企業 / 生意各個細節的能力：CFO 最基本的要求就是把企業整盤生意用財務報表的形式表達出來的並深刻了解內在邏輯的人。沒有對企業各處裏裏外外（市場規模 / 生產 / 營銷 / 渠道等等）的理解，數字也就只是數字。

最後說“+1”是什麼:

五）以原則爲基礎的決策能力：尤其是在壓力之下能做正確的事情。前面答案好像都沒談到這一點，可能大家覺得太基礎了，不值一提。而我恰恰覺得這是最基本的能力也是最終極的能力（好像很玄，相信有經驗的人能懂），沒有這個，上面的“4”個能力的發揮無從談起。至於什麼是正確的事情，見仁見智，這個是價值觀問題，我就不展開了。
