家庭裝修拆除有哪些注意事項？其中有哪些坑？

購買二手房的朋友，內心深處一定都很羨慕可以直接在毛坯房上動手腳的朋友。

相比光禿禿的毛坯房，裝得稀奇古怪的二手房，還要多經歷一步。

拆除。

可不是你想的那樣，拆除就是幾錘子的事情。

以下是要注意的地方——

一、連承重牆都分不清楚，你就敢砸牆？

牆是不能隨便拆的，大家都知道承重牆不能拆，可是哪面牆纔是承重牆呢？非承重牆就一定能拆嗎？

1、承重牆、剪力牆、非承重牆，傻傻分不清楚

所謂承重牆，簡單點兒說，就是承受樓上重量的牆體。也正是因爲要靠它來承受重量，所以不能拆除。

房子的結構不同，充當承重牆的牆體也不相同。

2、砌體 / 磚混結構

很多老房子，在開始建的時候，大多是直接用磚砌起來的，樓上的重量，主要靠樓下砌起來的牆和柱子來承重。

這種房子，一般被稱爲砌體 / 磚混結構，大多比較矮，一般不超過 6 層。

這種房子，裏面的牆體，絕大多數都是承重牆，動都不要動。

3、框架結構

框架結構的房子，用鋼筋混凝土澆灌成樑柱來承重，大部分的牆只起一個隔離的作用。這種結構的房子，絕大部分的牆體都是可以拆改的，自由度很大~

4、剪力牆結構 / 框剪結構

五六層的房子，能住多少人呢？還是要把房子建高點兒，這時候，就出現了剪力牆結構的房子。

剪力牆結構的房子，鋼筋混凝土的牆體也具有承重的功能。

現在建房子的時候，也會把框架結構和剪力牆結構結合起來，在框架結構的上面，增加一些剪力牆。

這兩種結構（剪力牆結構、框剪結構）的房子，牆體有鋼筋混凝土的承重牆，也有隻是作爲填充的非承重牆。

總的來說：

①矮層的房子，像八九十年代的老樓，大多是磚混結構，絕大部分的牆都是承重牆，不能隨便動。除非牆特別薄（小於 120mm），可能是非承重牆。

②框架結構的房子，絕大部分的牆，都不是承重牆，可以按你的需求拆掉。但純框架結構的房子比較少，主要是別墅多一些。

③剪力牆 / 框剪結構是目前大多數高層建築採用的方式，這種牆就不能簡單判斷是否是承重牆了。

不能從結構上看，最靠譜的，還是看戶型圖，如果戶型圖比較正規，承重牆的牆體一般都填實成黑色。如果還不能確定，最好去物業諮詢一下。

5、除了承重牆，還有那些牆不能隨便拆

還有一些牆也不能隨便拆，比如陽臺邊的矮牆。陽臺兩邊的矮牆，很多都是配重牆，拆除後，陽臺很可能會下沉，所以也別去動它。

除了各種牆不能隨便動，牆上的樑，更是不要隨便動。

6、拆牆的價格

報價方面，根據很多業主的反饋，一般工長的報價，拆除輕體牆在 30-60 元 / 平米左右。

二、牆面地板，怎麼拆？

牆面和地磚，有必要全部都拆除嗎？

爲什麼拆除的師傅說這個牆面不用鏟，拆還要加錢？

鋪地板還可以不用拆地磚？是不是在忽悠我？

1、牆面剷除

先簡單看下牆面塗料施工的結構。

覆蓋在牆體外的，先是膩子，膩子之外才是塗料層。

石膏和膩子一般是用來找平牆面的，在塗料粉刷之前，必須先刮膩子。而室內裝飾使用的膩子，按照是否防水，可以分爲耐水型(N 型)和非耐水型（Y 型）。耐水膩子有更好的耐水防腐蝕性就不用多說了，使用壽命也比非耐水膩子要長一些。

是否要剷除牆面，還是要看具體的情況。

①如果牆面需要貼瓷磚

像廚衛、陽臺這些地方的牆面，後續一般都會涉及到給牆面貼瓷磚，不管牆面使用的是不是耐水膩子，都需要將牆皮鏟乾淨，再貼瓷磚。

②如果牆面不需要貼瓷磚

其他空間的牆面，可能並不會涉及到貼瓷磚，這時候是否剷除牆面，就要看原牆面的情況了。

有很多老房子，牆面使用的膩子，都是不耐水的膩子，這時候就需要把牆皮鏟乾淨，再重新對牆面處理。

新一些的房子，牆面大多使用的是耐水的膩子，這時候如果牆面保存比較完好，沒有發黴，牆面的膩子是可以不用剷除的。

有一些屋主可能需要在牆面貼壁紙，這時候還要看牆面的誤差，如果誤差太大，也只能剷掉原來的牆皮，重新做平整度處理。

③怎麼看牆面使用的是什麼膩子呢？

在牆面上撒點兒水，然後用手指去揉一揉，如果能揉出來白漿，一般就是不耐水的膩子，這種一定要剷除。

也可以試試用剷刀去鏟牆面，如果很容易就能剷下來，一般也是使用的非耐水的膩子，需要全部剷除。

2、地板、地磚拆除

如果地面瓷磚保存得比較好，還可以使用，其實真的不用大費周章的去拆掉瓷磚。

如果準備鋪複合木地板（不需要打龍骨），其實也可以考慮不拆地磚（層高足夠的話），直接把木地板鋪在瓷磚上就行，不過這種做法，要求原來瓷磚的平整度達到鋪貼木地板的需求，而且瓷磚的空鼓率不能過高。

還要注意你的門，因爲直接把木地板鋪在瓷磚上，肯定會讓地面增加一定的高度。增加的高度，很可能會影響到門的開啓，不過，大部分的門，都會被你們換掉啦。

3、牆面、地板拆除價格

報價方面，根據北京地區的業主反饋，如果是不耐水的膩子，牆面剷除一般是 7-8 元 / 平米。

如果是耐水膩子的話，比較難剷除一些，工長可能會要求加價，一般會上浮 5 元左右。

地面方面，拆除地板是 5-10 元 / 平米，地磚 20-25 元 / 平米。

三、其他拆除

除了牆面地面，其他拆除的地方還有很多，不然一車一車往外運的垃圾，都是從哪來的呢。

1、水電

絕大多數的二手房，水電都要重新改造（全改或者斷點改），偶爾有例外。

如果原來的電線滿足你使用的需求，佈線也合理規範、而且也沒什麼老化，其實也可以不用拆，但很多老房子用的電線，甚至還是 1.5-2mm²線徑的鋁芯線，這個就不合適了，現在電器換代升級快，都要用 2.5~6mm²線徑的銅芯線了。

原來的線徑可以從電箱查看，也可以拆開面板查看。如果是斷點改造，一定要注意不要破壞以前的管線。

水管方面，水錶以後的管道，纔是你家的管道，纔是你可以動手腳的地方，水錶之前的是主管道，別動。

如果主管道確實管道老化、鏽蝕嚴重，需要通過物業或者自來水公司進行關閉總閥更換水管，同時需要與樓上樓下都協調好，畢竟會影響單元用水，還可能需要樓上樓下的鄰居配合更換。

如果原水管是 PPR 管材，而且使用狀況良好，沒漏水的情況，也可以不用全部拆除。如果你家的水管還是鍍鋅管、鑄鐵管，還是拆掉吧，這種金屬管材，應該被淘汰了，建議換成現在通行的 PPR 水管。

水錶如果已經老化嚴重，一定記得換個新的，因爲老房子的水錶，所在的位置，大多都會做櫃子。等櫃子做好了，纔想起來拆老舊的水錶，麻煩還費錢。

以及，安全最重要。拆除的時候，一定記得斷水斷電，避免工人觸電，跑水漏到樓下。

2、燃氣

燃氣改造，只能由燃氣公司自己操作。普通工人是沒有權限去拆除或者改造的。如果要拆除燃氣表，直接給燃氣公司打電話，約好時間上門拆除即可。

如果需要改換燃氣管道，在拆除後，可能還需要燃氣公司第二次上門給你重新布管。

3、吊頂

老房子的吊頂，估計沒幾個人能看得上眼的。

所以一般都會拆掉，但拆吊頂的時候，就要把吊頂的東西全部拆掉，不要留下之前吊頂的那些輔助的東西。而且吊頂裏，很可能有很多隱藏的管道，拆除的時候也一定要小心，不要破壞了。

報價方面，根據北京地區的業主反饋，一般石膏板吊頂拆除工長報價 15-20 元 / 平米，廚衛鋁扣板的一般 100-200 左右 / 項。

四、其他需要注意的問題

1、保護工作不能少

如果有所保留，比如想要保留地板，或者是一些還能用的傢俱，那保護工作就必不可少。跟施工方簽好協議，文字說明什麼是需要保護的，什麼是需要拆除的，明確責任。

2、下水道別堵了

拆除過程中，最容易，也最麻煩的地方，那就是地漏 / 下水道。

拆除地漏的過程中，會有很多的泥沙，石塊等等，如果不注意的話，會很容易掉到地漏和下水管道里面，造成下水道的堵塞。

所以從拆舊開始到結束，一定要把地漏堵起來。（下圖只是簡單示意，一定要保護好地漏，不讓進雜物垃圾）

3、找誰來拆

不建議隨便選擇路邊的拆除隊伍，沒保障。目前大部分業主選用的拆舊隊伍，都是施工的工長去找的，因爲工長會負責後面的施工，所以他找的拆舊隊伍，多多少少要好一點兒，不過自己也多把把關，特別是有東西需要保護的。

還看到很多人說，可以找收廢品的來幫你拆，拆的東西歸他，他也不收錢。很多年前估計可以，但現在估計很難這樣玩了，一扇防盜門，收廢品的價格才幾塊錢。
