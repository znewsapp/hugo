藉此機會，也總結下自己這些年的駐外經歷吧 （阿聯酋八月、印度班加羅爾兩個月、埃塞半月）

1.公司地位

常駐國外諸多不便，目前中國人常駐國外爲國內公司工作的情況下，無非就是工程、貿易、銀行、礦產等行業，又以工程行業居多。工程那就廣了，鐵路、路橋、港口、石油、冶金、化學各個行業，每個行業下面都有數以千計的相關工程公司。個人追求安逸的話，當然最好所屬企業是該行業的央企頭牌，且該公司是業主或總包商，這種情況下個人能夠享受到比較優厚的薪水，央企國企之類的也能把社保、食宿、補貼等照顧到位。反觀一些私企，或者省屬國企、或分包商，都會有各種能力短板，要麼食宿太差，要麼工作太辛苦，對於常駐來說，都是特別影響心情和工作能力的因素。

2. 駐紮國家

工程行業大都駐紮第三世界亞非拉美等，鮮有項目位於歐洲北美的。但是如果公司業務遍佈各個大洲的話，那麼你進入公司之前就最好做好充分的準備跟領導討價還價駐紮地點，否者一旦確定對象國家，變更起來就不是那麼容易了。

3. 自我娛樂與自我充實

真實的駐外中方企業一般都是包食宿的，好的企業可以帶自己的廚師，吃的肯定相當不錯，差的企業就千差萬別了，有的工程師還要自己做飯（簡直煉獄）。

說道住宿，基本中方企業都會安排集體住宿，這樣難免一些低級別的員工會安排兩人或三人同屋，這其實本來不該是什麼問題，因爲畢竟年輕人居多都是從大學時代過來的，都應有相互幫助和關懷的情懷。這個地方如果出了毛病，生活工作會受到很大影響，慎重慎重，也會影響到自己在領導心目中的形象。

剩下的就是閒暇時間的分配利用了。客觀說來，大部分駐外工作並非那麼緊張，每天的五點之後都是可以自由支配的，年輕一點的工程師利用這點時間學習下自己的工程專業知識，回國輕鬆考個註冊工程師，簡直不要太簡單（不過要靜下心來也挺難的）。沒有工作壓力的，不想太拼的，就可以開展各種娛樂社交活動啦，紙牌 /DOTA/ 逛街 / 攝影 / 踏青 / 博物館 / 展覽會 / 名勝 / 野餐 / 會餐，可選項目根據所在國家 / 城市而定。

其實比較難的是心態與情緒的穩定，一般在某地駐紮，當地情況好的話，三個月左右就會出現疲憊期，各種滿意度下降，心情、狀態下滑，可能愈演愈烈。當然各個公司的文化不一樣，和諧團結的公司裏面這種情況可能好點，勾心鬥角的公司這種情況簡直能要命了。前面說的豐富你的工作之餘生活就是這個意思，爲了避免你在長期的工作中出現這種精神上的狀況。

還有一個比較困難的是，如何處理同事領導之間的關係。正常國內上班的話，上班你是下屬或者同事，回家你就撒歡去吧。駐外集體生活的話……你的較色會介於同事和兄弟之間，工作上你要努力認真，下班生活中你要一起娛樂共同遊玩結伴出行，這是相當好的體驗，你會發現領導可愛的生活的一面，同事之間也會更加熟稔親密。但是不得不說這之間也有個尺度，畢竟人與人之間都有個一個安全距離，至於具體是多少，那就只能一句話了“因人而異，因公司而變”
