阿里全資收購中天微 力主研發 AI「中國芯」

如何看待阿里巴巴收購中天微？BAT 入局會改變現有芯片業格局嗎？

作爲一個在國內芯片公司做過、也在國際大公司做過操作系統的人，應該有一定的發言權吧？

1，芯片研發和互聯網產品開發不同，互聯網產品講究的是快速迭代、快速響應用戶需求，技術先進性反而在其次，隨着用戶規模的擴大，需要不停地重構。而芯片講究的是“十年磨一劍”，一次流片幾十萬美金就沒了，不能兩週出一個版本……我曾經見過給芯片“飛線”的……也說明了芯片研發的難度。

2，芯片優化是一個持續不斷的過程，09 年，我們拿到一家顯卡廠商出的 ARM 芯片，也是全球第一款集成了 3D 硬加速功能的 ARM CPU，我當時把《極品飛車》移植了上去，在 OpenGL ES 和 DriectX Moblie 之間的轉換費了不少功夫，幸好，最後移植成功，還在工程院 Kick off 上秀了一把，效果堪稱驚豔。有幾家國內公司也準備用這家的解決方案，後來實測時出了問題，芯片太熱……當時用的開發板是電容屏的，必須用手指頭點，測試的小姑娘燙得受不了，測試老大緊急買了一批電容筆……跟芯片廠商反饋，人家之前是做顯卡的，水冷、風冷伺候着，壓根沒考慮過散熱的事……據說散熱問題後來花了兩年才解決，但是錯過了智能手機發展的窗口期，這家後來就發力挖礦了。額外說一句，當時高通的片子各方面都不出色，但是最穩定、毛病最少。我們部門有不少人去了 MUI，後來小米堅持用高通的片子，不知道跟我們的同事有沒有關係？

3，芯片研發，其實是一個非常國際化的過程，想做 ARM，向 ARM 買授權、買參考設計，邏輯設計用 porlog，流片找臺積電。參考設計人人都能買來，但是優化成什麼樣子，都是獨門祕籍了。我們當初測一塊國內芯片，與同頻的 PXA270 做浮點運算速度測試，整整比人家慢了三倍……國內很牛的 X 芯，也是直接拿的 MIPS 的授權，也沒在優化上做什麼。

4，高通之所以牛，就是因爲把 AP 和 base band 一起賣，基帶部分，高通的專利牆……真的是無法逾越的。一塊片子的成本當然遠遠低於兩塊片子，而且降低了開發難度。捷通也是把 WI-FI、藍牙、GPS 綁在一起賣。這種商業模式也不是美國人的發明，最早是由 MTK 發明的……

5，最後說一句，在全球化的格局中，用“躲進小樓成一統”的心態去對待貿易戰是行不通的。其實咱們在標準上做過很多次努力，大家可以搜一下 WAPI，最後成爲了一個笑話……

在對方已經成熟的地方，去做抗爭一定是頭破血流，但是在新興領域的專利積累，會爲我們贏得未來。華爲在 5G 上的積累，就讓通訊技術發展起來了，芯片領域也應該採取相同的策略。比如，現在在芯片領域，有一類芯片，中國企業絕對是獨步天下：挖礦的 ASIC 芯片。挖礦本質上就是哈希運算，ASIC 礦機能用來挖礦，也能用來做作深度學習的訓練啊。所以，比特大陸應該符合芯片企業上市的標準了吧？

以上是吐槽黑歷史 + 開玩笑。接下來，認真建議大家多關注一下人工智能方向的芯片企業：寒武紀、地平線機器人，說不定有一天，美國總統辦公室的刷臉系統，還得用中國芯片呢。

-

先說一下，我只是在吐槽自己的黑歷史，涉及到的公司並沒有貶低的意思了。集中的幾個問題吧：

1，WAPI，之所以敢說，就是因爲它的確是個笑話……國家強推的那兩年逼得手機公司上躥下跳的，某公司的手機不帶 Wi-Fi 功能，後來論壇裏流傳了一個破解工具能夠增加 Wi-Fi 功能。我的第一個反應就是：“這麼神奇嗎？”我們看了一下，就是改了一下注冊表……仔細一想也沒錯，國家不讓我有 Wi-Fi 的功能，我得有藍牙和 GPS 啊？其實是一塊片子……再說市面上也買不到 WAPI 的路由器啊。

不得不提一句，這家手機公司的創始人絕對是一個強人，小米現在頗爲得意的粉絲玩法，其實是他的發明……

2，X 芯，我可沒說是哪家，MPIS 的 CPU 多了，我只是提了一句沒有做優化……沒說別的啊？指令集和參考設計，我還是能分的出來的。有一些故事也是本人親歷的，與移植操作系統有關，細節不方便說。大家如果真的有興趣，請搜一家叫做“意法半導體”的公司。

以上這些話，我的確沒有說出全部的事實，如果是行業內的人，開啓名偵探柯南模式，自然就明白是怎麼回事兒了；如果是行業外的看官，就當我講了一個不好笑的笑話，別跟我糾結指令集了……我前邊也說了，給你一套 Win32 API，你就能複製出 Windows 來啦？

3，你們怎麼就不關心比 PXA270 慢 3 倍的是哪家呢？那個故事更好笑。但是人家現在已經做起來了。芯片行業就是一個持續改進的過程，但是不產業化，永遠沒出路。

另外，我現在已經不在這個行業了，所以才把這些陳芝麻爛穀子拿出來曬一曬（某投資圈大佬出櫃的事，我也不敢說啊）。所以，提到的人和事，應該都翻篇了，大家看看，權當一個樂，就行了。

很多事，我說出來未必是真的，你信了纔是真的，對吧？

中國的芯片現狀如何？

作爲芯片行業逃兵，我只想說，幹這行真不賺錢，典型的菜錢粉心，哪怕你混到金字塔尖，intel 或者 IBM 的頂級工程師收入跟軟件和互聯網行業大牛比也是看着怪可憐的。而且芯片設計在國內基本出了一線是沒工作可找的，那點收入在一線生活真的感覺不值。製造方面略好點，基本都算當地中等工薪收入，過個美滋滋的小日子還是可以的。

早年間這行入行門檻還很高，國內不是 top 高校畢業的基本不要。然而幾年以後本來自視甚高的北大畢業生髮現自己賺的還沒有上了個北大青鳥的鄰居家二狗子賺的多，這怎麼留得住人？

如今入行的門檻基本上二本三本微電子相關專業，看着不是太水有點靈性的都 Welcome on board 了，即使這樣，其實還是不太留得住人。硬件或者芯片工程師都是要常年經驗積累的工種，畢業生三五年就被生活壓垮了不得不去軟件或者互聯網或者金融，技術沒法積累，一個企業也很難設計出什麼優秀的芯片。

當然收入不高的根源是這行業不賺錢，玩家多了大家就只能都喝湯了，芯片行業就是高投入高風險低收益投資週期還超長的一個領域，產品賺錢全靠走量，利潤率比農民種地還低，新玩家要是沒國家兜底，光靠民資，在這個已經紅的發黑的紅海里是撲騰不了幾下的。製造的前期投入是天文數字，新玩家估計十年八年是看不到回本的希望的。無論是設計還是製造，最大的制約還是錢。

希望中興掛了能讓國人稍微清醒點，但也別盲目自卑，我國芯片行業不差，某種程度上自稱全球第二也沒毛病，然而第一段位實在是太高了，如果一心想搞你，那真的是挺難搞的事兒。另外國家不是不重視，2006 年開始的核高基喊了那麼多年，科研院所和高校燒了那麼多經費，扶植了不少國資和民資企業，其實都是有成效的，只是你們太着急了，我們在追趕，但真的不可能像軟件和互聯網那麼快。如今國內的芯片行業總體水平，至少差不多把差距從 20 年縮小到 15 年了。還堅守在這行業的兄弟們，任重道遠，感謝你們負重前行，也相信未來我國的芯片行業會越來越有競爭力和話語權！

中國是有追趕上的機會的，設計方面，在美國各大芯片設計企業工作的中國或者華裔工程師大把。人才方面其實沒有本質上的短板，只要國家捨得氪金，一切都是有可能的。製造方面，芯片製程現在已經逼近了物理極限，領先我們的美國在下一步怎麼走上也很憂鬱，我們的差距越來越小，彎道超車也不是沒有可能。

總之別太沮喪，打雞血的說說，中國人最大的能耐就是能把被壟斷的高成本技術弄成白菜價，如果製造方面率先追上，大幅降低投產成本，設計方面捨得氪金，留住人才。追上甚至超過美帝也都不是夢。

但是消極一點說，這個行業本身有了不小的瓶頸，可能最後真的追上的原因不是我們有多給力，而是對手不玩了。

希望在下個戰場，量子計算上，我國能保持優勢，別再錯過黃金髮展期了~
