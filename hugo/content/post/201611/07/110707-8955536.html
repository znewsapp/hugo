哪些行業會成爲無人機最先規模化應用的市場？

無人機的行業應用其實是現在進行時。

多少人說道無人機先想到的是這個？

// 圖片來源：我要飛的更高——DJI 大疆 精靈 3 標準版體驗

Dji 的確是消費級無人機裏最棒的（沒有之一），但因爲 Dji 過於耀眼，很多人都忽略了無人機其實還有另外一大組成部分：工業級無人機。

之前在另外一個回答裏提到過工業級無人機：

如何評價大疆創新發布的新產品 Mavic（御）？ 

工業級無人機的特點是載重大，續航久——而且需要符合各種行業規範。其中最後一點是最重要的，比如很多工業級無人機都可以支持在海拔 4000m 以上的作業（消費級一般超過 2000m 就不行了）。以及高溫高壓等各種特種環境。當然這也決定了工業級無人機的售價肯定不會便宜。十幾萬算是便宜的，三四十萬都正常。如果用氫燃料的那些奇葩（中性詞）一臺八十萬也正常。

// 無人機真是什麼樣的都有，不都是千篇一律一個樣子。

當然，很多人對於無人機的另一大誤解是認爲無人機全部都是多旋翼的，但實際上無人直升機和固定翼無人機纔是過去很長一段時間內最流行的機器。

先說說固定翼無人機，很多場景多旋翼的速度和續航都不夠。尤其是定時定點巡查這種工作。這時候就固定翼無人機出馬了，比如電力巡線和石油線管道巡查都要依靠固定翼無人機。固定翼無人機的缺點是起飛降落要求高，不過現在有了彈射之後方便不少。

這個是常見的固定翼無人機。

還有充氣的無人機，成本很低。

還有這種大到裏面真能坐個人的。

最後是無人直升機，其實說實話無人直升機纔是幹活最爽的。載重續航都不用擔心。但一般比多旋翼貴。

去過展覽的都知道，無人機從來不標價。爲什麼這個敢標價？因爲實在是太便宜了！這家老闆比較傳奇，別家都要外包的活他自己就能幹，所以價格也低。其他的無人直升機幾十萬算不多的了，上百萬很常見。

看了這些，感覺像不像智能手機之前功能機百花齊放的效果？ @馬前卒 先生之前發過一篇《無人時代》的文章，其實可以多寫寫中國的無人機。非常值得中國人爲之自豪~

不過到目前爲止，工業級無人機還沒出現像 dji 一樣一家獨大的情況。

現在說說應用。

電力無人機巡線絕對個各個領域裏最成熟的一個。08 還是 09 年就開始有公司開始做了。這兩年國家電網和南方電網也很看重這一方面（和很多人想的不一樣，電網並不落伍，很多領導的知識儲備可一點也不少）。原因是因爲傳統的人工巡線根本沒法巡，一天拿着望遠鏡也看不完兩根杆塔。有了無人機之後一個季度巡線一次沒啥問題。

而且國網也有很規範的標準，對無人機的各種要求都以及有規範了。南網稍微弱一點，還暫時沒這麼多規範。

// 不要認爲客戶不懂，現在都明白着呢 XD

一般的流程是固定翼看線路，多旋翼看杆塔。

固定翼因爲速度快，看一圈線路很輕鬆。主要觀察點在於線路下方有沒有施工，有沒有外破。有些時候會搭載激光點雲進行 3d 重建。以便於判斷樹木到線路的距離什麼的。

多旋翼用來看杆塔，熟練的話 20 分鐘一根。主要用來看銷釘、絕緣子、間隔棒什麼的。

無人機在電力領域絕對是一把利器，能想象到當年開着有人直升機手持單反拍攝的畫面嗎？不過現在無人機也並沒有在全國所有省市鋪開，裏面還有很大的市場空間留給後來人。雖然無人機已經可以幫我們減輕很多工作，但是我們依舊需要飛手去各種艱難的地方去拍攝，在此向他們致敬！

btw，因爲絕大部分巡線照片都涉及保密，我就不上圖了。

另外一大領域是石油線管道的巡查。

其實問題點還是那幾個，施工，偷油漏油什麼的。另外和電力相比石油方面對於地理位置信息和空間數據的採集要求會更多一些。一般無人機吊艙會搭載夜視，熱成像和氣體檢測儀等攝像頭上天。

// 圖片來源：AHAB HawkEye ||| 傾斜攝影用的相機。

而且石油線對於傾斜攝影的需求量比較大。而且很多石油的問題不是能通過可見光看見的。所以除了無人機之外，吊艙（理解成無人機的攝像頭就行）的需求量也很大。吊艙相當於是無人機的研究。所以說無人機是航拍的話是對的也是不對的：不能簡單的認爲無人機都用單反或者可見光攝像頭。真正的需求很大一部分都存在在非可見光裏。

btw，這類照片也涉及保密我就不放了。

還有就是警用無人機。

另外一個用途是監督和指揮交通。

這裏講個八卦，各位看官就當看着一樂，真假我不保證：

某消費級無人機拿着產品去做某智慧城市的項目。結果正好趕上省裏領導來視察，說展示展示——結果萬萬沒想到領導北航畢業的懂行，一看是用消費級無人機做的就怒了，再一問該有的功能都沒有。最後讓市裏去換成工業級的。

除了這些，還有很多領域都在用無人機：比如環保，消防或者最傳統的航拍領域。估計肯定有答主會提到農業，所以我在此先不提了。

不過說了工業級無人機這麼多好，其實也是有問題的，後進場的玩家還有很多機會。

之前電網一位領導是這麼說的：所有無人機都在和我們說續航，說載重，但問題是我們根本不關心這些，我們最關心的能不能幫我們解決問題。——這就說到了工業級無人機的痛點：幾乎所有玩家都在比拼續航和載重，但是很少有人真正的去關注行業痛點。目前工業級無人機的很多技術可以說已經很成熟了，差異化的地方不大。現在更關鍵的是去尋找如何真正解決客戶的痛點。
