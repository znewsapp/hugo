迷影威尼斯

文章來自機構帳號：「單讀」

威尼斯電影節一直在致力發掘新導演以及電影小國佳片。比如去年的委內瑞拉電影《來自遠方》以及前年的瑞典電影《寒枝雀靜》等都曾獲金獅。今年的獎項同樣是被一位來自電影小國菲律賓的導演捧走。

《離開的女人》

第 73 屆威尼斯電影節金獅獎

導演: 拉夫·達茲

編劇: 拉夫·達茲主演: Charo Santos-Concio 類型: 劇情製片國家 / 地區: 菲律賓

本片講述一位坐了 30 年冤獄之後最終被證清白的女人的旅程，然而在她出獄之後，卻發現自己的入獄竟和曾經的愛人，一位身處上流社會中的成功人士有關。她的復仇大幕也就此拉開。

整部電影長達 226 分鐘，這也是拉夫·達茲一貫的作風，他曾獲洛迦諾金豹獎的影片《今來古往》有 338 分鐘，《憂鬱症》更是將近 8 小時。本片在電影節放映時儘管也是褒貶不一，有的記者甚至直接詬病時長以及因製作粗糙而使得畫面呈現上不夠精良，但是從故事的表達上來看，導演在將近 4 小時的時間裏不僅抒寫了一部“復仇史詩”，更是勾勒出菲律賓水深火熱的社會狀況。

在菲律賓社會轉型時期，內部一片狼藉，貧富的過大落差甚至開始造成人們心裏的扭曲。本片中女主角的被害正是“變形”的多個後果之一，而她在報復和寬恕之間的抉擇或許纔是這場看不見的戰爭的轉折點，是生存在如此社會中大部分的人最終的選擇。

《夜行動物》

第 73 屆威尼斯電影節評審團大獎

導演: 湯姆·福特編劇: 湯姆·福特 / Austin Wright 主演: 傑克·吉倫哈爾 / 艾米·亞當斯 / 亞倫·泰勒 - 約翰遜 / 阿米·漢莫 / 艾拉·菲捨爾類型: 劇情 / 驚悚製片國家 / 地區: 美國

湯姆·福德跨界之後的第二部影片仍舊驚豔，這位時尚界寵兒似乎在影像的表現上也慢慢形成了自己的風格。本片集合了黑色、懸疑、驚悚等元素，大有類型片的趨勢。在畫面呈現上，大設計師更是操控的十分精良，不管從人物的造型、物品的擺置甚至是出現在鏡頭裏的一草一木都是造型感十足。

但除卻這些，《夜行動物》在敘事上層次十分豐富且線索異常清晰。影片中的女主角蘇珊在丈夫離開後 20 年的某一天意外收到了他的手稿，上面的內容令蘇珊回憶起從前婚姻。伴隨手稿的記敘，蘇珊覺察出曾經的祕密真相，這些現實令她毛骨悚然。

如湯姆·福德自己就影片所說的一樣，女主角蘇珊似乎成爲了我們自己文化的見證者。她身上所表現的人的慾念，甚至是惶恐不安也是如今的現代人身上所存有的直通我們內心的線索，深掘之下，一些人性中黑暗的事物便表露無疑。

《戰爭天堂》

第 73 屆威尼斯電影節銀獅獎最佳導演

導演: 安德烈·康查洛夫斯基編劇: 安德烈·康查洛夫斯基 / Elena Kiseleva 主演: 朱莉婭·維斯托斯卡亞製片國家 / 地區: 俄羅斯 / 德國語言: 俄語 / 德語 / 法語 / 意第緒語

導演安德烈·康查洛夫斯基是編劇出身，曾參與塔可夫斯基的作品創作，如《壓路機與小提琴》、《安德烈·盧布廖夫》等。而他自己成爲導演之後的作品也頗爲精良，上次憑藉《郵差的白夜》獲得了第 71 屆威尼斯電影節評審團大獎，本次的水城之旅也爲他斬獲了最佳導演。

這次的影片中，導演將作爲警察的父親、德國軍官、以及一位救過猶太孩子的俄國女人，三人的獨白穿插進主線集中營的生活當中，通過“僞歷史口述史”的方式鋪開劇情。

戰爭無疑是人類遇到的最大災難之一，防不勝防的來自人的惡意猶如黑暗一般籠罩慌亂之地。最終的天堂到底存在與否，也只是人的一念之間。本片結尾處，導演有意在黑暗中扯開了一個缺口，一道光的到來似乎也讓人尋到了精神上的涅槃。

《野蠻地區》

73 屆威尼斯電影節銀獅獎最佳導演導演: 阿瑪特·伊斯卡拉特編劇: Gibrán Portela 主演: Ruth Ramos / Jesús Meza / Eden Villavicencio / Simone Bucio 類型: 劇情 / 科幻 / 同性製片國家 / 地區: 墨西哥 / 法國 / 德國 / 挪威 / 丹麥

今年的威尼斯電影節最佳導演出現了雙黃蛋。相較於另一部二戰題材作品來說，這部影片更多是情感上的縱慾。生命存在多久，慾望便存活了多深。

影片發生在一個小鎮上，小鎮上枯燥的生活顯然使人們內心壓抑過久，直到一位性感女人維羅妮卡的出現，她引導鎮子上的人們去森林中的廢棄小屋去“解決問題”。片中大尺度戲碼頗爲吃重，甚至上演跨種族的性愛。而人們之所以於此，或許正是因爲林中小屋是一個避難所般的存在，它擋住了成千上萬年的世俗的束縛，讓慾望盡情鋪展。

然而這部影片在想法上與喬納森·格雷澤的《皮囊之下》頗爲相似，在故事啓承階段又與祖拉斯基的《着魔》一致，同時，電影節放映期間評價也是兩極分化嚴重，這部獲最佳導演的影片又是否名副其實。

《愛樂之城》

導演: 達米安·沙澤勒

編劇: 達米安·沙澤勒主演: 艾瑪·斯通 / 瑞恩·高斯林 / 芬·維特洛克 / J·K·西蒙斯 / 水野索諾婭 / 類型: 劇情 / 喜劇 / 愛情 / 歌舞製片國家 / 地區: 美國

電影節放映以後，這部片子就頂起了近年來最好的歌舞片的頭銜。影片在開場 5 分鐘的歌舞橋段起，就定住了姿態。在後面的敘事中更是撐住了這口氣。

片中的女主角平日裏在咖啡廳打工受盡了委屈，在業餘時間參加的試鏡也從未有過音信。男主角則是一位深愛爵士音樂的樂手，每日只靠彈奏樂曲獲得的小費才能勉強度日。就是這樣失意的兩人卻在一段偶遇中相遇並且相愛了。

影片在導演達米安·沙澤勒的掌控下並未出現歌舞與劇情的跳脫以及與觀衆的割裂感，相較於達米安·沙澤勒上一部作品《爆裂鼓手》來說，兩部影片都是選擇了富於理想主義的音樂人來進行塑造，而在本片中，不僅畫面呈現精美、音樂引人熱淚，主人公近乎偏執的追求似乎也給這個輕浮的時代漂亮的一擊。
