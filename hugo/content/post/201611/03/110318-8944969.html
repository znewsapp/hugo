大腦的網絡結構是存儲信息的最佳模式嗎？

寫在前面的話

最近跟幾個背景不同的小夥伴們搞了個 journal club（戳這裏可以看細節內容），會從網絡結構方面來討論神經網絡和人工神經網絡。這個 journal club 的日程目前有五週，萬一以後還會有更多呢，所以叫 The 5-week Plus. 歡迎感興趣的同學一起來探討。

大腦不是全連接

考慮到大腦是一個高度摺疊的網絡結構，以及 pyramidal 神經元的 collaterals 在狹小的空間裏迂迴穿過樹突，直覺告訴我們大腦很有可能是全連接的。

然而電生理數據發現，皮層的神經元網絡連接程度只有 10% 而已。所以問題是，大腦呈現出這樣 10%的連接強度，難道因爲這是最適合信息存儲的設置嗎？或者只是偶然或者進化不全？

信息存儲的兩種形式

對於大腦中信息存儲的方式，做計算神經科學的同行們提出了不少的猜想和模擬，其中有兩個假設和實驗數據吻合而被廣泛接受。一個是物理學中的概念 attractor state，另一個是基於 feedforward network 的形式提出的對 firing sequence 的存儲。

(1) attractor state and engram

物理學的概念就不贅述了（因爲我不是學物理的不想把臉拿出來被打）。這個 attractor state 的簡單例子就是在網絡中，有一小撮神經元的 firing rate 比背景強度更高。再一次祭出舊文，因爲讓我想起了engram的形式。

比如在這個神經網絡中，這些藍色的神經元就是用來標記小鼠在這個環境下的記憶（或者信息）。不難看出，這個假設裏，被存儲的信息是基於特定的神經元 sub-group 的，sub-group 被激活，那麼記憶被提取。

(2) sequence storage and feedforward network

Attractor state 是被更廣爲接受的在 recurrent network 中用來存儲信息的形式。直到 Goldman 提出就算是一個沒有 feedback 的網絡中，也可以有信息被存儲下來。這是難以置信的。或者換句話說，信息因爲原樣傳遞而被保留下來了。

爲了保證信息的如實傳遞，即 firing rate 在不同位置序列上的神經元上保持不變，那麼隨着位置距離 raw inputs 的增加，相應的突觸強度應該增加，從上圖（中）到上圖（下）。這也說明在 feedforward network 中，信息雖然以 sequence 保留下來，然而最終體現在突觸強度上，和學習與記憶的生物基礎是吻合的。

從模型到連接

目前計算神經科學領域的研究，主要方法是提出模型，然後任意選定合適的（簡化的）網絡結構進行探索。這是長期爲實驗科學家所詬病的一點：你 run 出來的結果，跟我們有關係麼？

這一篇 Brunel 的新文章就扛起了大旗，用機器學習的方法改變網絡連接的 connection matrix，基於不同的訓練方式（attractor state 或者 sequence storage），然後將最終學習到飽和的網絡結構與電生理測量數據作比較。如同教科書扔到臉上：你服不服。

大量的弱突觸和較強的雙向突觸

理論推導出來的結果和實驗結果有非常令人驚喜的吻合。

比如，不管在哪種參數條件下，發現突觸強度的分佈都是零突觸居多，並且非零突觸呈現 truncated Gaussian distribution. 即弱突觸居多而強突觸更少。以及連接程度在貼近生物實際的參數範圍內，趨近於 0.1 的水平。上圖 C 的 EPSP amplitude 也是一個突觸強度的指標，灰色部分表示實驗數據，光滑曲線是理論值。

這些都是令人信服的發現，說明我們大腦的網絡結構是有利於信息存儲的，這個 sparsely connected network 不是偶然的結果。
