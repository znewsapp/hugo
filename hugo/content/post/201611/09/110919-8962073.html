保暖內衣的保暖原理是什麼？如何判斷他們是不是真的能保暖？

說一說“保暖內衣”這個詞。

這個詞是中文叫法，其他答案裏貼的 FZ/T 標準也是個紡織行業自己定的推薦標準。在嚴寒的環境下，很多文章都建議使用三層穿衣法，三層即 base layer，middle layer 和 shell layer。Base layer 對應的就是中文裏的“保暖內衣”。

但 Base layer 其實並不主要起保暖作用。最近微博上@畢導 THU 的一篇一個清華博士在供暖前給廣大學子最中肯的建議... 很火。

從文中可以看出，和外層衣物相比，內衣的隔熱性能就是個渣渣。

其實 base layer（保暖內衣）的主要功能是排汗，因爲最外層的防風層通常不太透氣，會防止水分蒸發帶走熱量，那麼貼身的衣服不吸汗會很難受。所以買“保暖內衣”，應該以貼身、吸汗、透氣，外加不會引起皮膚不適爲主要選擇目標，而不是保暖性能，或者廣告裏宣稱的其他功能。除非你只穿着它出門晃盪。

中間層（middle layer）纔是絕熱保溫的主力。

在常見材料裏，羽絨是最輕的，同樣重量比它的效果最好，缺點是比較臃腫。

也可以用一些哺乳動物的絨毛，比如山羊絨（cashmere），馬海毛（mohair），駱駝絨（camel hair），羊駝絨（Alpaca wool）等等來做衣服，一些人造纖維材料比如 Thinsulate 也類似。它們可能在同樣厚度的效果比羽絨更好，但重量就要比羽絨重了。

在 到底什麼材料的衣服是最保暖的？ 這個問題裏有回答貼出了使用 Aerogel（氣凝膠）來做絕熱層的防寒服，這不算是一種常見的的材料，所以這大概是衣物裏用到的保溫性能最好的東西，既輕又薄。

傳熱的原理無非傳遞、對流、輻射，只考慮前兩者的話，真空最絕熱。一般固體的熱傳遞都太快，所以絕熱性能好的材料密度一定都不大。但固體材料的密度低到一定程度之後，材料中空氣的熱對流開始在材料導熱中佔的比重顯著增大。Aerogel 本身可以做到非常輕，但它的絕熱性能之好並不是因爲特別輕，而是因爲孔徑非常小，使得氣體分子的移動模式變成 Ashley Knudsen diffusion，顯著降低了 Aerogel 中空氣的熱傳導。（因爲熱輻射的存在，還有高真空度不容易持久，現有的材料裏，實際可以應用的，大概絕熱最好的應該是真空封裝的 Aerogel。）

外層（shell layer）的主要作用是防風，防水，另外防止內層的水分蒸發過快，帶走熱量。所以外層是否保暖，一是纖維的織法，二是看袖口、褲腿、下襬等地方有沒有鬆緊帶可以收緊，拉鍊或者釦子那一排有沒有多餘的布料，防止空氣流動。下襬如果不能用鬆緊帶（不美觀），那就像大衣一樣做長一些，也可以比較有效。

類似的原理，扎圍巾也可以防止領口的空氣流動。最後保暖也別忘了帽子口罩手套襪子這些。
