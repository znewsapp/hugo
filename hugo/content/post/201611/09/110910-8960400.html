特斯拉取消了其用戶的一項最大福利：免費充電

爲什麼特斯拉宣佈不再完全免費提供超級充電樁服務？

謝邀，從個人理解來談一下 Tesla 爲什麼要在明年開始對 Supercharger 進行改革。

首先 Supercharger 是作爲增值服務在 2012 年才公開的的，對於最早 2009 年預約並於 2012 年交車的用戶來說，Supercharger 相當於是意外的免費的福利。Supercharger 的運作模式是鼓勵 Tesla 車主公路旅行，通過每駕駛 3-4 小時，休息 40 分鐘的方式來充電，趕往下一個目的地，所以其設置最初是集中在高速公路出口，隨着網絡擴展逐漸開始出現在城市中心。

而 2012 年後對於 Supercharger 最初是兩種付費模式 ：

1. 對於 60 kwh 的車，車驗收後需要繳納 $2500 來購買終身免費的 Supercharger 使用權

2. 對於 60 kwh 以上的車型，擁有終身免費的 Supercharger 使用權

但是由於絕大部分車主都會購買 Supercharger 使用權，大約 2013 年之後，所有的 Model S 以及後來的 Model X 都擁有終身免費的 Supercharger 使用權，作爲一個內置福利普及到全系。

今天的這新聞，算是 4 年之後第一次對 Supercharger 的模式進行進一步探索：

Tesla 不想再通過買斷的方式來運行 Supercharger 網絡，因爲這裏面遇到了很多問題：

1. 因爲充電免費，部分充電站出現本地人扎堆的情況：

例如下圖中山景城的充電站，經常需要排隊才能等到位子，是因爲很多人路經山景城嗎？當然不是，只是因爲充電免費，所以很多住在硅谷的買家喜歡蹭電罷了。

2. 充電樁利用率並不均衡：

例如下圖顯示，利用率最高的山景城充電站使用的電力比深圳起碼多了一倍

所以有的地方排着隊充電，有的充電站門可羅雀，但是隨着充電站總數的增加，都會增加運營成本。

4. 爲什麼現在公佈改變？

當然是爲了 Model 3，Model 3 從公佈至今很大一個懸念就是如何付費使用 Supercharger。

其實今年 8 月的時候 Model3 的預約狀態頁面就已經泄漏出來，Tesla 將會使用新的支付系統來應對明年激增的 Model 3 充電需求：

下面的圖中可以看到可以在 Tesla 賬戶裏綁定信用卡，或者預購買電力。實際充電時無需刷卡付費，直接插上充電線便能自動扣款也是個非常人性化的設計。

至於 Model 3 車主，這種用多少給多少的方式個人覺得比一刀切更好：

我們如果進行一個比較：

目前美國一度電的平均價格是 $0.2/ 度，按照過去的 Supercharger 激活費 $2500 來算，就是相當於購買 12500 度電，按照 100 度電跑 550km 估算，這也是近 7 萬公里了。

換言之，如果你天天去 supercharger 充電，要開下 7 萬公里才需要給 $2500 的電費。

類似，換成中國上海的平均電價：¥0.45/ 度，就是 21 萬公里…… 估計車都已經報廢了。

這種用多少給多少的方式不僅有利於疏通 Supercharger 的堵塞，從經濟上說也比花 $2500 買激活實惠。

所以綜上，Tesla 希望通過這種用多少買多少的市場方式來提升 Supercharger 的利用率，儘量減少把 Supercharger 作爲本地人蹭電的使用方式，而回歸到“長途旅行快速充能”的最初願景去。

由於 Supercharger 網絡始終是 Tesla 最大的賣點之一，所以 Tesla 也特別說明新車每一年有免費的 1600 公里的額度，同時該收費不以賺錢爲目的，更多是補貼成本里面的電力消耗成本，即便是你依然把 Supercharger 作爲主要充電方式，價格應該也比加油便宜，所以個人覺得還是合理可行的。

最後既然這是一項削弱福利的行爲，Tesla 把執行日期提前兩個月通知，交貨日提前 6 個月通知，這比上次突然宣佈隔日就開放 AP 2.0 要好一點，目前的 Model S／X 潛在買家還有時間考慮是否把“終身免費的 Supercharger” 作爲一個賣點來考慮，而作爲老車主，我是比較欣慰 Tesla 依然堅持自己永久免費的承諾，畢竟同樣一筆車款在 2009 年還是 2017 年給 Tesla 的帶來的意義是完全不同的，換另一個角度想這也算是提升二手賣價的辦法了。
