作爲一個營養師，別的廢話就不說了，直接談解決方案

《每日 5.5 元的營養餐》

去掉之前樓主說的方便麪成本，滿足一個月 50 元營養改善預算。

以下是解讀。

（一）烹調方法

對於任何人，從營養角度來說都應當保證每日足量的熱量供應，其次是蛋白質、維生素、礦物質。方便麪除了方便和提供熱量以外，無論如何算不上一個好的食材，經過精細加工其中的維生素已經非常少了，何況蔬菜水果中的蛋白質也可以忽略不計，因此食材方面必須增加高蛋白質食物，而蛋白質如果不經過加熱變性消化吸收利用率會很差，一般食材僅靠熱水漂燙、不徹底加熱也很不安全，因此我建議題主最好買一個電煮鍋，加工方法和食品安全上更容易得到保障，而且 1.2L 的也就 400 瓦 30 元，水煮味道也很小，允許用電熱水壺泡麪的房東沒理由發現得了，不過就不能強求美味了，煮熟了就吃吧。

（二）廉價食材

再來說食材，除了打烊前買打折菜、從電商購買促銷菜、參加免費試吃等渠道，以當前北京新發地市場批發價（每日價格行情 - 北京新發地市場）參考，以下幾種食材還是非常物美價廉的。有的人可能會質疑菜價，我剛剛查了下平時買菜的超市現在可以隨時下單的價格：雞胸肉 8.5 、白菜 7 毛、胡蘿蔔 1.5、大米 3.3、雞蛋 4.9、黃豆 4.3，也並沒有貴太多。

1.大白菜，0.4 元 / 斤，綠葉菜往往營養價值很高，作爲十字花科蔬菜的大白菜營養價值很高，還富含多種抗氧化營養成分。

2.胡蘿蔔，0.7 元 / 斤，含有豐富的胡蘿蔔素，有助於增強免疫力，方便儲藏。

3.桔子，1.5 元 / 斤，水果相比於蔬菜不需要加工，所以維生素 C 流失少，比起蘋果等常見但是維生素 C 極少的水果，柑橘類是維生素 C 大戶，市面上的小葉桔非常價廉物美。

4.雞蛋，4 元 / 斤，完美的蛋白質來源，對於運動量較大的男性，各種臨牀試驗顯示每天多吃幾個也沒什麼壞處。

5.大豆，3 元 / 斤，大豆含有高比例的優質蛋白質，同時富含人體每天所必需的脂肪酸（轉基因大豆油也就 4 元 / 斤）。幹豆買回來只要打碎徹底煮沸就可以放心吃，也可以從市場上直接買滷水豆腐補鈣、豆芽菜補充維生素 C 預防壞血病，都是價廉物美的食材。

6.雞肉，6 元 / 斤，雞肉價格比較便宜，又是優質蛋白質、硒的良好來源，雞胸肉飽和脂肪還很低。

7.大米，2.5 元 / 斤，對於平時食物充裕、隨時有熱量攝入過多的人來說，我們提倡保證粗糧、至少一半全穀物，但是其實精製大米也有着污染物少、低脂肪、碳水化合物和蛋白質容易消化吸收的特點，是一種溫和無負擔的食材。

8.維生素片。以上食物經過合理搭配已經可以滿足宏量營養素的需要了，但是爲了保險，我還是建議購買一些維生素 C、維生素 B 族甚至魚肝油等維生素藥片，一般藥店的櫃檯底部都會有貨，100 片也就幾塊錢。

（三）飲食結構

具體的食材上，我替題主計算了一下，如果按照一個輕體力成年男性的營養需要（中國營養學會. 中國居民膳食營養素參考攝入量:2013 版[M]. 科學出版社, 2014.），可以參考以下搭配，餐次安排看具體情況，也可以做好後隨餓隨吃。

方案 A 關鍵詞：極簡

大約是一斤的大白菜，八兩的米，四個雞蛋，45 克的大豆油、2 克的鹽。在不考慮口味的情況下，先用電煮鍋把米飯做熟，然後加入雞蛋油鹽，煮熟後倒入洗淨切好的大白菜即可。由於沒有水果攝入，鉀嚴重不足。

方案 B 關鍵詞：耐儲存

把上面的大白菜換成胡蘿蔔，做法類似，這套可以省去每天買菜的時間，由於沒有葉菜和水果，維生素 C 不足，記得吃補充劑。

方案 C 關鍵詞：均衡
大米 400g、雞蛋 60g、大白菜 400g、胡蘿蔔 100g、大豆 35g、柑橘 350g、雞胸肉 75g、大豆油 30g、鹽 2g，這套就比較均衡了，經過理論計算，成本 5.5 元，去掉樓主之前幾包方便麪的開銷，符合一個月 50 元營養改善的預算。

（四）最後提醒

1.由於沒有乳製品，這幾套方案中的鈣攝入都不足，建議題主多出去曬曬太陽，通過合成維生素 D 增加鈣的吸收。

2.我看了點贊最多的熱門答案，其中有人說不該吃廉價雞肉，還說翅尖淋巴多，其實淋巴多的是尖翅（囊上腔），這個售賣前應該已經去掉了，現在雞肉只是品種改良使得出欄快，還是值得吃的。

3.有很多人建議吃紫菜，其實紫菜中的碘實在太多了，天天吃會超過正常的營養需要甚至產生風險，按照題主的生活狀態並不推薦食用。

最後我認爲題主可以考慮一下其他的賺錢途徑，比如我上大學時寢室裏一位貧困生每個月只有 150 元的生活費，但他通過每天晚上用其他室友電腦做網絡遊戲代練，幾個月就自己買了一臺筆記本..... 題主可能需要改變一下思路。

祝好！
