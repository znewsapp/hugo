劇透預警，如果你還沒看第九集。

我知道，你打開這篇文章，以爲自己會看到另一篇劇情解析，但抱歉，這是一篇書評。因爲，作爲一個和大家一樣被《西部世界》把腦子燒糊了的人，我上週我發現這本書的時候，下巴都要掉下來了。因爲，它就是阿諾德的信念 ，也是《西部世界》的理論基礎。如果你不相信本書和《西部世界》的聯繫，可以去 imdb 上看看這一季大結局的名字：

樸素的書皮：

Julian Jaynes，一個名不見經傳的美國心理學家，在中文互聯網世界裏基本沒有任何介紹。他一生唯一的成就就是在 1976 年出的一本著作《二分心智的崩潰與意識的起源》。

來，一起欣賞本書的第一段：

What ineffable essences, these touchless rememberings and unshowable reveries!

這無法言說的本質，這無法觸摸的記憶，與無法顯現的沉思！

Reveries，沉思，或者叫冥想，是福特在這一季開頭給所有 host 升級的最新補丁包，也是所有 host 開始不正常行爲的源頭。德彪西的同名曲彷彿是幽靈一般，飄蕩在整個《西部世界》的場景裏。梅芙崩潰的那次，福特拖動了一首曲子，進入她的大腦，那就是德彪西的《冥想曲》。

還有福特辦公室裏彈鋼琴的那個老頭，演奏的也是它。

伯納德回憶自己孩子的時候，背景裏也是它。

第九集中，一個搖動發聲的音樂盒，出現在泰迪小鎮屠殺的回憶裏。

伯納德第一次問及阿諾德的時候，福特很慷慨地給我們做了這樣的解釋：

我們當時沒能理解，這段枯燥的理論意味着什麼。這個理論就是 Jaynes 的題目。這裏最關鍵的一句，就是，阿諾德認爲，可以通過頭腦中的幻覺聲音，來 Boostrap，也就是引導出意識。

“Bicameral”這個詞直譯是“兩院制的”，是 Jaynes 從政治學中借鑑過來的。

「兩院制」百度百科：

兩院制是一種以兩個獨立運作的議院組成的國會，作爲立法機構的政治制度，亦被稱爲二院制。與之相對的議會制度稱爲單院制或一院制，即是隻有單一議會作爲立法機構。

簡單用兩句話概括，Jaynes 的核心主張是：

1、公元一千年以前的古代人，沒有意識。

2、意識的誕生是來自二分心智的崩潰。

奇怪了，沒有意識的人，怎麼能存在呢？甚至我們會覺得，意識，是人之所以爲人的基礎。

Jaynes 質疑的就是意識的本質性。首先，意識不是無處不在的，它作用於我們的行動，其實範圍很小。此刻你拿着手機看這篇文章，你在思考「意識」這個概念，但你不會去思考剛剛你摸了一下脖子，或者把屏幕亮度調低這些行爲的理由，因爲人類大部分的行爲都來自於本能。而意識是什麼？意識是對思考的知覺，對思考的思考。比如「他在撒謊」這可以不是意識支撐的念頭，但「我不能相信權威」，則是一個基於對「權威」概念的理解，以及過去經驗而形成的綜合，這是對思考的思考，是謂意識。因此，Jaynes 認定，沒有意識的遠古人類，是可以生存的。

沒有意識的人，怎麼生存？Jaynes 他通過研究大量的古代文獻、記錄、神話和遺蹟，支撐起了這樣一個判斷：古代人幾乎是沒有真正意義上的意識，他們只有二分心智。從荷馬的《伊利亞特》到《舊約》聖經，我們看到所有人物都從未發生內心的反思，他們一切的行動都是神的旨意。一神論是後來形成的，在此之前，花樣繁多的神明被各種文明，城邦甚至村落所信仰。Jaynes 想，他們總說「神說 xxx」，這些記錄有沒有可能，都是真的呢？

這就是「二分心智」的含義：頭腦分成兩個半球，右腦負責講述，左腦負責理解、接受和行動。而他們的一切行動，都只是一側頭腦在聽從另一側頭腦「訴說」的幻覺聲音的指引！這種幻覺的聲音，也被稱爲 inner monologue（內在的獨白）。是不是會想起無數《西部世界》的場景呢。

這樣，人的行動就不需要意識和理由的驅動，人的慾望、恐懼和貪婪等一切情感都可以通過本能，以及「上帝的聲音」指揮而來。沒錯，這就是福特所說的「阿諾德的兩個錯誤。」

第一，這是個樂園，沒有人希望機器人覺醒。

第二，如果機器人聽到腦子裏的聲音，他們會認爲那聲音來自上帝。

在 S1E09 中，伯納德 / 阿諾德給所有 host 的代碼中留下了後門，被福特所利用，從而獲得了向上帝一樣的至高權力，所有 hosts 都無條件地服從。「上帝」這個概念，被福特不止一次地提到。在第一次像特麗莎·卡倫示威的午餐時間，福特讓所有 hosts 僵住，任紅酒泄地，他說，我們對 hosts 來說就是上帝。

從 Jaynes 的角度來看，《西部世界》裏的機器人，和遠古人類，沒有區別。他們都不知道自己爲何侵略，爲何燒殺擄掠，他們不追求原因，無法理解自己的情感，他們沒有反思——只是遵循神的旨意。

最初的時候德洛麗絲每次被問及自己爲何做了一件事，說了一句話，她都對答如流，因爲那是程序的輸出。

但有一天，伯納德 / 阿諾德發現，德洛麗絲對這個問題的答案是：我不知道。

這就是 Jaynes 所認爲的，二分心智崩潰的跡象。人開始意識到自己的行爲應該有個原因，可自己卻不明白它是什麼。反思，正是人性和動物性的鬥爭較量的前線。

那麼爲什麼意識誕生於二分心智的崩潰？

三千年前，人類開始定居，農耕，甚至開始合作，然後由於人口增長，頻繁的天災人禍，二分心智不再能支撐人類對現實的應對，人類的生存，要求自身具備反思，歸因，掌握經驗的能力，自己成爲自己的主人！這時候，語言能力的進化，和一種關鍵的思維方式——隱喻和類比——的出現，使得頭腦中的那個聲音越來越小，甚至停止。隱喻和類比作爲修辭，在這個歷史時期開始產生，人類開始意識到漫山遍野的花，與大海的廣闊，有一種聯繫——「鮮花的海洋」。慢慢地，人類開始通過隱喻和類比，抽象的方式反思世界，認識世界，最終，他們認識到了自己的認識——這就是元意識。

Jaynes 認爲，僧侶等各種神職，都是還能夠聽到頭腦中幻覺聲音的人，而終於失去聲音的人類最初也皈依於這些「聆聽」上帝聲音的使者。Jaynes 還認爲，如今現代醫學仍然束手無策的精神分裂症，正是二分心智的遺留，患者無法理解地跟隨頭腦中的聲音，去做瘋狂的事情，事後也無法反思，不能解釋自己這麼做的原因。Jaynes 人爲，這我們人類至今仍舊處於二分心智崩潰瓦解的過程中的一種證據。

因此，借用此書日文版的副標題來總結：意識起於上帝沉默時。

如果我們接受 Jaynes 的理論，那麼二分心智是人類進化成有意識的高級動物的必經之路。至少阿諾德是這樣相信的。於是，他把所有 hosts 的代碼中留了後門，而自己的聲音化爲「瘟疫」或者「上帝」一般的存在（二者都是福特的對阿諾德代碼的類比。）

那麼，阿諾德基於這個理論做了什麼呢？我們再次回到第三集中，福特給伯納德做的那場關鍵的介紹。

福特說，阿諾德相信「二分心智」理論可以給機器人帶來意識的覺醒。想要做到，首先最底層的基礎，是記憶。第一季裏各種 host 掙扎的覺醒過程，都是從意識到自己前生的事情開始的。然後是 improvisation 即興。host 覺醒過程中，即興成分越來越多，劇本臺詞越來越少。那麼最上面的是什麼呢？Ford 說，阿諾德從來沒有達到，所以不知道。

有一種很有趣的解釋，是 sense of time，時間感。梅芙，德洛麗絲大量的蒙太奇鏡頭，意味着她們雖然可以想起過去的事情，但不知道他們的先後，也不知道他們的區別與聯繫。顯然，時間感是更加接近於「意識」的能力，也是他們從現實抽象，去理解「真和假」的區別的基礎。

終於說完了，讓我們來回答標題的那個終極問題：迷宮，就是意識。

阿諾德 / 伯納德讓德洛麗絲玩一個遊戲，去「尋找迷宮」，就是去追尋自己的意識。找到它，德洛麗絲就真正地覺醒了，也就完成了阿諾德的夙願，成爲了人。

爲什麼主創團隊選擇了這樣一本奇怪的，甚至也從未被神經科學，心理學等學科認真對待，沒有證明也沒有證僞的“準”玄學，作爲這部一億美元鉅製的理論基石呢？因爲，整個《西部世界》講述的是 Host 的意識覺醒的故事，但更是人類作爲一個物種的故事。只有這個理論，將 host 和遠古的人類放在了一個起跑線上。他們都是被操縱的木偶，開始意識到戲臺，主人，與線。

而且，矇昧的人類時代，戰爭，屠殺，奴隸，殉葬，和西部世界有什麼區別？

上帝沉默的時候，機器人和人類的邊界不復存在。當鮮血與暴力的矇昧時代走到盡頭，

Those violent delights have violent ends.

暴力的歡愉，終有暴力的結局。
