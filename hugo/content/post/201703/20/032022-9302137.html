試探就是傷害

* * *

如何試探室友是不是偷我東西的人？

先說態度：絕不姑息。

事情回溯到我大二那年，我前後丟了一千二左右的現金，對一個學生來說，這不是一個小數目，而更令人難以接受的是，這是在一星期之內不斷丟失的。這多半是被什麼人盯上了。

如果被欺負到這個份上還不反擊的話，我真是不會原諒我自己了。

 

大學時，我一直有着良好的持現習慣，每次只取兩三百，一多半充飯卡、再在身上帶一些零錢，所以通常不會有身上攜帶大量現金的情況出現。某次取錢之後也一如既往去買東西吃，剩了九十七在口袋，晚上回寢之後並未在意，第二天出門時發現，口袋裏的錢不見了。

我沒想到會是被朋友偷的，只覺得是自己不小心，讓外賊得手了。

第二天又取了一百，這次剩了九十八在口袋，晚上回寢之後也沒有注意，次日出門時再次發現，錢又不見了。

於是我把這事當成笑話說去給我的朋友聽，空氣裏都是快活的空氣，聽完以後無非是勸我以後出門小心，與我一道譴責校園附近治安差云云。

不過，其實當晚和媽媽電話的時候，她就已經提醒過我，好好注意一下身邊的人了，但我沒放在心上。

 

第四天，社團吃飯，那天晚上，好友 C 與我一道前往。

C 君貌美膚白，爲人灑脫，她與我住在同一棟寢室樓裏，只是樓層不同。那時我與她是很好的朋友，因爲 C 是極有人格魅力的女孩子，我們在社團相識之後，她常常來我寢室，很快就博得我們寢室另外三人的全體好感。我們四人平日裏都不太喜歡有陌生人進入寢室，但這條潛在規則隨着 C 的到來打破了。她的話題總是拿捏得恰到好處，與人親近如魚得水，我們都很喜歡她。

那個時候她欠我和室友一些錢，不過我們也不急，沒有催過。

那天去飯店之前，我先去取了錢，一是因爲收到了班長的飛信，馬上要補交書費，大概是二百塊，二是當晚的飯局大概是需要一些現金，所以我乾脆就取了五百出來。

帶着這五百塊錢呢，我和 C 高高興興地跑去參加社團的聚會，當晚飯吃到一半時，C 君拿了一千四給我，說是還欠我與室友 X 的錢，請我順路把錢給 X 捎回去。我答應下來，並將錢分成九百一沓，五百一沓，放進了包裏，連同之前取出來的五百，這時的包裏統共有了一千九。

那天晚上吃飯的都是社團的核心成員，沒有閒雜人等，大家坐在一個房間裏，我完全放鬆了警惕，把包放在了原地沒有管。那天吃完飯，和另外幾個對喝酒不感興趣的朋友一起出外聊天，彼此相談甚歡，結果回來得太晚，連最後大家結賬的時候都沒趕上。等和 C 一起回了寢室，已是將近熄燈的時間。走到門口時，我忽然想起來她放在我這裏的錢，和 C 說，五百塊還給你，既然你都來了，就自己把錢還給我室友吧，我不代勞了。

當時沒有清點包裏的餘額，實在是很好笑啊。

C 很快接過，進屋與我談笑，臨走前忽然拉着我道，誒誒，我晚上再放五百在你這裏吧？

我有點奇怪，問爲什麼。

她支吾道，啊，明天有個小學同學要來看我，我擔心回不來。

這話回想起來起來其實挺神邏輯的，看同學回不來和放錢在我這裏有什麼關係？但也沒有多問，把錢夾在了書架上，告訴她明天自己來取。C走後，我匆忙趁着最後的幾分鐘洗漱，然後便上牀休息了。

 

第二天是週日，那天我剛好有個心理諮詢師的培訓，課上記着筆記，突然想起來包裏該是還有很多錢還沒有收拾，連忙打開拉鍊，發覺裏面只剩了九百。

我愣住了，哪裏還有心情聽課。

算起來，昨晚自己取了五百，C君還了九百，該是有一千四啊……可是五百不見了。

我知道，如果是外賊，哪裏還會給我留個九百塊放包裏，肯定是一起偷走了。

我都不知道怎麼形容那時的震驚，原來對我下手的，是身旁的人呢。

這不是很蹊蹺嗎？我想起先前丟失的那兩百塊，忽然明白過來……恐怕，根本沒有什麼外賊。

但即便是那時我也沒有想到會是 C。

我第一個懷疑的，是我的室友 Y 君。Y 君在大一下學期曾有一段非常反常的舉動，我和其他幾人常常會發現丟了一個蘋果、一袋麥麗素，會發現洗髮水少了，洗衣粉突然用得有些快。那時我們把這話題視爲寢室中的禁忌，彼此心照不宣地知曉是誰所爲，卻一直沒有去追問緣由。時至今日，這一番推理在我腦中初步成型——一定是 Y 君吧！

我把我的猜想和一個室友說了，無奈對方與 C 君關係太好，隨即漏了口風，C 大概立刻就知道了我在懷疑她和 Y。不過那時我並不知曉，就自己開始試探這兩人了，對 C，我大約是秉承着“公平起見”的態度，實際上並未起疑的。

於是，我挑了個時間微信她說，“哎，我又丟錢了”。

C 的反應十分強烈，先是發來一段冷冷的語音，“你在寢室等我，我一會兒上來。”

我反而愣住了，她很快徑直就衝進我寢室，把我拉了出去，怒氣衝衝地問我，“你一直丟錢是怎麼回事兒啊？”

我反而一時語塞，想了想，把對室友 Y 君的懷疑說了出來。

C 君是如此的義正言辭，使我根本沒有底氣在她面前說，我也懷疑你啊。

她激動地躥騰我報警，表情是那麼地真摯，連對我的訓斥都帶着一點恨鐵不成鋼的遺憾，她說“你知道你這樣就是一包子麼？老這麼丟錢，以後誰還不知道你是軟柿子了？”然後，C 又冷靜下來，告訴我，現在的科技是多麼多麼發達，不能驗指紋兒麼？不能調查取證麼？末了又問，那你現在還有錢麼？

我點頭，有，我寢室裏一直放了一些現金，平時不動，應急用的。

她問，錢還在麼？

我想了想，應該還在吧？那個我可是藏起來了的。

她皺眉，斥道，回去看看！

於是我和她一道回了寢室，頗有幾分無奈地，當面從抽屜裏取出一個盒子，就在打開盒子的那一刻，簡直就見證了奇蹟……那盒子裏的五百塊也沒有了。

於是，我的損失金額，升到了一千二。

 

那一刻的心情是很微妙的，我可以說是出離了憤怒，連最後的一點兒理智也沒有了。我在寢室裏各種撂狠話，其實就是說給 Y 君聽的，C 在一旁幫腔，煽風點火。我現在已經記不太起當時 Y 到底是以什麼表情聽我的那番豪言了，大概是有苦說不出吧。

第二天，我和室友 Y 君攤牌，我說，之前對你的種種行爲一直沉默，但現在你未免太過分了，我不能忍。

她沒有多說什麼，只是說，是我乾的事情，我會承認……我希望你好好想一想。

這話十分誠懇，可惜那時我已經被憤怒衝昏了頭腦，一點也沒有聽進去，不過好在一直保持着起碼的冷靜，從未與外人說起這件事情。

那天下午我去找了輔導員老師，大致說了一下我的遭遇，老師也十分震驚，但無奈丟的是錢並不好找，只能叮囑我以後小心。於是我緊接着又去報警，警察說錢這種東西沒有記號，你又是寢室內部丟的，根本不好取證，我頂多幫你威懾一下，做不了實質的工作。

但……這幾乎是我一個月生活費啊。

於是我冷靜下來。

那天，在微信上，我和 C 君提起我報警的事情，C 君沒有立刻回覆，我也就放下了手機。因爲我的手機長期靜音，所以沒有聽見響動，等到晚上再逃出來一看，發現有三個未接來電，數條短信，數條微信，qq 上也有留言。

全部來自 C 君。

於是和她在 qq 上聊了起來。

開始她問我，你打算咋辦啊？

我回，報警了。

她問道，那警察怎麼說？

我想了想，岔開了話題說，室友 Y 君的行爲實在讓我太失望了，之類。

她順着我的話說了幾句，又問道，那警察那邊打算怎麼辦呢？

我又回覆，輔導員老師看起來根本不想管這種事情，就是想讓我息事寧人嘛。

她繼續順着我的話說了幾句，第三次問道，那警察那邊有什麼消息？

她如此執着地問了我三次警察那邊的態度，我心裏已經有了波瀾，於是告訴她，警察說他們已經處理過好多起類似案件了，在這種事情上很有經驗。

她立刻回覆，你還是不要把事情做得這麼絕吧？你畢竟以後還要和Y君相處啊！你嚇一嚇她得了，備不住她哪天就把錢給還了呢？

我心裏一沉，所有矛頭霎時全部指向 C 君。

 

第二天，我收到室友 Y 君的一份郵件，她在郵件中，將上學期她曾承受的一切變故悉數盡陳，那些事情全部事出有因，而在這番解釋之後，之前所有反常的舉動，就都有了合理的解釋。她說她原本不想將這些事情告訴別人，與我說這些，只是想向我證明，偷錢的事情，確實不是她做的。

那時我心裏已經有了一個計劃，看完了這封郵件，也取得了 Y 君的配合。

那周的週六，我在亞馬遜上訂購了一個外型是鬧鐘的針孔攝像機……貌似是一百六十多還是三百二十多，我孤注一擲非要給自己一個交代，錢不錢的早就不想管它了。在寢室裏，我與Y君彼此約定，依然保持着劍拔弩張的態度，私下裏卻一直保持着聯絡。

爲了安撫 C 君的疑慮，我對她道，這幾日我要關起門來解決寢室內部問題，你就不要再來我們寢室了。她問爲什麼，我說，原先警察還打算管，後來一聽學校是維穩的態度，就說不插手了。

她聽後消停了幾天，並未再與我聯絡，直到週三下午。

那個關鍵的小道具終於來了，我在寢室之中調適許久，學習了一下怎麼用。當晚，我告訴 Y 君，你先不要回寢，我喊 C 到我們寢室坐坐。

然後就給 C 君發了消息：Y 君不知道幹什麼去了，你要有時間就來一趟？我有話要對你們三個說。

C 君很快趕到。

我在寢室中板着臉，嚴肅地說，前幾天，我一直沒有發作，是因爲我在等一個結果，我已經將抽屜裏的東西全部送交派出所，做指紋的鑑定。今天下午結果終於出來了，上面除了我的指紋，還有另一個人的。

C 君反駁道，那不一定啊，你那個盒子說不定我也碰過呢。

我說，不，如果真是這樣，那麼盒子上就會有我，你，和Y君三個人的指紋，但現在，除我之外，上面只有一個人的，就只可能是做賊的那一個。

C 君點頭，那你打算怎麼辦？

我說，這兩天我會再和 Y 君交涉，會旁敲側擊，畢竟，其實我在意的不是錢，我要她一個態度，如果這兩天她要抵賴到底，那麼我們週末就一起去一趟警察局，當場就能出結果。

C 君擊節讚歎，又各種出謀劃策。

我搖頭嘆息，其實這件事情我真的已經很累了，只要 Y 君給我一個臺階下，把偷我的錢還給我，哪怕不用全還呢。只要她有心，我就不追究了。

寢室的另兩人不明真相，各種附和。

於是當晚 C 君回寢之後，又各種 qq，大意只是在確認，是不是隻要收到了錢，就息事寧人，我自然說是了。

次日星期四，一整天的水課，也是釣魚的好機會。

這一整天的時間裏，只有 Y 君一人呆在寢室，我臨出門前打開了攝像，便匆匆離去了。

中午的時候，我忽然收到Y君的一條短信，信中說，“錢只有這麼多，放你桌上了。”

我一時不知道發生了什麼，不久便接到 Y 君電話，她在電話這頭問我，“剛纔 C 是不是拿我的手機給你發短信了？”

我說是，怎麼了？

Y 君說，中午 C 突然到了寢室裏來，說是手機欠費了，要藉手機去打個電話。

……於是我就收到了那條短信。

同時 Y 還告訴我，我桌上多了一百塊錢。

 

我心裏的石頭終於落了地，魚終於咬餌了。下午上課的時候，我又變得心不在焉，一時心血來潮，與這位好友微信聊了起來，我只道，“……到底，還是承認了啊。”

她很快回復我七八條，先是表示震驚，接着表示遺憾，最後又勸我不要再追究。

我想了想，在微信裏問她，我只是想知道一件事……爲什麼是我？

她回道，你啊，傻唄。不過沒事兒，還有我在呢。

我也不記得那一堂課後來是怎麼過的了，總之一下課就去找了輔導員老師，向老師說明了來龍去脈以後，一起去了 C 的學院，將此事始末陳述清楚，並再三重申我一定要追究的態度，最終得到答覆，他們必定嚴肅處理。

又過了一天，我見到了她父母。

父母自然是求情，我也沒什麼好說，心軟下來，把事情的始末告訴了社團裏的一些朋友，請大家各自小心。然後請 Y 君出去吃飯，賠禮道歉，回寢室以後，關上門把事情和大家說了個清楚明白。

最後，與 C 君徹底斷了關係。

後來的嚴肅處理沒有等到，倒是聽到了另一個消息，C 君又去偷了某人的手機，最後被人用定位找到了她出手的那家鋪子，順藤摸瓜又被抓了個現行。

但 C 君現在依然過得很好，大抵是因爲她膚白貌美吧，依然灑脫吧：）

 

看到題主的問題補充，也忍不住補充兩句：

1、我和Y君後來的關係。

在事情發生之後，很長一段時間裏，我們都很疏遠，儘管在一起吃飯、上課、一起去教室、回寢室，但隔閡真實存在。大約花了一年半的時間，我們才漸漸恢復原先的關係。她是很好的人，但因爲我的誤解，我失去了一年半的與她真心相處的機會。

試探就是傷害。

2、我是如何構思出這個計劃的。

要留證據，所以需要道具，錄音筆、攝像，之類，然後買道具。

要想清楚到底要套什麼證據出來，如果僅僅是放個一百塊在桌上看對方拿不拿，就算拍到了對方偷竊的鐵證，也只能證明這一次是她拿的而已，所以必須緊扣主題，設計情境。

根據對方智商，設定計劃細節。指紋鑑定什麼哪裏這麼好做……但凡 C 親自去打聽一下就會懷疑我是不是在誆她，但她沒有。不過就算有也沒關係，畢竟校警承諾需要的時候會幫我威懾，我覺得這就夠了。如果C君過程中起疑的話，就讓警察聯繫她，把驗指紋什麼的事情重複一遍。

3、周圍人的態度

之前口頭上支持着我把真相追查到底的，最後幾乎全部站到了C那邊。

很顯然，在拿到證據之後，C 突然就變成了弱者。

長輩甚至指責我，我身上帶着錢，又對 C 不設防，這根本是在引誘 C 犯罪，所以對這件事情不怪人家，反而是我有重責。

無外乎是“蒼蠅不叮無縫的蛋”、“爲什麼她偷你不偷別人”、“得饒人處且饒人”、“你就不能爲她想想嗎”、“原來你是這樣的人”，云云。

 

以上個人經歷貼在這裏，僅供參考，最後一些經驗想和題主分享。

在這件事情上，除了你自己，很少有人能真正站在你的立場上去思考。學校的第一反應必然是息事寧人；告訴父母，也不過換來幾句無關痛癢的說教和安慰。至於和同學討論……旁人茶餘飯後樂得提起便提起，隨之也不過拋之於腦後不會在意的。如果想做些什麼，第一要義是少訴苦，多行動，很多話說出去了，後面的事也就難辦了。

然後，要考慮清楚自己想要的究竟是什麼，原則是什麼。想要得到什麼就必須付出什麼，沒有犧牲就什麼也得不到。所謂“試探”本身就是對同學關係的一種傷害，如果不想傷及同學關係，還想再好好相處兩年，就不要試探，如果想試探，那就堅持到最後，不要被所謂的“同學情誼”束縛住手腳。

真要追究下去，題主也要爲隨之而來的心理壓力做好準備，你是否有擔起後果的覺悟？即便真的到最後查清了所謂的真相，你也未必能得到你想要的。就像 C 君的盜竊行爲最終還是不了了之，就像我處心積慮終於爲自己討回公道，還是不免要承受“她能做到這份兒上，說明她也不是什麼好人”之類的攻擊。

道阻且長，要想清楚。
