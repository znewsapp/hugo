古老世界和古老時代

人們如何書寫和描繪植物？

沿着吉光片羽和蛛絲馬跡

輕聲接近那些古早年代的植物手抄本

各種奇幻作品裏，巫師或藥劑師人手一本“植物圖譜”是標配；魔法師和煉金術士的私宅中，也少不了幾架子“魔法草藥大全”。從流行爆款《哈利•波特》系列到小衆經典《歷代巫術》（Häxan, 1922），不放兩本植物大冊子壓臺子，根本不好意思跟觀衆 / 讀者打招呼。

霍格沃茨學院內建有專門的草藥溫室，也開設草藥學課程。還記得胖女巫波莫娜•斯普勞特教授和她的曼德拉草嗎？

“草藥學（Herbology）是一門研究魔法與普通的植物和蕈類的學科，相當於麻瓜的植物學。在草藥學中，學生會學習到如何照料與利用植物，並瞭解它們的魔法屬性和應用。”——哈利波特 WIKI
由丹麥導演 Benjamin Christensen 執導的《歷代巫術》（Häxan，又譯“女巫”）於 1922 年上映，是一部關於巫術的準紀錄片，被稱爲“超越時代的傑作”，也是後來巫術 / 女巫題材電影的教科書級作品。

在文學或電影裏拋頭露面的植物書們，大多都有古老原型。它們要麼安然躺在著名圖書館幽深的珍本庫內，要麼被癡情藏書家嘔心搜索、瀝血珍藏，每一部都是幾百歲的老古董。

它們也許不是真的“有魔力”，但卻着實記載着古人關於植物的離奇傳聞、怪異想象。某程度講，它們是穿越時光的信使，再現了迷霧中的歷史。

現存最古老的博物類書籍之一

成了隨後 1500 年間

《藥物論》那不勒斯抄本（‘Naples Dioscurides’，公元 7 世紀）內頁。

所謂“那不勒斯抄本”，即收藏於意大利國家圖書館（Biblioteca Nazionale，位於那不勒斯）的《藥物論》裝飾畫手抄本，該抄本成書於 7 世紀，在《藥物論》1 世紀原著的基礎上進行了一定的編輯修改。

不要被“藥物論”字樣嚴肅正經的書名嚇到，這部書其實很奇（che）幻（dan）。相比作者迪奧弗拉斯圖拗口的希臘名，你也許更熟悉書中近似人形的曼德拉草（Mandrake）造型。

原始版《藥物論》的寫作年代距離今天已有近 2000 年，是流傳至今最古老的自然史 / 博物學類書籍之一。

我是小姿勢開始的分割線

/ 什麼是「 手抄本 」/

在印刷術普及之前的歐洲，人們徒手抄書讓文本得以流傳。這些靠逐字抄寫、逐筆畫圖而成的書卷，被稱爲手抄本（manuscript）。圖書館單部手抄本藏品簡寫作「MS.」，而「MSS.」則是複數形式。

因爲沒有印刷術的加持，那些年代特別久遠的古書，常常發生原版佚失、只有後世各時期手抄本流傳至今的情況。而成書於不同年份的手抄本，在傳抄過程中會發生重新編輯、誤抄漏抄、抄者刻意更改文本等狀況，因此手抄本與原始版本之間，存在着不同程度的差異。

《藥物論》的兩部著名抄本

手抄本除了爲今天的我們保留下許多業已失傳的古書的輪廓，也爲人類貢獻了堪稱超級的精美插畫藝術和字體藝術。

成書於 1 世紀的原版《藥物論》（手寫）已經泯然於時間長河。由於該書在古時擁有強大影響力，也就經常被傳抄。6 世紀的“維也納抄本”（'Vienna Dioscorides'）、7 世紀的“那不勒斯抄本”（'Naples Dioscurides'）是現存較爲著名的遺存。

這些植物或其它自然事物古老圖案，簡單，甚至粗糙；植物最明顯的特徵被表現出來，許多部分被簡化或者被忽視；失誤、錯謬很常見；真實與想象時常混爲一談。它們是人類古早觀察自然、瞭解自然的嘗試，與今天精確到原子的成果相比縱使差距千里，卻有着孩童般的質樸、執著，一分一寸都是純真的好奇。

2 { Tacuinum Sanitatis }
中 世 紀 的 “養 生 雞 湯”


“《健康全書》講述了每個人日常保健所需的六件頭等大事。”

— 《健康全書》魯昂抄本，15 世紀

Tacuinum Sanitatis 精裝版，15 世紀
右側書頁曼德拉草再次出鏡

這部老古董的中譯名有點兒辣眼睛——《健康全書》。它是 11 世紀巴格達醫生伊本•巴特蘭（Ibn Buṭlān）作品《保持健康》（ ‎）的改編手抄本，用拉丁文寫成，配圖涉及植物、動物、礦物和中世紀的日常保健生活🌚。書中記載了各式各樣關於植物和其它藥物的好處壞處，主旨只有一個：教你養生。

今時今日的地攤保健書刊在 Tacuinum Sanitatis 面前應該自慚形穢，因爲跟這部誕生於一千年前的手抄本相比，實在沒什麼進步；當然，Tacuinum Sanitatis 也存在着不少謬論和誤解。

Tacuinum Sanitatis 在中世紀的歐洲和阿拉伯地區極爲流行，因而抄本衆多。不同時代的抄本，在插圖繪製、章節安排乃至文本內容等方面都會有所不同。

左圖爲成書於 14 世紀末 15 世紀初的“維也納抄本”局部，右圖爲成書於 15 世紀中葉的“萊茵蘭抄本”局部，同爲“菠菜”章節插圖。

全書共有超過四十個小節，每節講一種事物。有些小節關於植物，蒔蘿、甜菜、栗子、玫瑰、鼠尾草、葡萄、松果兒等；有些小節關於四季，則是春、夏、秋、冬（書中提及四季的順序爲秋、夏、冬、春🌚）；還有關於食材配料的，如杏仁油、橄欖油、糖；也有關於環境天氣的，如暖水、南風；甚至還涉及織物（羊毛、亞麻）。一起來領教中世紀歐洲最流行養生書的魅力：

3 { The Tudor Pattern Book }
中 世 紀 的 “插 畫 師 寶 書”

“流傳至今的中世紀圖樣書，極爲珍稀

'The Tudor Pattern Book'

“The Tudor Pattern Book”是一本來自 1520 年代英格蘭東安格利亞地區的“圖樣書”（Model and Pattern Book），其中收集的元素包含植物、動物、裝飾性字母，以及各種中世紀日常事物。技法細緻，色澤飽滿，濃濃中世紀風味。目前由牛津大學伯德利圖書館收藏。

'The Tudor Pattern Book' 內頁
天仙子（henbane）、鐵角蕨（hart's-tongue）配一桌大菜
鐵角蕨在歐洲被稱爲“鹿舌頭”
當時的歐洲人認爲天仙子和鐵角蕨都含藥性
這樣草藥＋大餐的搭配，真是意味深長

中世紀畫師們致力於爲抄本繪製豐富瑰麗的裝飾性插圖，圖樣書應運而生。所謂圖樣書，是專門用於收集圖案元素的冊子，在印刷術普及之前，也是全憑手繪流傳。有了圖樣書，畫師有地兒尋找靈感，效率也更高。換句話說，圖樣書頗爲類似於今天的“設計師素材庫”。

由於圖樣書的使用和轉手都很頻繁，這就導致它們折損率極高。因此留存到今天的中世紀版本少之又少、極爲珍貴。至於像“The Tudor Pattern Book”這樣品相超正的滄海遺珠，只能說，“相遇全憑緣分”。
