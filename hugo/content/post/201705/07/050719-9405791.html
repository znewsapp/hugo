如何看待「南極突然出現了一條大裂縫」的新聞？

如何看待呢？還是先吐個槽吧。

其實這個冰架裂縫的新聞去年年底已經熱過一次了，當時 NASA 公佈了這條裂縫的衛星圖，然後國內一票媒體普遍採取的標題是——《南極最大冰架……》

但其實這條裂縫所在的拉森 C 冰架根本不是什麼南極最大冰架，它的面積在南極冰架中只能排到第四位。去年年底，知友 @大地 也專門寫回答吐槽了這個事如何看待南極半島最大冰架出現裂縫？

拉森 C：我絕對不知道，我一個面積第四大冰川，怎麼就成了南極最大了呢？

然而僅僅過了半年，就在昨天中午，新浪科技微博號又帶了一波節奏，這次標題更嚇人——《南極最大冰架……裂縫……把南極大陸一分爲二》

什麼就“南極大陸一分爲二”嘛！如果這條裂縫不可避免的會導致一部分冰架脫落，它所佔的面積也只是拉森 C 冰架的 12% 左右，相對於整個南極大陸而言這個比例就更小了。當然，有朋友會認爲我太較真，畢竟南極半島也是南極大陸的一部分，新聞也沒說是均分成兩塊。但我總感覺你跑了 50 米就到處宣揚“我跑過小半個馬拉松”，也不妥吧……

唉……這些媒體啊，總是想弄個大新聞！

但是我們吐槽歸吐槽，拉森 C 裂縫快速擴大、即將剝落出一座巨型冰山，還真確確實實不是一件小事。

其實拉森 C 上的這條裂縫已經存在了幾十年了，而在拉森 C 長達萬年的生涯中，這樣的冰裂縫也並不罕見。其實這些南極的冰架啊，就是因爲冰川運動，由南極冰蓋上緩慢移動到海里的，它們就像陸地上冰川伸入海中的舌頭，憑藉海水的浮力維持着平衡。那麼有的時候，一些冰架的邊緣，就會因爲海洋的潮汐運動一上一下的撬動而鬆動，剝落下來就成爲冰山；也會因爲極晝期間溫度的回升，被融冰流水沖刷出裂縫，剝落下來成爲冰山；甚至還會因爲地震啊，海嘯啊之類的突發情況被震落下來成爲冰山。但是隻要南極地區大環境是穩定的，那麼這些“損失”掉的冰架，還會因爲陸地冰川的緩慢推動，以及冬季降雪而得到補充，整體來說，它們的體積是會保持一個動態穩定的。

冰架（ICE SHELF）的產生

但是這次拉森 C 的巨大裂縫，卻並非是正常產生的。

我們知道，浸入水中的物體，都會收到一個向上的、與它所排開的水重力相同的浮力。我前邊說過，冰架是冰川伸入水中的舌頭，它要想維持平衡，也需要海水浮力的支撐。那麼隨着海水變暖，拉森 C 冰架位於水中的部分融化了一些，它排開的水少了，產生的浮力也少了。

而另一方面，因爲全球變暖，南極地區的融雪增加了，拉森 C 的上部由於融雪的影響，變得越來越緻密、越來越沉了。

拉森 C 一面和陸地上的冰川連接在一起，導致它不能隨意的上下浮動。一面又產生了頭重腳輕的現狀。如此一來，冰架受力不均，就折斷了，這也就是我們今天看到的這條大裂縫的成因。

這條裂縫之所以產生的如此之快，如此之長，早已超過了冰架自然剝落的正常速度，從 2014 年開始，這條裂縫就是不斷地加速增長。去年那次新聞火爆的時候，它還只有 113 公里長，而僅僅過了半年，到今天它已經發展到了 180 公里……2 月的時候，它的前端甚至還出現了一個分叉，而就在前幾天我們歡慶五一的時候，這個新產生的分叉，正在朝着海洋的方向，大踏步的邁進……

就目前的態勢來看，可能用不了多長時間，這條裂縫就將把拉森 C 靠近海面的這一大塊剝落出來，由此形成一座巨大的冰山（當然，這塊冰山可能也會破碎成許多小塊），面積差不多有整個上海市那麼大。

像我一樣生活在沿海地區的人們，難免會產生一個擔憂：那麼形成了這麼大的冰山，是不是會導致我們的海平面上漲呢？答案是否定的，又同時是肯定的。

單純來看這做即將生成的冰山（由於冰山還沒有正式形成，所以貌似還沒有被命名，我們可以暫且稱呼其爲拉森 C 冰山吧），它本身是不太會對全球海平面構成什麼直接影響的。我們剛纔說過，一個物體在水中所受的浮力，與它所排開的水所受的重力相同。我們假設有一個游泳池，泳池中有 99 噸水，再放入一個 1 噸重的大冰塊，那麼無論它是 1 噸重的冰塊，還是最終融化成了 1 噸重的水，泳池裏的總重量都是 100 噸。對於這個拉森 C 冰山也是如此，它本來就一直浮在水中，即便它從整個冰架上脫落，也不會再排開額外的水，即便它完全融化了，也不會造成海平面的上升。

但問題之棘手在於，這個拉森 C 冰山的剝落，會導致一系列多米諾效應，最終導致海平面的上漲。

我們前面說過，冰架其實就是陸地上冰川伸入海中的那一部分，而在南極洲，真正巨量的冰其實是位於陸地之上的冰蓋，正是這些冰蓋因爲重力作用的緩慢移動而產生的冰川，才造就了沿海地區的冰架。而這些冰架，無疑於是冰川的減速器，它們漂浮在水面上，成功地遏制了陸地上的冰蓋無節制的傾瀉到大洋中。

但拉森 C 冰蓋的這次脫落，就相當於這個減速器減少了相當一部分力量，那麼，它們頂住身後冰川的能力就降低了，冰川就可以以更快的速度向海中推進，而由於冰川流動速度增快了，冰川的厚度變得薄了一些，這些變薄的冰川浮到水面上形成的新的拉森 C 冰架，相對於以前的拉森 C 冰架，也薄了一些。

那麼，更薄的拉森 C 冰架，就更容易崩裂脫落，也更無法抗住身後冰川的運動，於是惡性循環，冰架越來越薄，冰川越來越快，冰架更薄，冰川更快……最終，冰架完全消失，這個區域的冰蓋完全衝到水裏。

而這些厚度達到 2000-4000 米、蘊藏了全球 90% 的冰量、全球 70% 淡水量的冰蓋，本來是由南極大陸的陸地來承載的，它們原本是沒有排開海水的……現在它們衝到了海中，它們排開的海水……最終就會成爲淹沒我們沿海地區的那些上漲的海平面……

細思極恐

也就是說，冰架一旦開始了非正常的崩解（就是說，冬天的降雪補充速度再也無法追上冰架消失的速度），就會進入快車道，這次是 12%，下次崩裂的可能就是 30%，再一次可能就是 50%……我們只需要看一下拉森 C 旁邊不遠的拉森 B 冰架的崩裂過程，就很容易理解這個過程。

上世紀八九十年代開始，拉森 B 冰架不斷崩解，到 2002 年徹底消失，受此影響，原本由拉森 B 阻擋的三個冰川，流動速度普遍加快，其中克蘭冰川流動速度加快了 3 倍，厚度則減少了 20 多米

更可悲的是，正在崩解的冰架，絕不僅僅只有 1995 年的拉森 A，2002 年的拉森 B，和今天的拉森 C，實際上，南極許多地區的冰架都在經歷這一階段。尤其是南極的西南極地區， 百年增溫率接近 2℃，這一現象更加明顯。不過也有值得慶幸的是，南極東南極地區，增溫並不顯著，這個區域的冰架和冰蓋還處在比較穩定的階段。

然而……儘管前景是如此的緊迫，我們旨在減少溫室氣體排放，遏制溫室效應惡化的《京都議定書》，卻最終難產而死；好不容易達成的《巴黎協定》，卻又因爲海的對面那位金髮長者的表態，變得前途未卜。

或許，在我們後人的記載中，2017 年的 5 月或 6 月，會留下濃墨重彩的一筆：在遙遠的地方，有一座冰山誕生，有一片冰架消融，有一個自負的故事。一切歷史都是現代史，而我們都是歷史的見證人。
