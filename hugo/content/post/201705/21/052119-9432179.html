「全額退款」騙局的一點分析

今天又看到有回答提及一個簡單的騙局：包生男孩 / 包過考試 / 包……，如不成功全額退款。

在這個騙局中，騙子什麼都不需要做：畢竟這些“包成功”的事情本來就有一定成功的概率，即使不成功全額退款，騙子也沒有多少損失。

讓我們從機制設計的角度來分析一下這個騙局，這其實是一個簡單的 Hidden Action 的問題：如何設計報酬機制，才能使別人真的花力氣去“包成功”呢？

我們設定模型參數如下：

1. 有兩個參與者：委託人 P 和代理人 A。

2. 代理人 A 可以選擇努力（“包成功”，成本爲 c）或者不努力（什麼都不做，成本爲 0）。

3. 當代理人 A 選擇努力時，以概率 1 事件成功；選擇不努力時，以概率 p 事件成功。

4. 事件成功給委託人 P 的收益是 U，事件不成功給委託人 P 的收益是 0。

5. 委託人無法觀察到代理人 A 是否努力，只能觀察到事件是否成功，因此委託人只能根據事件是否成功給予報酬：當事件成功時，委託人支付給代理人 R，事件不成功時，支付 r。

6. 委託人和代理人都是風險中性的，即代理人的效用函數是報酬減去成本，委託人的效用函數是事件收益減去報酬。

7. 代理人如果不接受委託，其收益是 0（即 Outside Option）。

也就是說，一個使委託人效用最大化且激勵相容的機制應該滿足：

1. R=c，也就是說，當事件成功時，報酬應當達到代理人所需付出的成本。

2. r<0，也就是說，當事件不成功時，不僅要全額退款，還要倒貼補償。

進一步地，讓我們考慮一個更現實的設定：

8. 市場上有兩類代理人，其成本分別是 C 和 c（成本爲 c 的所佔比例爲 q），且 C>U>c，換言之，委託人永遠不會找成本爲 C 的這類代理人；此外，委託人無法觀察到代理人的成本，而是由代理人自己報告。

這樣一來，我們的模型就既有 Hidden Action，而且也有 Hidden Information。但是由於我們已經知道委託人永遠不會找成本爲 C 的代理人，所以所有代理人都會彙報自己成本爲 c。

接下來：

（1）我們顯然不可能讓所有代理人都努力，因爲我們支付不起成本爲 C 的代理人的報酬；

（2）如果我們支付的報酬使得所有代理人都不努力，那我們應當索性不支付報酬；

（3）除開這兩種情況，我們可以讓成本高的那部分代理人選擇 Outside Option，來解決 Hidden Information 的問題。剩下來的就是成本低的那部分代理人，我們希望這些代理人選擇“努力”。

（注意這裏都是嚴格的不等號），也就是說，當事件不成功時，不僅要全額退款，而且要倒貼補償。

注意，在 Hidden Action 問題中，這一要求來自於（激勵相容下的）效用最大化（注意不是來自激勵相容），而在加入 Hidden Information 之後，這一要求直接來自於 Screening 的需要。
