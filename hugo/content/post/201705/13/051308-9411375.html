從目前的實證結果看：在大學校園內實施的在線教育，很可能確實拉大了學生之間成績的差距。文中用黑體標出了各項研究的主要結論，結尾列出了參考文獻。

評估在線教育影響的研究汗牛充棟，但是，其中絕大部分都不夠嚴謹，結果受遺漏變量等問題影響（Figlio，Rush 和 Lu，2013）。做得比較好的幾篇主要是兩種方法：一是通過一些計量的方法來解決掉內生性，比如找工具變量；二是在大學裏做實驗，直接把學生隨機分配到傳統教育組和在線教育組，然後看結果的差別。

在展示結果之前補充兩點。首先，爲了得到比較精確的因果識別結果，這些研究中，是否接受在線教育的學生之間，除了聽課方式不同，課程內容幾乎沒有差別——兩類學生面對同樣的小測和考試題目，享受相同的答疑時間，等等。其次，在外推這些結論時應該十分謹慎。嚴格意義上說，結論只適用於部分引進了在線教學方式的大學課程。爲了結果的精確，這是很難避免的侷限。

圖 1 來自 Bettinger 等（即將刊出）。選課前累積 GPA 與選擇在線教學方式的影響，包括影響的點估計與 95% 置信區間。

第一種方法的代表作是 Bettinger、Fox、Loeb 和 Taylor 即將刊出的研究。四位作者都來自哈佛或斯坦福，擅長教育政策評估。他們從美國的 DeVry 學院拿到了歷年來 750 門課，230000 名學生的數據，然後用“當年該校區是否開課”，以及“學生住址離校園距離”做工具變量[1]。估計結果如上圖：選擇在線方式會導致這門課 GPA 下降 0.33。之前 GPA 越低，負面影響越大（見上圖）。

圖 2 來自 Xu 和 Jaggars（2014）。上面這一部分是在全樣本回歸中，加入 GPA 與是否接受在線教育的交互項；下面這一部分是按 GPA 高低，將樣本分成兩塊迴歸。兩部分結果一致。

哥倫比亞大學的 Jaggars 和加州大學爾灣分校的許笛用華盛頓州 500000 門課程，40000 門學生的數據做了不少有關在線教育的研究（2013；2014）。其中，2014 年的研究評估了在線教育對不同成績學生的影響。如果把之前的累積 GPA 在 3 以上稱作“高”，否則稱爲“低”，結果發現：選擇在線教育對成績有負面影響，會增加退課概率。同時，低 GPA 學生受影響更嚴重（見上圖）。

圖 3 來自 Figlio，Rush 和 Lu（2013）。實驗中分配到在線教育組對學生成績的負面影響。數字越大，說明負面影響越嚴重。

第二種方法的代表作是 Figlio，Rush 和 Lu 2013 年的研究。他們在某大學全校必修的微觀經濟學課程中招募了 312 名參與者。傳統組的學生照常上課，但不能在教學網中看網課；在線教育組的學生只能看網課，老師不允許他們進入課堂。結果顯示：接受在線教育會輕微降低學生成績，影響不顯著。但是，在線教育對拉丁裔學生、男生，以及 GPA 在後一半的學生成績有顯著的負面影響。

圖 4 來自 Bowen 等（2014）。接受傳統教育與混合教育（加入在線內容）的兩組學生成績的分佈，其中橙色爲傳統組，藍色爲混合組。

在 Bowens、Chingos、Lack 和 Nygren 等四位學者 2014 年發表的研究中，他們聯合紐約城市大學、紐約州立大學、卡內基梅隆大學等多所學校，在多門課程中實施了多種對照實驗，比方說允許部分學生免費使用電子書、給部分學生髮放電子書券，等等。整體來看，這些措施沒有取得顯著效果。以 GPA=3 爲界將學生分兩組，沒有證據說明這些措施的影響，在兩組間有明顯差別。

最後總結一下：首先，如果是評估大學校園中，引入在線教育是不是會拉大學生間的差距，大部分研究給出了肯定的結論；其次，如果是評估引入在線教育的絕對效果，部分研究發現有負面影響，也有部分研究發現沒有顯著影響。外推結論應謹慎，比方說，在大學課堂中引入在線教育對成績有負面影響，並不等於將在線教育覆蓋到之前接觸不到這些教育資源的人也會有負面影響。

[1] 文中的識別方法是使用兩個工具變量的交互項，這一方法依賴的假設弱於一般的工具變量法。詳情可見 Card（1995）。

當我們說公平的時候，在不同的人眼裏是有不同的含義的。

學生的維度從來不是單一的，成績好的和不好的算一個維度；家庭富裕和貧窮算一個維度，有的時候我們說公平，是說能夠讓成績不好的也提高一下，縮小和成績好的人之間的差距；有的時候我們說公平，是讓窮人也能分享到以前富人所擁有的教育資源……

@Manolo 的答案是說，在線教育拉大了學生之間的差距，尤其是降低了男生、GPA 靠後的學生的成績。這個結論是合理的，因爲同在大學裏面，ＧＰＡ靠後的學生一般來說是三種：

1. 自發學習動力不足的

2. 缺乏自律的

3. 理解能力較差的

而在線教育比傳統教育更需要學生的這三種能力。所以在這裏 GPA 其實是一個學生能力的不完美信號，而在線教育這種形式比傳統教育更好的能夠把上述三種能力不同的學生篩選出來。

對於這種不公平，我認爲是大學制度造成的。人和人之間的特點本來就不同，一羣人擺在那裏，總有更適合自學的，和不適合自學的兩種人，傳統教育下兩種人差別可能不那麼大，但是網絡教育下就顯示出來了。所以這說明現在的大學其實只是一個混同均衡，沒有區分出來這兩種人。那麼以後可能兩種教育制度並行，傳統教育，因爲其帶有一點隱性的強制性，就專門爲滿足以上三種條件的人而開設，而網絡教育爲自學能力比較強的人開設。

而另一方面，在線教育還是有助於公平的，那就是讓因爲家庭或者其他原因的限制而沒有機會接觸到如此優質教育的人得到學習的機會。也就是說在線教育至少放鬆了人們財務上的約束條件，讓人們能夠在更寬鬆的條件下作出自由的決定。就像我比較欣賞的一位開源軟件開發者主頁上的自我介紹（人很好很友善，在 Github 上提交 issue 幾乎是當天就回復），我有時候覺得低落的時候，看看這段話都覺得挺感動：

I am a self taught IT professional, I am what we can define as a late technology adopter, my first computer was a Pentium II and my age was already 21, so no, I never went to college to study Computer Science. Thanks to the Free Software Foundation, I was able to use and study a complete Unix system in my old x86 computer in the late 90's, that system was called GNU/Linux. I was able to start studying C and C++ using the GNU C and C++ compiler gcc. Was thanks to the Free Software Foundation and the GNU project that a guy coming from a humble family with not much resources learned how to program and achieve success as valuable professional in the IT industry. Today, I share my work under the GPL license that guarantees that anyone using software that I've written will be able to use it, execute it, study it, modify it, redistribute it or even sell it with totally freedom.

這位開發者從來沒有在大學上過計算機科班，21 歲才接觸計算機，但是就因爲自由軟件運動，讓像他這樣家庭出身的人也能夠接觸到計算機科學的知識並學習編程，然後自學成才在一家 IT 公司找到了體面的工作，成爲了“IT 領域有價值的專業人員”。

最後，對公平的問題說下自己的總體看法：

我認爲，一個好的公平的體制，是讓人儘可能的在更少的外部約束（主要體現在時間和財產上的約束）下表達自己的興趣並作出自己的決定。但是這種以“自發參與”爲基礎的機制，最終往往依然會形成新的“不公平”。但是，從社會福利局部均衡的意義上講，基於內在能力或者生產率差異的不公平是好過基於外部約束所造成的不公平的。
