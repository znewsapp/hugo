爲什麼 iPhone X 之前都用阿拉伯數字，iPhone X 用羅馬數字？

iPhone X 延續了 Mac OS X 的命名方式。而 Mac OS X 的命名有段歷史。

-

蘋果公司 1984 年發佈了 Macintosh 電腦。Macintosh 這個名字是項目發起人 Jeff Raskin 根據一個蘋果品種 McIntosh 起的。爲避免麥金託什實驗室（McIntosh Laboratory) 名字衝突，故意改變字母的拼寫，就成了 Macintosh。

喬布斯在 Lisa 項目失敗後，盯上了 Macintosh 項目。Jeff Raskin 受不了離開蘋果公司，Macintosh 就成爲了喬布斯的項目。而喬布斯在 Macintosh 發佈後一年，於 1985 年離開了蘋果公司，同年創辦了 NeXT 公司。NeXT 公司後來開發了一個操作系統 NEXTSTEP。

Macintosh 電腦的操作系統最開始名爲 System，8.0 時改名爲 Mac OS。Mac OS 一直維護升級，但由於開發時間太早，有點先天不足，並不具有一個現代操作系統所應有的特性。在 1994 年，System 7.5（代號 Mozart）發佈後，蘋果公司打算創建一個全新的操作系統，這個系統將有一個全新的內核。

跟 Mac OS X 以貓科動物和風景爲代號不同，Mac OS 以音樂家名字爲代號。這個新操作系統的代號爲 Copland，在 1995 年 3 月正式公佈計劃，預期 1996 年發佈。但到了 1996 年，Copland 還很糟糕。經過評估，Copland 是沒有指望了，按照蘋果公司的狀態，這個系統是永遠都不可能發佈，還不如早點取消爲好。

自己造不成系統，蘋果公司就打算想向外購買一個新操作系統。最開始跟看中了 Be OS，但價錢沒有談攏。最後在 1997 年收購了 NeXT 公司，擁有了 NEXTSTEP。喬布斯轉了一圈，重新執掌了蘋果公司。

於是開始以 NEXTSTEP 爲基礎，整合原有的 Mac OS 系統。整合需要時間，舊版 Mac OS 也升級維護，1999 年 10 月，Mac OS 9 發佈。2001 年 3 月，整合完成，Mac OS X 最終發佈。Mac OS X 是全新的操作系統，擁有全新的內核，之後 Mac OS X 一直延續到今天，改名爲 macOS。

-

瞭解這段歷史，大家就知道 X 這個字母有更深的含義。這裏的 X 不單單是羅馬數字 10 的意思。更有點重啓，面向未來的意味。

在 iPhone 發佈 10 週年，發佈 iPhone X，有着特殊的意義。我猜蘋果公司是想表達，iPhone X 是 iPhone 面向未來的重啓之作。iPhone X 不應該理解成 iPhone 的第 10 代產品，而應該是 iPhone X 的一代產品。正如 Mac OS X 最初發布的時候，並非是 Mac OS 的第 10 版，而是 Mac OS X 的第一版。
