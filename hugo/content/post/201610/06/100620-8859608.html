月球表面的這塊區域會是什麼？

利用嫦娥二號數據展示系統查看了一下嫦娥二號拍攝的衛星圖片，在這個地點發現發現了一塊與周邊地理特點不同的區域。具體座標是經度 152.24862；緯度 24.69524。仔細觀察發現周邊地區都是十分均勻的撞擊坑。旁邊也有一片類似於月海的小塊區域。比較奇怪的是撞擊坑內的線條，呈現一種網格狀的情況。

請問，這種情況會是人工構成的麼？

首先不知道是題主這張影像的分辨率還是光照什麼的問題，凹凸感和粗糙度感很容易讓我這種眼神不好的覺得是坍塌的熔岩管道（lava tube），但查了下 LROC 給出的 Komarov 撞擊坑的影像，感覺就很不同了。

（LROC Komarov）

這種底部具有不規則線性條紋的撞擊坑叫做 Floor-fractured craters (FFCs)，是月球上比較常見的一類地形，比如：

以下分爲兩個部分展開：【FFCs 的形態和分類】和【FFCs 地形的形成機制】。

【FFCs 的形態和分類】

人們在七八十年代就已經發現了兩百多個這樣的撞擊坑並給這類地形專門分出一類，還根據具體撞擊坑形態和裂隙的形態進一步分了 6-7 個子類(Schultz,1976 Floor-fractured lunar craters; Wilhelms,1987)。事實上除了月球，火星和水星上也有 FFCs 地形。

進入 20 世紀，新的更高分辨率的月球高程數據 LOLA（ Lunar Orbiter Laser Altimeter）和影像數據 LROC (Lunar Reconnaissance Orbiter Camera) 使之前的分類更加清晰了。對這類地形最新的綜述性研究來自 Jozwiak et al. 2012 和 2015 年的論文，她 / 他們把170 個FFCs地形並分成了 8 個子類（但基本分類思想依然沿襲了Schultz 那篇文章），給出了每一個子類的描述和分佈。題主提出的Komarov 撞擊坑就屬於 Class 3 的情況，恩，不想看廢話的直接跳轉過去就好了。

  • Class 1：通常是較大的撞擊坑，直徑基本在 50~300km，撞擊坑底部(floor)較深，中央峯(centrak peak)地形複雜，有延伸的撞擊坑壁(extensive wall terraces)。撞擊坑底部通常有徑向和 / 或同心圓狀的裂隙(fractures)，坑壁的斜坡帶(scarp)通常有新月形的月海沈積物(mare deposits)。代表撞擊坑有 Humboldt, Atlas, Schülter 和 Cardanus，下圖 ABCD 分別爲 Humboldt 撞擊坑(座標 27.2° S, 80.9° E, 直徑 207 km)的等高線圖，地質示意圖（B 圖中那些線性結構就是 fractures），LOLA 格網地形圖，白線對應的地形剖面圖，最後一張是 Class 1 的撞擊坑在月球上的分佈圖
  • Class 2：中等尺度的撞擊坑，通常直徑 13-75km，坑壁非常明顯，坑底上凸，有中央峯，有明顯的同心圓狀的 fractures，代表撞擊坑有 Vitello,Encke 和 Davy，下圖 ABCD 分別爲 Vitello 撞擊坑(座標 30.4° S, 37.5° W, 直徑 44 km)，最後一張是 Class 2 的撞擊坑在月球上的分佈圖
  • Class 3：撞擊坑尺度範圍較大，直徑在 12-170km 都有，大部分直徑約 30-60km，底部較平坦（有些甚至沒有中央峯），坑壁邊緣有寬而明顯的溝壑(moat)，fractures 通常爲多邊形和 / 或經向分佈但不會延伸進溝壑區域，部分撞擊坑內部有月海物質，代表撞擊坑有 Gassendi, Taruntius 和 Lavoisier，Haldane 和 Runge 撞擊坑有典型的很寬的溝壑，下圖 ABCD 分別爲 Gassendi 撞擊坑(座標 17.5° S, 39.9° W, 直徑 110 km)，最後一張是 Class 3 的撞擊坑在月球上的分佈圖，幾乎所有 Class 3 的 FFCs 撞擊坑都位於盆地（basin）的內部或邊緣。題主給出的Komarov 撞擊坑也屬於這一類（紅框部分）。
  • Class 4：特徵爲具有 V 型的溝壑（V-shaped moat），有些具有經向和 / 或同心圓狀的 fractures,根據溝壑 V 型的程度可進一步分爲三類 Class 4a, 4b, 4c 三類。下圖爲 Class 4a 的代表撞擊坑 Bohnenberger 以及三類撞擊坑的分佈。
  • Class 5：通常較古老，直徑在 12-177km，撞擊坑邊緣已退化模糊，內部有明顯的徑向、同心圓和 / 或多邊形的 fractures，這些撞擊坑內部通常爲古老的高地物質，代表撞擊坑有 Von Braun，Repsold 和 Alphonsus。
  • Class 6：除了特別大和特別小的撞擊坑之外，內部完全被月海物質填充，同心圓狀的 fractures 全部緊貼撞擊坑壁，部分撞擊坑有中央峯和經向 fractures。代表撞擊坑有 Pitatus 和 Fracastorius。

【FFCs 地形的形成機制】

對 FFCs 地形的形成機制，主要有兩種猜想：

1）粘彈性鬆弛(viscous relaxation)

(Masursky, 1964; Daneš, 1965; Cathles, 1975; Hall et al., 1981)

2）岩漿侵入(magmatic intrusion)

(Brennan, 1975; Schultz, 1976; Wichman and Schultz, 1995; Dombard and Gillis, 2001; Jozwiak et al.,2012, 2015)

粘彈性鬆弛假說（Lunar floor-fractured craters: Evidence for viscous relaxation of crater topography）的理論基礎是：新鮮撞擊坑引起的地形起伏隨着時間推移會慢慢回彈趨於平坦（總體趨勢是低處變高高處變低），但長波（大尺度）地形比短波地形鬆弛速度更快，所以大尺度地形起伏隨着時間推移更快的恢復平坦，而短波地形則變化較慢相對保持原樣，FFCs 是這種差異性的回彈形成的線性 fractures。

↓ 粘彈性鬆弛的過程示意圖，A 爲新鮮撞擊坑形態，B 爲經過一段時間後的撞擊坑形態

但後來的諸多數值模擬的結果都無法支持粘彈性回彈假說（Testing the viability of topographic relaxation as a mechanism for the formation of lunar floor-fractured craters），新的更高分辨率的月球高程數據 LOLA 和影像數據 LROC 也更支持岩漿侵入假說。岩漿侵入假說的理論基礎是：撞擊坑形成之後發生岩漿侵入，巖牆(dike)向地表方向不斷延伸，延伸最多的 OP1 類巖牆直接穿透地表引起熔岩流噴出，延伸最少的 OP3 直接在地下較深處固化，而中等程度的 OP2 類侵入到地殼淺層附近形成岩牀(sill)，這類岩漿侵入就可能形成 FFCs 地形。

下圖展示了岩漿侵入假說認爲的 FFCs 形成過程：A. 巖牆(dike)向月球地殼方向侵入→ B. 巖牆侵入殼層，被撞擊坑底部地下的角礫岩阻擋於是停在了這裏 → C. 壓力累積使得巖牆開始橫向侵入形成岩牀(sill) → D. 岩漿繼續累積使得岩牀逐漸轉變爲穹頂狀的巖蓋(laccolith)或者 E. 更厚的岩牀，FFCs 地形就是岩牀 / 巖蓋向上擠壓撞擊坑底部形成的裂隙。

當然這一過程還會同時形成一些排氣出口(vent)和地表以下的布格重力異常，這些也都被 LROC 影像和 GRAIL 重力數據所支持。
