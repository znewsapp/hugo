回答來自機構帳號：「以太資本」

將 CEO 打造爲網紅，對企業來說有什麼好處和風險？什麼樣的 CEO 適合做網紅？

開宗明義的講，將 CEO 打造成網紅，好處大於風險。本質的邏輯是，早期大部分的創業公司都需要各方面的協助才能順利步入市場正軌，通過 PR 可以有效的吸引優質資源， 而打造一個網紅 CEO 恰是一種作用顯著的 PR 方式。

好處

首先是對客戶的 PR。CEO 是網紅，尤其是當你在專業領域頗有建樹的時候，會有更多業內的資深人士和初入行的年輕人會願意相信你，這對於客戶合作方案和招募人才的時候都會有很大幫助。

其二是對媒體的 PR。在中國資本市場走向成熟的過程中, 媒體作爲獲取資訊的主流渠道之一，伴隨着互聯網的迅猛發展，在資本中發揮的作用也在日益明顯。網紅 CEO 坐擁大量豐富的媒體資源，是資訊擁躉的焦點，因此也是發揮 PR 作用最明顯的環節。

其三是對資本的 PR。發展 CEO 作爲網紅可以提升整個公司的形象，從而獲得更多的資本關注，這一點其實對融資是非常有利的，至少創始人和項目可以被更多投資人知道。這也是通過 CEO 個人 PR 達到的吸引客戶、拓展業務，招募團隊、推助公司加速擴張發展，成爲資訊熱點的必然結果。

風險

這其中明顯的風險在於，CEO 本人需要在 PR 這一件事情上付出大量的時間和精力。

一般來說任何一個網紅都需要維持，從而保持一定量的曝光度；否則除非是非常有名的業界大佬，只要稍過一段時間不在媒體上露面，關於創始人本人和項目的熱度就會消減，其中成本和難度和其他 PR 原理一樣。所以，如果 CEO 耗費大量時間在各種活動上，其實必然會縮減對業務的投入。

此外，現在的品牌傳播越來越人格化了，從過去品牌到機構的聯想，到現在將營銷發展爲情感上的聯繫。所以，成爲網紅後的企業家，確實更牢靠的與自己的企業捆綁在了一起。但是，個人品牌與企業品牌的羈絆也帶來了副作用——如果你不喜歡這個人，很可能就不會喜歡這個品牌了。這種相關基本是決定性的，所以成也蕭何敗蕭何。

畢竟炒作一個人和炒作一個品牌難度、成本、原理都內在相關。如果創業者是剛剛開始經營公司的新手，很可能因爲業務能力跟不上過度宣傳而遭到“差評”。

關於什麼樣的 CEO 比較適合作網紅，簡單的邏輯在於，網紅 CEO 可以做什麼事情，這裏分爲兩種類型。

第一類是 TO C 的大衆層面網紅 CEO，主要功能是提高企業的知名度，也就是 PR 的效果。以做 C 端消費品業務的公司爲例。項目產品越是面向公衆，個人知名度越會提升公司和品牌的知名度。典型的是羅輯思維，羅振宇自己是網紅，對業務的拓展和發揮項目甚至行業的影響力都有很大幫助。

第二類，是垂直領域網紅 CEO 的角色，掌握行業話語權；能夠在有效引導行業潮流的同時，代表行業發出更多有力度的聲音，從而進行資源協同，這點很重要。以科技公司爲例。這類公司普遍不是靠實際業務去融資的，因爲科技公司做的事情通常都很前沿，所以他們需要告訴市場，未來是什麼，並且說服投資人和團隊去相信這個未來。這種情況下，CEO 作爲網紅就可以聚集更多資源幫他解釋和建構，這在早期創業中的作用尤其顯著。

雷軍是一個同時佔了大衆和垂直領域兩種屬性的網紅 CEO。以我對他個性的瞭解，他其實是一個非常不願意當網紅的人，而且他自己也說過 CEO 應該踏踏實實的做一些事情，不希望花太多時間在參加活動上。但是他依然會是不是出現跟媒體互動，正是因爲他知道，從小米作爲消費品的角度上來說，他依然需要更多的知名度。

總之，好的 CEO 就是“找人、找錢、找方向“。找人和找錢直接需要影響力，因而具有廣泛影響力的 CEO 是關鍵助力。找方向需要自己要去找，這也是以人和錢爲基礎的。
