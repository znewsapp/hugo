如果《權力的遊戲》發生在中國，各個省份怎麼與七國對應？

很早之前自己就腦補過這個問題。今天正好撞見了，試作一答。

把小說裏的地理代入到中國歷史中確實很有意思。我自己也這樣想過，當歷史寒冷期降臨，北方少數民族被迫南遷長城之內（凜冬將至，異鬼、野人南下）；同時中原的中央政權崩潰（勞勃國王駕崩、獅家掌權、七國分裂），各個地方脫離中央各自爲戰；又同時，在這片土地之外（維斯特洛大陸以東的厄索斯大陸）還在孕育着新的欲要吞噬中原的力量（可腦補爲蒙古、日本這些外部的非漢人文明）。但題主既然說要把現代中國的各個省份類比維斯特洛大陸的七國，那就結合兩個世界裏各個地域的地理、經濟、人文特點作個類比寫在這裏，權當一樂。

首先放結論：

  • 王領 = 北京 + 天津 + 河北 （+ 遼寧省內南部及遼東半島）（君臨城 = 北京， 黑水灣 = 渤海灣）
  • 北境（史塔克家族） = 東北三省 （臨冬城 = 長春）
  • 河間地（徒利家族） = 河南 + 安徽 + 湖北 （奔流城 = 武漢，孿河城 = 襄陽）
  • 鐵羣島（葛雷喬伊家族） = 臺灣 + 澎湖列島 （ 派克城 = 臺北 ）
  • 西境（蘭尼斯特家族） = 江蘇 + 浙江 + 上海 （凱巖城 = 南京，蘭尼斯港 = 上海）
  • 谷地（艾林家族） = 四川 + 重慶 （鷹巢城 = 成都）（又有些知友認爲，鷹巢城應該給重慶）
  • 河灣地（提利爾家族） = 湖南 + 江西 + 福建 （高庭 = 長沙， 舊鎮 = 福州）
  • 風暴地（拜拉席恩家族） = 廣東 + 廣西 （風息堡 = 廣州）
  • 多恩（馬泰爾家族） = 新疆 （陽戟城 = 烏魯木齊）

蟹爪半島 = 遼東半島；尖角 = 山東半島 ；那麼很自然地，黑水灣 = 渤海灣；黑水河 = 海河。僅考慮地理位置的話，似乎港口城市天津更像是君臨城。但若作爲政治中心，那麼君臨非帝都莫屬。（順便，天安門可以看作是紅堡...） 至於御林，那便是北京西北方的防護林了 （不知道冰火的世界裏，御林是不是也是用來防護從多恩邊疆地吹來的沙塵暴的 _(:з」∠)_ ）

北境（史塔克家族）

中國的東北地區。

兩者都是氣候寒冷、地域廣闊、人口稀疏、森林廣佈；西北部有山脈（大興安嶺、小興安嶺、長白山脈），東部有平原 （東北平原）。而且這裏的人都是性子耿直。

河間地（徒利家族）

中原地區，包括河南、湖北、安徽。

兩者都有着豐富的水系 （長江、黃河、淮河），有山有平原，富庶、豐饒、人口稠密。但同時也是戰爭的頻發地區。孿河城扼守要道，同襄陽城一樣有着極高的戰略價值。（襄陽城西可進長安、蜀川，北可略洛陽等地，是江南地區的屏障；孿河城東可進谷底（四川），又是溝通北境與南方的重要位置。）武漢位於長江與漢江的交叉口，同樣奔流城也位於兩河的交叉口。（況且七月的武漢，真可以稱作是「奔流城」了......）

西境（蘭尼斯特家族）

江浙滬

同樣都是富得流油的沿海地區。有山嶺有平原，交通發達，七國首富。政治勢力在七國之中僅次於王領；在現實世界裏也一樣。

蘭尼斯特家族的主城凱巖城作爲政治中心，等同於南京；蘭尼斯特港作爲沿海的經濟中心，等同於上海。而且聽說這兩個城邦還特別愛相互撕逼……

谷地（艾琳家族）

中國的谷地……

二者都是易守難攻之地，有高峻的山嶺包圍。成都即是谷底的政治中心鷹巢城（但如何按城市形態來看，似乎重慶更符合）；而位於進入谷底唯一通道的血門，恐怕就相當於四川盆地之北，扼守大巴山脈的劍閣關了。

河灣地（提利爾家族）

湘贛閩

同樣溫熱溼潤的季風性氣候，風景宜人，盛產美女。高庭居於內陸河岸，類似長沙；舊鎮位於河流入海口，類似福州。

風暴地（拜拉席恩家族）

兩廣地區

兩者都是位於南部沿海區域，有大面積的山地阻隔北方 （南嶺）。森林廣闊，海岸多磐石。雖然無論小說中還是影視劇中都未提風暴地的經濟發展如何，但畢竟作爲離厄索斯大陸只隔了一個峽海的地區，貿易往來應該不會差。

多恩（馬泰爾家族）

新疆

二者都是氣候炎熱乾燥、沙漠廣佈。也多產有異域風情的帥哥美女。作爲政治中心，陽戟城的地位便等同於烏魯木齊。
