經常很多人在我曬早餐的微博裏問我，我打的 smoothie 的配方。

今天我就歸置歸置，都跟你們分享～

很早就買了一臺料理機，從此營養健康又快手的 smoothie 成爲我家的早餐飯桌上的常客，保留了完整的水果纖維，沒有浪費食材，也更具飽腹感。

每天早上五分鐘，把水果去個皮丟進去，就能成就一個元氣滿滿的健康清晨！

接下來我們一起看看我用它做的 smoothie 都可以有什麼搭配吧！（配方兩人份）

西柚 + 甜橙

家人喜歡西柚微微清苦的口感，兩顆西柚搭配一顆甜橙，奶昔模式一分鐘，就可以喝到有柑橘纖維果粒的一杯 smoothie 了，柑橘類的水果水分很多，所以口感更偏重於清爽的果汁。

喜歡紅色果汁的，也可以用半顆西瓜來打，籽不去也能打得非常細膩。

番茄 + 蘋果 + 柑橘類水果

用一顆番茄加一顆蘋果和一顆西柚（或者橙子，檸檬）。

蘋果打起來水分比較少，更傾向果泥，所以需要水分多的水果一起打，稀釋一下也更易入口。

水蜜桃 + 橙子 + 養樂多

這是我近期打過的最喜歡的一款味道。

兩個橙子 + 一顆水蜜桃（黃桃）+ 一瓶養樂多，打出來除了慕斯一般的口感之外，還有一股很清新的桃子的香氣，柑橘微酸又中和了水蜜桃的甜，養樂多又補充了乳酸菌，這個組合可以喝一個禮拜都不會膩噠。

獼猴桃 + 蘋果 + 百香果 + 養樂多

三顆獼猴桃 + 一顆蘋果 + 一顆百香果 + 一瓶養樂多，百香果很酸，如果把籽全部加在一起打碎會有一點澀味，所以我就把果肉挖出來，直接放在打好的果昔裏面，獼猴桃的籽被碾碎，但有一些非常細膩的顆粒感在舌尖，像小時候吃過的跳跳糖。

香蕉 + 獼猴桃 + 檸檬

兩根香蕉 + 兩顆獼猴桃 + 一顆檸檬，香蕉的綿密口感加上一點獼猴桃籽的層次，最後有一擊檸檬的酸收尾，是非常有趣的搭配。

兩個獼猴桃 + 兩個橙子或者兩顆黃桃 + 兩顆獼猴桃也能打出類似的 smoothie 來~~

牛油果 + 橙子 + 藍莓

半盒藍莓 + 一顆牛油果 + 兩顆橙子。破壁後的藍莓沒有了澀澀的外皮，加上綿密的牛油果，用甜橙帶入水分，讓果昔更順滑。

另外如果打牛油果，加入牛奶，是很經典營養的搭配。

紅心火龍果 + 酸奶

兩顆紅心火龍果 + 酸奶打出來會帶來非常詭異的顏色~放在餐桌上頓時很亮眼。

芒果 + 水蜜桃 + 檸檬

兩顆芒果 + 一顆水蜜桃 + 一顆檸檬。Smoothie 很適合打已經熟透的水果，太軟爛的水果吃起來口感不好，打成果泥倒是可以舒舒服服的喝下去~芒果的香，帶上柑橘的酸，是正正好的早餐開胃飲品。

蘋果 + 青檸 + 牛油果 + 酸奶蓋

兩顆蘋果 + 一顆青檸 + 一顆牛油果。有時候酸奶不一起打，蓋在果昔上面，也挺美的。一咕嚕喝光，就可以舒坦地去衛生間報道了（笑~

破壁機拿來做西餐的濃湯也是一把好手，濃湯菜譜可以戳：食譜 / 一份來自"危險妻子"的洋蔥蘑菇湯

Smoothie 小貼士

  • 如果喜歡果昔口感綿密厚重的，可以在家裏多買一些牛油果，蘋果，香蕉等果肉結實的水果屯着，家裏的破壁機可以打出細緻的果泥口感。
  • 而做 smoothie 比較百搭的水果呢，我認爲是柑橘類的水果，橙子和檸檬都非常萬能！可以爲果昔補充水分~~不會讓做出來的果昔太濃稠，酸度又可以豐富果昔的層次。
  • 胡蘿蔔和芹菜非常排毒，如果能接受兩款蔬菜的味道，建議大家可以一起加進去哦！
  • 如果機器可以打堅果，加一點堅果一起，也非常營養。
  • 比起吃水果，喝果汁會不小心攝入很多糖分，除了打好的果汁要及時喝之外，不要貪杯~
  • 文章僅供參考，希望大家都可以經常在家裏屯很多新鮮又好吃的水果，自己通過不停的實驗和搭配，調出自己最喜歡的味道！
