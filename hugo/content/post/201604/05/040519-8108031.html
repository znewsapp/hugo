遊戲廠商的反盜版與被盜版的歷史是怎樣的？

反盜版措施大致可以粗分爲軟防和硬防。

純的軟防非常弱，說白了就是key。你買遊戲的時候，給你個密碼，安裝之前或者玩之前要求你輸入這個密碼。但有效密碼是固定的，只要任何一個玩家把密碼散出去就完了，基本上還是靠自覺。

然後就有加強版軟防，也就是密碼本，通常就印在遊戲說明書上。每次玩遊戲之前（或者遊戲過程中）蹦出來，讓你上第幾頁第幾行找密碼，或者選擇圖片某個部分的正確顏色，每次出現的問題是隨機的，類似現在有些網絡遊戲或銀行的密保卡（區別在於現在的密保卡有使用次數限制，安全性要強得多；而老遊戲的說明書是永續的，因爲程序寫死了）。

儘管一定程度上損害遊戲體驗（好麻煩啊好麻煩），採用這種做法的 DOS 遊戲非常多，非常、非常多。它比前一種略強，至少略複雜一些。但本質上依然是簡單密碼，一旦說明書被複制，就完了。因此廠商爲了防止複製說明書，採取了各種手段。例如使用全綵印刷，文字顏色和底色的深度很相近，黑白複印出來就是糊一坨看不清；更喪心病狂的還要用半透明的彩卡蓋在上面才能看；有的還專門做一個雙層轉盤密保卡，要轉到指定的位置，看洞裏露出來的文字。

當然，肯定沒辦法完全擋住複製，只是增加複製的難度，比如你買了遊戲你同學想玩，密碼本太複雜了你就懶得抄一份給他，還是勸他自己也買一份這樣。有人閒得沒事專門去複製散播怎麼辦？——當然是去法院告死它（如果有用的話）。

進入光盤時代後，很多 PC 遊戲除了 key 以外，還要求每次玩的時候必須放入遊戲 CD/DVD，這個也是防盜版的措施之一。我估計主要還是對付一人購買全班安裝這種「分享」。破解方法要麼是原封不動複製遊戲原版 CD/DVD（但正版盤大多經過技術處理，防止隨便複製光盤內容），要麼是跳過光盤驗證。現實中大多盜版遊戲採用後者，由於實現了所謂「硬盤版」，體驗反而比正版更好……

主機遊戲比較青睞硬防，而且也更容易實現，畢竟從主機到遊戲，硬件全是定製的，加個專門的反盜版芯片什麼的根本不是事，你想想，「主機虧錢賣，賺軟件權利金」這種模式下，主機平臺方本身就有巨大的反盜版動力。相反，哪個 PC 廠商會特意強調自己的硬件玩不了盜版遊戲？

這種硬加密的歷史也比較悠久，有的是主機上有驗證芯片，運行時驗證正版遊戲裏的特殊信息；或者進一步地，遊戲程序被加密，必須由主機運行時解密。玩盜版要麼引導（麻煩）、要麼改機（失去質保），或者兩者都需要。有興趣可以讀一讀 任天堂在北美髮售的 NES 爲何與日本發售的 FC 外形不同？ - Thinkraft 的回答，後半段就是有關 NES 驗證芯片的故事。

硬防當中變態程度比較出名的當屬 CPS2 和 CPS3 街機基板，光 dump 下來 ROM 沒有用，這玩意的解密 key 是存在 RAM 中的，靠鈕釦電池維繫（俗稱自毀電池）。一旦 RAM 掉電，遊戲就運行不了了，即使正常運行至電池壽終正寢遊戲也會壞掉，需要把基板寄回 C 社換電池。這個加密手段相當有效，直到遊戲過時也沒有盜版機臺。在相當長一段時間內，CPS2 模擬器只能靠XOR 表來解密 ROM，CPS3 更是完全無法模擬，直到後來 Andreas Naive（andreasnaive (Andreas Naive) · GitHub）反向出解密算法才告破。

然而大部分主機並沒有這麼走運。對於離線時代的主機，硬防的致命缺陷在於售出的硬件不可能隨意更改，這意味着一旦加密告破，相同平臺上的後續所有遊戲全完蛋。不光遊戲，DVD 影碟的加密也是這麼被操翻的。於是有些遊戲本身也會搭載自己獨有的反盜版技術，例如 @Xi Wa 介紹的 K 社大法。挺多種多樣的，玩盜版遊戲就無法通關啊、遭到 NPC 嘲諷啊什麼的。

如今是聯機時代，有的遊戲輸入 key 的時候會聯機驗證並註冊，註冊成功後此 key 無法再次使用（或者與硬件信息綁定，原電腦可以重裝，換機器就不行）。更兇狠一點的（最典型的是網絡遊戲）要求玩家登錄賬號並全程保持聯機，從服務器端實現反盜版，這個算是大招了。對應的盜版就是私服模擬，用戶數量和服務質量可能都無法與官方服務器相比。

另外，近年的主機也有了在線更新系統的功能，在系統升級時更新解密程序，盜版用戶只能停留在上一個版本，無法玩新出的遊戲；或是無法在官方平臺上進行多人對戰。

對了，忘說了。剛纔說嘲諷，id 用的就是精神污染法。
