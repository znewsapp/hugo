你是如何轉行的？轉行容易嗎？

本人 94 年的妹子，家裏窮，沒上高中，上了箇中專，學了專業是服裝設計與工藝，學歷這麼低，知識淺薄，出來工作後也就是在工廠裏做車間工人，想當初剪線頭流水線做衣服都做過

工廠很累的，時間長又辛苦，而且大部分都是大媽

日復一日，沒有出路，看不到未來

那時我才 17 歲（2011 年），骨子裏還是不甘的，然後做了一年多的工廠流水線，毅然的辭職了。

離職後，我什麼都不會，我爸讓我回家種地，對於一個女孩子，種地太苦了，而且更沒有出路，我拒絕了

後來我爸通過關係找了我表嫂，讓我去她開的美容店裏去做美容，我不喜歡，而且我覺得自己做不來，也沒去

2012 年的時候零基礎的我，憑着自身的愛好，走上了設計，進了一家要求不高的淘寶店

在裏面就是打打雜，拍拍照，發佈一下寶貝，

你說設計，哦，並沒有，最多就是美白一下圖片，做了一年多一點，離職了。那時我才 18 歲

2013 年 19 歲 因上家公司瞭解了一些淘寶的信息，

突發奇想地開了一家淘寶店在家學習 Ps 跟店鋪裝修，學了 7 天，然後投簡歷找了工作，

非常傻逼而又不怕死的出去找工作了

於是我的設計之路真的開始了，近了一家小型公司，對方讓我做的是網站設計，而不是美工，Web 設計這個詞，我還不理解，網站這個東西我還沒有摸熟，二愣愣的硬着頭皮上崗了，（對於這份工作，我有一點是幸運了，感謝貴公司給我的機會，以及總監的指導）

苦啊，

白天工作，無從下手，每天都看無數個網站，怎麼設計，怎麼排版，不是看，還要懂，還要會用

晚上回家還要做兼職，從摳圖做起，扣了幾百張，鋼筆工具用的不要太熟練，（爲什麼不用套索，你們看個人喜歡吧，我覺得套索扣的不完美）

我屬於壓力叢生，爲了儘快學好 Ps，就開始裝逼了，裝着很會的樣子，（當然我有真的學，而且學會了）在網上宣傳可以教設計

然後真的收了三個學生，在語音教課

當然，設計，我並不會，工具我並不熟練，可以說 80% 不會

都是事先先搜索學了教程，壓力很大，然後我很用心地學通，因爲必須要對別人負責，然後學會再去教別人

我是不逼一下自己，自己不知道努力的，逼着逼着 Ps 熟練了起來，

就這樣，並是不很正規的路程，我竟然誤打誤撞的學會了精通 Ps

後經總監提拔，加班加點學習，然後 7 個月後網站上線了（中、英、德三個網站），作品很好，

反應很大，第一次很有成就感，也很滿足。

（看評論，哪有那麼多好運氣，如果我不硬着頭皮找工作，去學習，哪有機會，我只有邁出那一步，纔有機會走下一步，如果我當初按兵不動，說不定我現在還是個廠妹）

我比較喜歡畫畫，個人愛好，好多手稿被搬家搞丟了

2014 年 3 月，20 歲，無聊，靈感來了，花了兩天做了一張個人簡歷，以設計形式做出來，

發佈到設計網站上，被推薦，轉載多次，

然後被一家軟件公司挖去，他們公司是外包公司，（其實他們要求也嚴，HR 給我發 offer 的，給我打了個電話通知我，說：你學歷很低，一般我們公司不會招收這麼低學歷的，但是你是你們主管看上的，他覺得你作品很不錯，不在乎你學歷，所以希望你好好努力）

那時學歷才中專，你們問我學歷重要嗎，學歷是敲門磚，我覺得重要

零 UI 基礎的我被派去深圳華爲（華爲外包對學歷稍微鬆一點，但是也抓的很嚴，部分看面試你的人吧）

爲什麼沒有被派到南京華爲，南京點名說只要本科朝上學歷，

2014 年 4 月到達深圳，開始了爲期 8 個月的華爲（外包）工作，

零 UI 基礎的有些困難，剛開始是無休止的畫 icon，畫的還不錯，

後來就負責了華爲內部部分 icon 的繪製（icon 繪製可以用 Ps 與 AI 這個完全看個人習慣)，

業餘練習，無關公司

由於畫的好，領導開始讓我負責 app 了，第一次接觸還是有些緊張的（app 工具用 Ps，方便切圖與開發，現在有新軟件 Sketch，部分開發會要求你用 Sketch 來做）

趁晚上回家時，在網上搜了些兼職，接一些軟件的項目來做，先自己練了一下，每天熬到凌晨兩點，人稍微有點壓力太大了，

一些兼職交互手稿，有點凌亂

兼職的交互需求功能擴展

因爲華爲這種公司要求很嚴，會有七天的試用期，不合格會被退掉，我覺得退掉是一種丟人的事情，所以只有努力，

做了一些兼職後，可以上手，雖說不精，但是好在懂了，不至於出錯太多

然後就慢慢好起來了，華爲還是很磨鍊人的，

2015 年調回南京，3-6 月生了一場大病，從而辭職在家，

病好了後，開始工作，但是已經明確的做 UI 了，

可以說深圳一年提升很大，無論是設計，還是這個行業，收貨很多。

2016 年 5 月，我還是在做 UI，目前是一切都步入正軌了，

我工作 5 年了，設計經驗三年多一點，但憑藉 3 年的多數兼職，

能力跟正常人的三年比起來還是稍強了一些。

94 年的我，年紀還算小，哈哈，我是我們公司最小的那個，哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈……

現在晚上下班偶爾兼職，兼職也會做一些交互，後期有考慮往交互方面發展（交互工具 Axure Sketch）

關於設計，我總是在努力，我會想很多辦法，嘗試新軟件，

學習新的東西，去提升自己，前段時間剛出的 Axure（支持 Windows 與 OS X 系統）跟 Adobe Experience Design 我也有嘗試（目前只支持 OS X 系統）

作爲一個落後的農村出身，沒錢，沒學歷，沒鼓勵，沒人脈，有的只是青春的叛逆期，與一腔熱血，

我以前沒想過能不能成功，我想的是，我應該嘗試去做，即使我失敗了，但是我曾努力後，也不至於後悔

雖然我現在並不是很成功，但是我已經很知足了

我的學歷比較低，目前已經抽出一半的兼職時間來自考，學習。

忘了說了，我報的自考專業是英語，一方面爲了工作，英語好，可以加分

另一方面，以後有出國的打算，或是想在一些國外網站上兼職（目前只是想法，還在努力，希望以後可以實現）

（手機上傳有點影響像素，又或者我上傳方式不對）

畢竟學歷是個敲門磚，不求太高，但是也不想太低，中專終究還是太差了。（做軟件行業，學歷幾乎都很高，每次入職時交的學歷證明，我都很自卑，因爲我看別人都是本科，碩士，我卻是個中專，這個感覺真的很不好）

我比較喜歡看書，上面最喜歡《飄》（飄是看完了紙本書，又看了一邊電子書，改編的電影亂世佳人也看完了）

《荊棘鳥》也是我喜歡的書之一，看了好幾遍了，看到拉爾夫去世的時候，我還哭了好久，

我很喜歡午後有陽光，有書，有一杯啤酒 ，

對，你沒看錯，就是啤酒，我很愛喝啤酒，我不喜歡喝咖啡，沒有那個調調，看書講究隨意，個人舒服就好，

左邊的兩個厚厚的本子全部寫完了，密密麻麻的單詞與句子，現在已經開始寫第四本了，

其實我很遺憾沒有上大學，我覺得大學校園是青春最美的時光，我很遺憾，但是不後悔

我沒有什麼太大的追求，

我就是想讓生活變好一點，工作穩定一點，收入高一點，誰不想更好。

貧不足羞，可羞是貧而無志。

當然大家的生活態度也不一樣，因人而異

很多人說很幸運，我有那麼一丟丟的幸運，但是還是付出了很多努力，別看我的結果，也看我的辛苦，哪會有那麼多的運氣，公司看的是你的能力，執行力，技能能力，不是運氣，運氣當不了飯吃，大家還是都實際一點吧

有人說歸咎天賦很重要，人有多少天賦，舉例子爲什麼人的穿衣品味會越來越高，你看多了淘寶各色各異的賣家搭配，你看衆多明星的穿衣打扮，你每天都在淘寶，都在各路網紅，看多了，品味能低嗎，你也會爲了好看嘗試着學別人搭配，審美慢慢的提高，你會開始自己搭配，知道搭配纔會好看，你也知道哪些搭配會顯得 low，這是天賦嗎？

不知道是不是我表達不到位，部分人可能覺得設計容易了，我只是想說，看東西不要看表面，請深入瞭解，好好學習纔是硬道理

我並不厲害，也沒有自誇，公司給我定義的是中級 UI 設計師，你看，我離高級還遠着呢，比我牛逼的人多的是，在牛逼大神的眼裏，我還是很差的，關於這篇文章，我只是很符合題目的述說了轉行過程

有人說我有點誤人子弟了，我也不知道誤沒誤，

但是還是謝謝你們的贊與鼓勵，也希望你們好好努力，畢竟技術過硬纔是真本事，

還有大部分說要不要繼續學習的，想學習的還爲時不晚，植樹的最好時機一是十年前，二是現在

另外希望在校的學生們好好學習

工作的人好好工作

願你們都好

祝福你們
