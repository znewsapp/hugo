#抑鬱性沉思#鑽牛角尖，越想越壞？讓你隨時活力回來！

鑽牛角尖會上癮的。難過煎熬的時候，越是常常胡思亂想的人，越不由自主的陷進去。他們和#抑鬱性沉思#走的更近，也和抑鬱走的更近，這樣說會不會給人一種欽定的感覺？

日內瓦大學的 Piguet 等發現，容易捲入牛角尖裏的人，在低難度任務甚至休息的時候，#內嗅皮層 // 亂七八糟你自己的回憶都在這兒#以及巴拉巴拉的更興奮，#視覺區 // 感知斑斕世界的核心接口#、#右側腦島 // 當前自己靠不靠譜的檢察官#以及巴拉巴拉的更抑制；在高難度任務的時候又看不出什麼差別。

看樣子，「沉思者」們在沒什麼負擔的時候，已經偏愛放空世界、沉浸在自己的混亂記憶中。說不準就調用起前一場無疾而終的戀愛，上一段年輕天真的糟心事。正當時再有些物是人非勾起回憶，念想就飄啊飄啊落在所有你的傷口上。

這一切的罪魁禍首是我們無處安放的注意。RAM 的佔用率沒有因爲你不跑應用下滑到底，系統進程和自啓應用趁着山中無老虎，瞬間瓜分了你的 RAM 資源。如果現實中沒有什麼刺激獨自佔有你的注意時，#內嗅皮層#中的自啓進程不由分說的奪走了 TA，TA 們就這樣沒羞沒臊 // 痛心疾首的在一起了。

目前來看，我們還沒辦法在大腦裏植入 360 或者百度大禮包，「一鍵清理」也無從談起。微不足道的事情，可能喚醒我們的#抑鬱性沉思#，我們試圖搜索所有與之相似相關的記憶，希望給出答案 // 爲什麼變成了這樣？爲什麼每次都是這樣？是我的錯還是世界的錯？當注意收窄到這些抽象到「宏大敘事」樣的問題，就很難再逃逸出來。

那麼問題來了：鑽進牛角尖，「神」丟了，能讓你隨時找回狀態的動力在哪裏呢？

1 注意你的注意

什麼能讓操場上追逐亂跑的「熊孩子」們瞬間聚在一起？——上 / 下課鈴。對付我們的注意，也可以用上同樣的手段。

#抑鬱性沉思#的核心要件是彌散開的注意，如果把注意聚攏回來，自然走不到鑽牛角尖那一步。值得慶幸的是，我們手邊就有足夠豐富的資源助我們一臂之力，比如電腦比如手機比如 VR 眼鏡。

開一盤 LOL 至少能在草叢和河道盯上 20 分鐘，來一局消消樂也能在紅黃藍綠上搜索差不多 5 分鐘，VR 電影那種身臨其境的感覺對注意簡直霸道總裁的存在。也不只這些，任何有趣的有吸引力的有挑戰性的能達到#心流#體驗的工作，都是最便宜的抗沉思「良藥」。

2 豐富生活的滋味

心理學上沒什麼事兒是一次#放鬆#解決不了的；如果有，那就兩次。

如果你不會點#放鬆訓練#，已經不好意思和做心理諮詢的朋友打招呼了。然而這裏指的不只是放鬆訓練，還有真正生活意義上的放鬆。如果在注意上做不出文章，那不如給沉思換個場景，勾不起愛恨情仇的#自傳式回憶#，就沒有涌上心頭的消極評價。

甭管是遊樂場還是健身房，或者午後慵懶陽光裏愜意的咖啡店。總有些舒適安全值得依賴的地方。生活又不是挫折失敗再挫折再失敗，總該有些有滋味有情調的場景，事實上在美好中的沉思也能讓幸福更持久 // 並不是春秋大夢。

3 一罐富含維生素的傾訴

內部言語和外部言語的偏好不太一樣，自問自答式的追問通常指向了抽象的、形而上的評價。

傾訴倒苦水的時候，我們可不能斷斷續續的反覆穿插回溯式的敘事，這樣反而有益於重新捋清事情的脈絡，不再陷於#抑鬱性沉思#的思維中痛不欲生，說不定隨着傾訴事情也就迎刃而解了。

傾聽者情緒上的迴應和陪伴是不可或缺的維生資源，沉思帶來的傷感低落痛苦抑鬱需要一個出口傾瀉出來，而他們回覆的動力更少不了傾聽者的心理支撐。

4 堅持服食，效果拔羣

鑽牛角尖會上癮的，即使一點都不舒爽；戒除也需要付出代價，痛苦卻變得越來越少。

很多人的#抑鬱性沉思#已經成爲一種習慣，無疑要花些精力時刻注視着自己的動向，我們要把火苗掐滅在初生的那一時間。希望靠一兩點建議就能「狀態回神」，希望在鑽牛角尖上一蹴而就，毫無疑問沒有實現的可能。不過堅持着注視，堅持着改變一段時間，一定會發現「生活根本沒有那麼艱難，只是我的#內嗅皮層#欺騙了我。」
