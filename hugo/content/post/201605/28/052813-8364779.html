領完鑰匙都該裝修了，你卻還是不知道做什麼？（全文無配圖）

一般來說，我們可能 5 到 8 年才需要面臨一次重新裝修，而非常多的年輕業主在此次裝修之前甚至沒有任何相關經驗。

鑑於這樣的狀況，對於新業主來說，無論找不找得到合適的設計師或者靠譜的裝修公司，只有自己把功課儘量做到位才能最大程度地減少遺憾。

爲了幫大家理清頭緒，接下來，我還是先列舉幾個整體構想時需要考慮的問題和相關知識點，第三部分會列舉一些建材家居產品選購中可能遇到的名詞。

除本文所述概念名詞外，如需瞭解關於燈具光源選購、傢俱板材挑選、小戶型收納規劃等方面的知識，可以查閱知乎專欄內其他文章，也可通過公衆號“ 器研新舍 ”查看歷史文章。

以下爲簡要提綱及分段敘述。

  • 預算規劃與初步設計構想
  • 設計施工之初需要考慮的其他問題
  • 建材家居產品選購中可能需要了解的名詞

【一】 預算規劃與初步設計構想

    • 拿到新房鑰匙或者是確定了交房時間後，首先是抓緊時間去做審美提升和裝修知識儲備，爲期 2 到 4 個月。
    • 可以通過 Pinterest、好好住、Houzz 等線上社區，以及宜家商場、無印良品店面、HAY 店面、本地最好的建材家居賣場等途徑瞭解一些當下比較流行的家居產品和場景案例。
    • 如果想參加每年春秋兩季的家居賣場促銷活動，請在 3.15 之前做好初期規劃，如若不然，切勿隨意交定金參加各種活動。
  • 請勿輕易相信各種全包裝修的定額優惠套餐，試想，自己需求都不明確，方案根本沒定，如何得出施工報價？
    • 通過 2 到 4 個月的瞭解，自己先做一個初步設計規劃或者是需求清單，然後聯繫裝修公司或者設計師進行溝通確認。
  • 在與設計施工方溝通之前，要先確定一個搬入新家前能拿得出的最大預算，然後以這個預算的 90%爲基準，自己列一個預算表格，以便在初步設計施工方案完成後，對裝修施工費用、設計費用、主材輔料、傢俱家電、其他設施、各項雜費進行初步覈算，無明顯超支且分配相對合理時，再行開工。
    • 在沒有列出整體入住預算前不要做任何具體規劃，在做好整體規劃之前不要發生任何實際採購行爲。
  • 如若現階段總入住預算明顯不足，切勿選用廉價地磚（約摺合每平米人民幣 160 元以下）用於客餐廳區域，哪怕按照鋪設地板的要求先行進行水泥找平，暫用其他地毯地墊類材質鋪設地面都可以考慮。
    • 裝修設計時要隨時注意，儘可能多給“人”留出活動空間，如果確有較多的集中收納需求，請優先考慮地臺牀而不是頂櫃吊櫃；
  • 如家中只有一處主要的衣物儲存區，儘量考慮多掛放少疊放，確實需要疊放時請增加櫃內多層不鏽鋼拉籃或薄型抽屜，如房間較寬，也可以考慮薄型衣櫃或牆面掛架收納系統。
    • 另外，在設計構想時要儘量考慮減少衛生死角，尤其是小戶型住宅，一旦入住，幾乎沒有時間空間進行大的清理和格局改動，一定要做好設計預想。
  • 在窗簾遮光方式的選擇上，要根據採光通風條件和居家生活習慣，合理選配遮光窗簾、捲簾、百葉窗、紗簾，以儘量達到既能保護日常隱私，又能保證採光，還能降噪保溫的目的。
    • 在計算鞋櫃深度時，注意要在總厚度基礎上考慮減去背板和門板的厚度，覈對內空淨深度是否大於家庭成員最大鞋碼的鞋子的外部尺寸，還要了解一下櫃門是選用直臂鉸鏈還是半彎鉸鏈還是全彎鉸鏈，櫃門開啓方式的不同，會影響到靠邊一側櫃門的使用，也可能與門套線相互衝突或被迫留縫。另外，相鄰櫃門之間的安裝間隙，也會影響櫃門的使用壽命，尤其是烤漆櫃門。
  • 若非頂部暖通管線走向所需，吊頂能少可少，也可以考慮減少純上翻式的吊頂使用，無論有無預留燈帶。
  • 衛生間的設計中首先要注意馬桶坑距、地漏排水坡度及排水方向，其次是選擇更耐髒更耐看的牆地材質。而如果沒有單獨的淋浴區，或者說做不到先淋浴再泡澡，最好不要只裝浴缸，以及，淋浴區的玻璃隔斷必須要有防爆膜。

【二】 設計施工之初需要考慮的其他問題

有了一些關於裝修設計及施工採購的知識儲備以後，你很容易就可能會發現，市場上常見的所謂“半包”施工及“全包”裝修的概念是很難滿足現實中完整的設計施工採購需求的。

在整個前期準備直到最終入住的過程中，所有的工作都是需要有人來做的，所有的銜接事項也都需要人來安排，裝修公司半包了，那剩下的事情誰來操心？

大多數情況下，只能是業主自己。

以常見的施工方式來說，裝修公司在半包業務裏能爲業主提供的服務基本上包括：牆體改造及清渣、水電改造及試驗、防水工程及試驗、牆地面補平找平、鋪設瓷磚、吊頂工程、牆面基層處理及批刮打磨、安裝地板、貼壁紙及噴刷牆漆、電視牆設計施工及部分現場製作的櫃體、一部分安裝服務等，個別裝修公司也在半包服務中幫業主協助選購製作櫥櫃。

而在裝修公司未提供整體設計，且因爲各種原因沒找到合適的設計師的情況下，除去上一段所述各項工作，業主自己還可能需要哪些決定呢？

基礎設施設置及改造

  • 是否更換外窗和入戶防盜門；
  • 強電箱、弱電箱的位置和類型，是否需要更換或改造；
  • 網絡通信設備的選用和放置位置；
  • 是否選用新風系統；
  • 新風系統主機位置和送風迴風路徑；
  • 各空間的空調設備如何選配；
  • 各房間空調預留孔位及中央空調室外機位置；
  • 是否自備地暖；
  • 冷水處理路徑和設施選購；
  • 排水排污路徑是否需要改造；
  • 不同用途和使用場景下的熱水供應方式；

廚房 / 餐廳 / 與廚房連接的陽臺或生活陽臺

  • 尋找合適的櫥櫃廠家；
  • 雙方對廚房平面尺寸和管線點位進行確認標註；
  • 決定是否需要洗碗機（確認型號）；
  • 水槽類型（確認尺寸）；
  • 是否需要垃圾處理器（型號尺寸）；
  • 選擇集成竈還是煙機竈具組合；
  • 初選吊櫃尺寸；
  • 選擇冰箱容量（冰箱型號）；
  • 是否需要高櫃；
  • 是否需要中央軟水機；
  • 如何選配燃氣熱水器 / 壁掛鍋爐 / 水箱；
  • 選擇洗衣機 / 烘乾機 / 洗烘一體機；
  • 確定餐桌尺寸與朝向；
  • 確定固定餐椅的數量和類型（扶手、靠背、卡座）；
  • 確定是否需要餐邊櫃以及餐邊櫃尺寸；

主臥衛生間

  • 衛生間的設計中首先要注意馬桶坑距及朝向、地漏的排水坡度及方向；
  • 其次是選擇更耐髒更耐看的牆地材質；
  • 根據點位尺寸決定能否三式分離（沐浴 / 如廁 / 洗漱）；
  • 是否選用壁馬桶及智能蓋板；
  • 如果沒有單獨的淋浴區，或者說做不到先淋浴再泡澡，最好不要只裝浴缸；
  • 淋浴區如有玻璃隔斷，必須要貼防爆膜；
  • 洗面盆選用臺上盆還是嵌入式臺盆還是半嵌入式面盆；
  • 選用一體式陶瓷面盆加下方櫃體還是單獨懸空面盆；
  • 如何選配鏡櫃、薄型櫃、側面落地收納櫃、成品樹脂櫃架；

公共衛生間

  • 除主臥需要考慮的事項外，還注意地漏位置的不同，以確定公衛如何分區；

家務間 / 儲藏間 / 衣帽間

  • 能否留出單獨的家務間或儲藏間或衣帽間；
  • 如有可能，聯繫廠家進行商議設計；
  • 注意控制總預算，定製衣帽間很容易超出預算；

衣櫃

  • 選用 60cm 深度的推拉門衣櫃還是對開門衣櫃；
  • 還是選配 30-60cm 推拉門或平開門；
  • 還是選擇牆面掛架加布簾的方式當做衣櫃；
  • 選用型材櫃門還是板材吸塑櫃門還是板材烤漆櫃門；
  • 選用實木原色櫃門還是實木開放漆色漆櫃門；
  • 還是選用手繪櫃門、木框布面櫃門或其他櫃門；
  • 櫃內搭配宜家收納筐還是網購其他品牌收納筐；
  • 不同房間內衣櫃的掛放區最大長度和其他功能需求；
  • 是否雙層掛放、是否增加頂櫃；
  • 櫃內照明是使用 220V 變壓供電還是電池供電；

臥室

  • 選擇各臥室牀身類型（軟包箱式、板式箱式、木框架、鋼架）；
  • 選擇各臥室牀墊類型及尺寸（海綿、乳膠、一體彈簧、袋裝彈簧、乳膠彈簧）；
  • 是否需要牀頭桌（電腦桌、小書桌、梳妝桌）；
  • 牀頭櫃是單獨採購還是與牀身同時選購；
  • 是否現在就預留嬰兒牀位置；

客餐廳

  • 起居室照明組合方式和各不同區域照度需求；是否選用壁燈和吊線燈；
  • 選用中央空調還是立式櫃機；
  • 沙發的預計長度、是否帶扶手、是否帶單獨腳踏、是否選用貴妃位；
  • 兩人位加轉角、一字三人位、一字三人位加同色腳踏、三加二、三加二加二沙發；
  • 長方形平板茶几、長方形單層收納茶几、長方形大容量茶几、正方形單個茶几；
  • 橢圓形茶几、圓形單個茶几、其他茶几組合類型；
  • 金屬管棒支腿茶几、石材檯面茶几、實木白色烤漆茶几、其他材質茶几；

其他事項

  • 玄關是否使用不同材質區分淨區和髒區；
  • 玄關位置是否有進門置物檯面和換鞋凳；
  • 陳列櫃需求（紀念品、玩具、小擺件）；
  • 書架書櫃最小容量（或者預計有多少本書）；
  • 非百葉窗非捲簾非電動窗簾的普通窗簾的軌道或掛杆樣式；
  • 各處踢腳線的高度、顏色、材質、購買途徑；
  • 做好工序驗收、成品保護、油漆遮蓋、補縫收尾；
