黑產主要是做什麼的？

看見知乎上有很多關於「黑產」的討論，看得出來，隨着互聯網的快速發展，大衆對黑產的關注度也越來越高。在回答這個問題之前，我們先來看看 2017 年幾件與黑產相關的事件，你會發現這種不斷提升的關注度是有原因的。

1、「9.27 特大竊取販賣公民個人信息專案」；

2、非法提供「秒撥」動態 IP 服務案；

3、「暗夜攻擊小組」等 DDoS 攻擊系列案；

4、通過 AI 技術破解驗證碼的打碼平臺非法獲取販賣公民信息案；

這些規模「億」級的案件是 2017 年網絡罪案的典型案例，背後有完整的黑色產業鏈支撐，規模之大，案情之複雜，策劃之周密，足以反映當前黑產逐漸走向頭部犯罪的趨勢。下面，我們就來聊聊黑產到底是什麼。

黑產是什麼？

黑產是黑色產業的簡稱，廣義上包括販毒、高利貸、私彩、網絡詐騙等行業，狹義上是指非法利用病毒木馬來獲得利益的一個行業，現在提到黑產一般指狹義上的黑產。黑產一般有交易鏈，通常在搜索引擎搜索不到、一般人也難以進入的暗網進行大規模交易，也會在私人網站或者交易論壇進行交易。

黑產在做什麼？

黑產利用病毒木馬非法盜取或者通過機構泄密人員購買他人身份證號、手機號、遊戲帳號、郵箱、家庭住址等私人信息，以及被盜者家庭成員的上述隱私，甚至包括所養寵物的信息，然後進行非法販賣。具體如下：

1、非法獲得大量用戶賬戶和密碼數據。獲取的方式可以是通過黑客手段攻擊一些網站，或者向一些非法人員購買，一般價格是每 10 萬條 50-100 元。

2、驗證賬戶和密碼數據的準確性，這個過程一般被叫做「曬密」。曬密最常見的方式是「撞庫」，即通過專門的軟件或程序批量訪問郵箱、社交軟件等。經過撞庫曬密後獲得的有效數據，價格會更高，一個賬戶能賣到 1.2-1.5 元。

3、利用賬戶和密碼數據獲得更多用戶信息，進行敲詐、勒索等詐騙操作。經過曬密並進一步獲得的用戶精準信息，價格往往更高，非法利潤是驅動黑產的主要因素。

黑產會經過多重洗劫，比如遊戲帳號密碼會先將該帳號的裝備金幣等進行折現或者轉移，然後再賣給下一個買家，經過再處理後再銷售。

黑產有哪些技術手段？

對於黑產分子來說，「曬密」技術難度很大，阻力在於平臺的安全策略，最常見的是驗證碼和 IP 限制。舉例來說，驗證碼包括數字、文字、圖形，甚至需要拖拽操作等更高難度的驗證方式，用以區分人或機器行爲。IP 限制包括諸如同一個 IP 在一定時間範圍內只能登錄一次或者幾次，或者僅限指定區域用戶登錄等措施。

爲突破網站的安全策略，在黑產中出現了「打碼」平臺和「秒撥」動態 IP 服務。

1、打碼

軟件黑產商售賣幾十種甚至上百種撞庫軟件，此類軟件一般都集成有「打碼」功能，通過鏈接到打碼平臺實現對驗證碼的識別破解。打碼平臺往往採用人工智能深度學習技術訓練機器，使其有效識別字符、圖片等驗證碼，大幅提升驗證碼的破解率。對於極其複雜、機器難以操作的情況，打碼平臺還提供基於衆包的人工打碼解決方案，發展大量網賺人員，以人工方式識別驗證碼。

2、秒撥

「秒撥」動態 IP 黑產服務的背後，是可調用全國甚至國外的 ADSL 寬帶動態 IP 資源，面向非法分子的界面只要通過簡單配置，就可以實現 IP 的「自動切換」、「秒級切換」、「斷線重撥」、清理 COOKIES 緩存、虛擬網卡（MAC）信息、多地域 IP 資源調換等服務，規避網站的限制策略。

當黑產人員獲得了一批用戶名和密碼信息後，通過打碼平臺和秒撥服務就可以突破大部分網站的安全策略，獲取更多個人隱私信息，造成更大的安全威脅。

除了撞庫曬密以外，黑產還有以下手段：

1、電商平臺放出優惠券和優惠碼，或者紅包獎勵，以此進行拉新或促銷，往往限制一個 IP 僅能參加一次領券活動。此時有黑產人員通過機器方式突破 IP 限制大量參與活動，用非法批量註冊的帳號獲取優惠券，這種行爲被稱爲「薅羊毛」，這些人被叫做「羊毛黨」。

2、電商平臺的商家爲了獲得某些權益寄希望於商品刷單，自媒體人爲獲得在客戶眼中的所謂影響力要進行閱讀刷量，第三方刷單刷量人員都會採用到動態 IP 服務規避平臺限制策略。

除了在互聯網領域對商家及個人造成危害以外，還會威脅國家網絡安全。市面上大部分動態 IP 黑產服務商，都有橋接境外多個國家的服務器、雲主機等國際網絡鏈路資源，能夠接入境外服務器，用境外 IP 訪問非法網站、發佈非法信息，包括訪問暗網、國際信用卡 CVV 盜刷、境外帳號作惡等多種場景。

因爲有惡意商家競爭需求、黑產分子牟取暴利的慾望等，「秒撥」動態 IP 服務得以快速發展，甚至在電商平臺上堂而皇之的售賣。

技術進步與互聯網發展讓我們正在進入一個「平臺 + 個體」的時代，大平臺往往提供了資源和能力等基礎設施，而個體利用平臺對羣體產生的影響力正在變得越來越強大。在技術賦能下，個體創新能力越來越大，作惡的能力也越來越強，一個個體完全可以操控黑產，對更多人或者組織形成較大的危害。我們之前在【個人信息的泄露在今天已經嚴重到了什麼地步？對普通人的生活有多大的影響？】中，具體聊了黑產導致的個人隱私泄露對我們有多大的危害，歡迎查閱。

隨着人工智能、雲計算、大數據等技術的發展，黑產也正在走進人工智能和雲計算時代。我們更要注重提高防範意識，更多情況下，需要平臺對個人信息加以保護。

在打擊黑產的路上，每一小步都是大勝利。什麼時候離光明最近？那就是你覺得黑暗太黑的時候。
