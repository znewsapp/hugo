布袋戲的取名是遵循了什麼原則？

布袋戲人物名字的完整組合，主要是“職業 + 稱號 + 姓 + 名（化名）”，最典型的突兀感就是姓名一體而無姓。

稱號，也就是混名，在江湖上混而取的名字，有自己取的，也有別人送的。

早期布袋戲的人名主要是稱號 + 常規模式的姓名，如 雲州大儒俠 + 史豔文。史是常見姓，雲州是史豔文的老家，大儒俠是因爲他奪得文武雙狀元，別人給的稱號。他的職業是太子少師什麼的，還有人送外號"五皮賢人"。

還有萬惡罪魁 + 藏鏡人。藏鏡人本名羅碧，萬惡罪魁即是自稱也是他稱，藏鏡人則是指他一直都隱藏在鏡子裏行動，後來通用做姓名。早期的名字都還算正常，姓氏不算少見。

霹靂布袋戲中，開始走取帶有玄幻氣息的名字。比如清香白蓮 + 素 + 還真。清香白蓮是他一出場就自帶的稱號(以後大多角色均是如此)，素還真是姓 + 名，素姓算是少見姓，但還不算自造。同樣還有脫俗仙子 + 談 + 無慾。

此時稱號和名字可見有下功夫。素爲顏色還真，清香爲氣味還真，而白與素同義；仙子是人脫俗，無慾是心境脫俗，(早期有人懷疑根據對仗應該是淡無慾，如此的話淡就是程度脫俗)

此時姓與名與稱號均是關係密切。

同時期的有單鋒劍尊 + 宇文 + 天，黑白郎君 + 南宮 + 恨等。同時也有凍液成體 + 蔭屍人，照世明燈 + 慈郎，隱蔽紅塵 + 一線生，荒野金刀 + 獨眼龍這樣的有稱號但姓名不分，而該種角色的姓名也多能召示人物特點，無論是形象性格還是武功。姓名不分是個代替常見姓 + 名的有效取名法，也因此出現了不少不知道人物姓啥的各種奇特名字。而沒有稱號只有名字的角色多是二線三線角色。沒有名字只有稱號(多是職稱)的角色如文帝武尊，則多出於組織之中。

不少一線角色或主要角色出來時，就已經成對湊好，而對應角色多是夥伴或對手。典型名字就是"百世經綸 + 一頁書"和“寰宇武典 + 半尺劍”，意爲百世經綸載於一頁紙內，天下武學盡納半尺劍中。完整的對仗，同時也凸顯角色的狂傲。

中期出現了以三先天爲代表的四字人名，此時的名字絕大多數均無法分離姓氏，多是截取前兩字加敬稱，如劍子先生，佛劍大師。

而後出現具有相當規模的有系統的組織後，職稱開始代替混名大量出現。諸如異度魔界，紫耀天朝，內部人員稱呼多用職稱稱呼。如禍皇 + 六禍蒼龍。

而到震寰宇時期，出現了一堆堆的四字名字，典型的就是詩意天城和慈光之塔的大部分人。這樣的情況大約持續到聖魔時期，除卻一堆和尚的法名，其餘不少人名均回覆到四字稱號 + 三字名字甚至是姓名的模式上，而且四字稱號都很完整，幾乎是個人名都有四字稱號，在古代可是很稀罕的，也說明了時代的進步……（有姓的好處就是在稱呼上可以有敬稱，比如素賢人，靖先生）。

進步到創神時期，出現了大量的，大量的，大量的生僻字（江郎才盡，不過也沒辦法，名字取了那麼多，啥類型的都取過了，沒點生僻字混不起來啊），如紅冕七元。

諸如倦收天這樣的利用反差造成氣勢的名字已不多見，或者說這樣的名字本身就容易造成審美疲勞。

關於取名，其實最主要的是要多看書，古書中有很多可以借鑑的詞彙和片段。比如山龍"隱秀"，靈犀"指瑕"，"神思"，均是出自《文心雕龍》，朝如青絲 + 暮成雪，出自李白《將進酒》，塵外孤標 + 意琦行，語出成語"塵外孤標"和"瑰意琦行"，均形容人超凡脫俗遺世獨立。
