C羅傷退替補加時絕殺，葡萄牙擒法國奪冠

2016 歐洲盃葡萄牙不敗奪冠，歷史上首次奪得德勞內杯，有哪些亮點和意義？

終於，也有一隻我喜歡的球隊能拿到冠軍了。

2000 年加時點殺，2006 年點殺，大仇得報。

順便，把 2004 年科斯塔和 C 羅流在自家門前的眼淚，一併送給法國人。

0.

有一個葡萄牙系列的腕錶，定位說明是這樣的：

葡萄牙的航海先鋒們能夠成功駕馭看似水火不容的特質，脾氣剛烈衝動卻又善於冷靜計算……

1519 年，5 條船出發；1522 年，18 人凱旋。

麥哲倫們咬牙堅持着不想死，一不小心就環球了。

葡萄牙隊咬牙堅持着不想輸，一不小心就冠軍了。

船長倒下，壯舉完成。

1.

這隻葡萄牙隊，算是個人實力的窪地，卻恰是當之無愧的團隊戰鬥力巔峯。

7 場比賽不敗，可也就只有一個 90 分鐘內的勝利。

四場淘汰賽三場加時，最累球隊沒有之一。

半決賽缺佩佩 + 威廉卡瓦略，斷了半條中軸；決賽傷 C 羅，還沒正式交鋒旗幟倒下了。

可葡萄牙人就是一球一球的咬，一腳一腳的拼，一次一次的追，沒有半點動搖、鬆懈、彷徨。

最後，冠軍。

2.

將每一個弱（強）隊當強（弱）隊打，一招“抱住對方滿地打滾最後爭取站起來走下去”貫穿全程。

小組賽第三，最後一場三次落後三次扳平，堪堪小組第三。

4 場淘汰賽，3 場加時，卻永遠不用體力當藉口，除了決賽一直沒人倒地抽筋。

淘汰賽階段，每個人都煥然一新，招之即戰，戰就戰到底，面對傷病和局面上的被動，冷靜應對。

遇弱不急，遇強不慫。

3.

英雄不是一個人，每個人都是英雄。

調整後效果奇佳的佩佩 + 馮特中衛組合。

兩個邊後衛神一樣的體力。

誇雷斯馬、阿爾維斯、德尼洛、穆蒂尼奧，完美完成任務的替補席戰士。

慢吞吞可位置感不錯，憨呼呼可傳球選擇上乘的鐵腰威廉卡瓦略。

艾德，決賽替補登場，奉獻了德羅巴級的樁鋒演出，本屆盃賽樁鋒時尚的華彩絕唱。

身體羸弱但精神堅如磐石，對抗不足可回防絕對積極，每每在對方鬆懈之時露出獠牙的前場攻擊組，納尼，馬里奧，席爾瓦們。

桑切斯，缺陷和優點一樣顯眼的超級新人。

C 羅，領袖。

4.

以及，我心裏的 MVP，帕特里西奧。

5.

從 2008 年開始，發現葡萄牙實力走弱，但精神屬性逐漸超過了黃金一代。

到這屆，可以光明正大將精神屬性視作我葡獨一無二的殺手鐗。

而這恰好是競技體育最能打動我的元素。

不輕敵，不認慫，不放棄。

冠軍是對這種精神最高的褒獎。

1996 德國如是，2016 葡萄牙亦如是。

6.

足球，至少在我自個心裏，永遠是傾向精神強者的運動。

如何評價 2016 歐洲盃決賽法國 0:1 葡萄牙的比賽？

上一場，德國和法國的比賽結束後，我在自己的答案裏這麼提了一句：

這屆歐洲盃到了決賽，很有意思的一組對決。有意思的地方在於，你閉上眼睛去想兩隻球隊對戰的時候，很難想象出他們會踢成什麼樣，言外之意肯定有知友能領會得。

嗯…這番話，於葡萄牙而言有失公允。

看球的時候，我突然想起來：

上一次法國隊有成熟的戰術體系與套路，已經不知道是啥時候了。

決賽，法國一如既往地搶開局。並且，面對中場控制力不足的葡萄牙隊，法國人努力想把場面和球權控制在自己手裏。在比賽的前 30 分鐘裏，一切勉強還算在法國隊的計劃之內。尤其是，C 羅下場的那一刻。我相信法國人已經情不自禁在想冠軍的事情了，這和輕敵無關。

結果 C 羅下場後，法國隊幾乎沒有任何套路將自己的優勢擴大。甚至，當比賽超過 65 分鐘，大家體能進入瓶頸期的時候，葡萄牙隊開始逐漸搶回了主動權。而最具標誌性的節點，是兩位主帥幾乎同時的一次換人：德尚用吉尼亞克換下吉魯，和之前幾場一樣；桑托斯，用埃徳換下了桑切斯。

德尚做這次換人的時候，可能都沒怎麼想過爲啥要換…就和科芒換帕耶一樣，他已經習慣了做這些對位換人。而桑托斯，則是隱忍了幾乎整場之後放出最後一記險招：撤掉中場攻守轉換推反擊的核心，爲球隊補上缺了將近一個小時的中鋒。信號和思路都很明確。

在這之後，比賽的形勢不斷在向葡萄牙隊傾斜。思路清晰且主動求變的一方獲得了優勢。而另一邊，直到丟球前，德尚居然都還沒用最後一個換人名額。加時落後才換上馬夏爾，真的當他是救世主嗎？埃徳的進球，有運氣的成分，但也是葡萄牙努力和拼搏的獎賞。

回頭看法國，這一屆比賽，對陣愛爾蘭的下半場是轉折點。那 45 分鐘是德尚徹底確立格里茲曼核心戰術的時候，但同時，也是他自己放棄思考更多戰術變化的時候。對陣德國已經顯示出備選方案的不足，而今天當天時地利都最靠近自己的時候，德尚卻沒有多進一招打翻葡萄牙的底氣。

－

多說一句 C 羅的傷。那一刻應該有些人和我一樣想到了櫻木花道的劇情。多希望他可以像漫畫裏一樣站起來，告訴全世界這是他代表葡萄牙隊最重要的一刻，然後回到場上爆射打進一球。但現實終究是現實。更何況，他還需要一個好的身體繼續踢下去。

當然，他有了一個國家隊層面的頂級榮譽。這一點上，他真的足夠偉大，也足夠幸福。比尤西比奧和菲戈幸福，也比吉格斯，伊布，舍瓦們幸福。仔細想想，真的，圓這個夢，太難了。

也許在好勝的 C 羅心裏，這個冠軍仍然稍顯不足，但無論如何，沒有人能否認這一路走來他的強大與堅毅。還有，這哥們真的太拼了。

當然，格里茲曼又一次輸給了 C 羅。但成長爲巨星的路上，有 C 羅這樣的巨人站在自己的對面，對他是有益無害的。在未來一段時間裏，格里茲曼仍需要不斷和 C 羅對戰。有偉大的對手，才能實現對自己的超越。嗯，歐冠亞軍和歐洲盃亞軍的滋味很難受。但，誰不是這樣過來的呢？

要說天賦，這屆比賽大概沒誰比法國隊更高了（也許是比利時？）。只是，這一次的功敗垂成，不知道會不會對這支充滿希望的球隊有什麼負面影響。當你看到科芒，馬夏爾， 格里茲曼，博格巴在一支球隊的時候，你會在想，到底會不會有主教練能把這幫天才捏合到一起呢？這批新的天才，能不能將這一切兌現成一座金盃，還是……

這場決賽，好看。
