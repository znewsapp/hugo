iPhone X: A Step Beyond

iPhone X 真正所關乎的、引領的，從來就不是什麼「全面屏風潮」，而是「making the physical object disappear into the experience」這件事。蘋果所定下的未來 10 年的目標，是讓體驗與現實之間的邊界最終消失。

在 iPhone X 產品視頻《Introducing iPhone X》的開頭，Jony Ive 用了這麼一句話來概括 iPhone X 的前世今生：

「For more than a decade, our intention has been to create an iPhone that is all-display. A physical object that disappears into the experience.」

過去的一年裏，所有人都認爲，iPhone X 所關乎的重點，一定是這一塊「全面屏」。但隨着發佈會越來越臨近，越來越多的聲音，包括我自己都開始意識到，「全面屏」本身其實根本不是問題的關鍵。

Tim Cook 說 iPhone X 要爲科技行業的未來 10 年指明方向，很難讓人不覺得這是一句大話。一塊長着「劉海」的全面屏，如何引領科技行業的未來 10 年？

從四周都有邊框的屏幕走向「全面屏」，這是 iPhone 所跨出的重要一步。但在 iPhone X 發佈接近三個月，開售一個月之後，我才終於完全釐清這件事：

iPhone X 真正所關乎的、引領的，從來就不是什麼「全面屏風潮」，而是「making the physical object disappear into the experience」這件事。蘋果所定下的未來 10 年的目標，是讓體驗與現實之間的邊界最終消失。

全面屏

就如 Jony Ive 所說，全面屏是一個夢想，這個理念在蘋果存在了 10 年以上。如果說十幾年前觸屏手機都還沒有成爲主流，很多人不一定能意識到這件事，5 年前 iPhone 已經席捲全世界的時候，我相信所有人都明白，iPhone 邊框越來越窄，直到整個正面都是屏幕的那一天，遲早會到來。

這是最順理成章的，「讓體驗與現實之間的邊界消失」的方式。當 iPhone 不再有屏幕邊框，內容就會好像浮現在空中一樣，和現實融爲一體。

在今天討論 iPhone X 改用 OLED 所引發的燒屏、色準有偏差、可視角度差等等問題，其實沒什麼太大的意義。這一不是蘋果買了低端的元器件，二不是製造過程中存在品控問題。至於這個問題最終到底是由 OLED 屏幕廠商自己解決，還是由傳聞中蘋果正在和臺積電共同開發的所謂「Micro-LCD」屏幕來解決，說到底都是顯示行業內的一個技術問題，這裏就不過分糾結了。

所以關於 iPhone X 上所配的這塊「Super Retina」全面屏，唯一值得討論的問題，就只剩下其形狀了。

關於 iPhone X 這塊屏幕的形狀，首先必須要釐清的事情是：這個「劉海」的存在，並不是蘋果在一塊完整的屏幕上「掏」了一個缺口，而是蘋果從 iPhone 8 的屏幕形態出發，把屏幕延伸至了左、右、下三個方向的邊框，以及頂部的兩個角，最終剩下了這麼一條 Notch。

全面屏不是天然存在的，是從設備中間的一塊小屏幕，通過廠商在工程上的不斷進化，才最終能儘可能做到貼近邊框。

所以這個 Notch，歸根結底還是一個「合理」的存在，無論你對其存在感到多麼地不適。就好像曾經的 iPhone 不防水，現在防水了，未來的年輕人肯定會覺得手機防水是理所當然的事，但在特定時代的特定的技術背景下，「手機不防水」就是「合理的」。今天的 Notch 也一樣。

我始終認爲，iPhone X 花費更多的成本，將屏幕延伸至頂部的兩個角，豎起兩隻耳朵，而不是像包括三星在內的很多廠商直接留下一整條黑邊，是非常明智的做法。

一方面在蘋果的軟件生態中，蘋果有底氣、也有條件去做一個「異形屏」，絕大部分開發者都會積極地去適配，讓自己的 App 在 iPhone X 上至少看起來不會「格格不入」，而同時也一定會有不少開發者迸發出靈感，利用好這兩片空間，做到一些令人驚歎的設計。

另一方面就是 Marco Arment 在 9 月的發佈會結束之後提到的那個觀點，這樣一片暫時技術上還無法去除的 Notch，讓 iPhone X 的輪廓是獨一無二的、極具辨識度的、簡潔明瞭的。

當然有人會說，那麼也有廠商在今天的條件下，幾乎做到了去掉 Notch 啊。

所以接下來要聊的，就是 Notch 的存在，是如何爲「讓體驗與現實之間的邊界消失」這件事做出巨大貢獻的。

Face ID

在 iPhone X 正式發佈之前，我對 Face ID 有着百般的擔心。

這會不會是因爲「屏下指紋識別」技術還無法量產才臨時上馬的備選計劃？會不會識別率很低？會不會對環境光線要求很高？會不會像某些手機一樣解鎖的同時還要先對準，然後強行預覽一下自拍？

事實證明所有的擔心都是沒必要的，除了 Craig Federighi 在發佈會上演示時出現的一個小誤會，Face ID 的實際使用效果幾乎獲得了壓倒性的一致好評。而不要忘了，這還只是第一代 Face ID，我們完全有理由相信隨着技術的進一步迭代，Face ID 在未來產品上的表現能夠更加令人驚歎。

我相信大部分人，包括三個月前的我，都認爲：之所以會有 Face ID，是因爲要做「全面屏」。因爲 Home 鍵將不復存在，所以 Touch ID 在 iPhone 的正面沒有地方可放，必須用一個什麼新技術來取代 Touch ID。

iPhone X 發佈之後我也有過疑問，如果未來一兩年「屏下指紋識別技術」成功量產，蘋果會不會又調轉車頭，回到 Touch ID 的老路上？

Craig Federighi 在《The Talk Show with John Gruber》上徹底澄清了這一點，蘋果從最初就確定並堅持 Face ID 是未來的生物識別技術發展的方向。

所以問題回到最初的原點，Face ID 的意義到底在哪？

第一，Face ID 比 Touch ID，進一步減少了用戶需要主動做的動作。相比 Touch ID 在驗證時需要把手指放到 Home 鍵上，Face ID 將「受驗物體接觸傳感器」這個步驟嫁接在了「人看手機」這個自然的動作上，讓體驗更沉浸，讓你可以更加忽視手機這個「物體」的存在，而全神貫注於內容與體驗。

這件事在 iPhone 上或許還不那麼容易感知。畢竟在 iPhone 上，「按 Home 鍵」本身也是一個相當自然的動作。但如果從 iPad 的角度去考慮這件事，就會變得特別容易理解。

Touch ID 在 iPad 上的存在，從一開始就並不是自然，其讓 Smart Cover 無法自然地做到「擡起保護蓋亮屏、進入主界面」。而在 iOS 11 的 iPad 橫屏交互邏輯裏，存在於屏幕一側的 Touch ID 更是顯得礙手礙腳。所以我纔會非常期待並準備換到明年可能發佈的配備 Face ID 的 iPad Pro。

第二，Face ID 將驗證授權這件事，從「離散」變爲「連貫」。

今天的 iPhone X 上，Face ID 在安全邏輯上的作用機制與 Touch ID 還基本是一樣的：提出授權需求，啓動硬件，檢測人臉，完成授權。

但隨着產品技術的迭代，Face ID 在工作機制上完全有條件帶來進一步的革命。畢竟當你在使用手機時，臉是永遠處於屏幕前的。如果 Face ID 可以做到更好的檢測效果、更低的功耗，以至於無時無刻不在工作，當用戶的臉出現在設備前時便開放安全權限，檢測不到時則立即關閉，這將給手機安全以及相關的軟件邏輯帶來一次顯著的進化。

iPhone 5s 的產品視頻中，Jony Ive 是這樣總結 Touch ID 的。

「We believe that technology is at its very best, at its most empowering, when it simply disappears.」

而 Face ID 顯然是同一套哲學，經過了 4 年發展之後的最新成果。其幫助 iPhone X，在「讓體驗與現實之間的邊界消失」這件事上邁出了革命性的一步。

AR

半年前的 WWDC 上，ARKit 隨着 iOS 11 問世。

我的第一反應仍然是不解，特別是當其第一個 demo 是一個蹩腳的遊戲時。

但後來的一些對 Tim Cook 的採訪，包括去東京旅行的經歷，讓我對 AR 有了更深刻的認識。大部分感想我都寫在了這篇文章裏。

當然，ARKit 在今天還處於一個很初級的階段，蘋果所做的，更多是進行概念上的選竄，以及吸引開發者，積累利用 AR 工具進行開發的經驗。真正要在產品層面帶來革命性的改變，我認爲很可能需要等到蘋果的 AR 眼鏡問世的那一天。

但即使只是停留在概念上，AR 也已經可以用來解釋很多東西了。

如果說「全面屏」是爲了「消除體驗與現實之間的邊界」，那我們必須要思考的一個問題就是，如果真的有一天，屏幕可以完完全全地佔滿整個手機之後，What’s Next？

AR 的答案是，讓內容流出屏幕、流出手機，真正地做到與現實共存。

這應當是蘋果暢想的，比 iPhone X 所開啓的未來，還要更遠更廣闊的世界。

寫在最後

夏天的時候，有蘋果前員工發 Twitter 說，早在 2014 年春天他離開蘋果時，iPhone X 就已經在籌劃之中了。

這令人意外嗎？早在 4、5 年前，Jony Ive 就闡明過這個道理：

「To create the new iPhone, we began with a design that we really loved. But to build it, to implement it, we had to look way beyond, what we knew to be possible. It took all our learning, our best thinking, to realize something so simple, so clear, and yet so truly extraordinary.」

對於蘋果來說，全面屏就是其所「熱愛的設計」。3 年之後，iPhone X 終於邁出了最重要的那一步，從虛無縹緲的理想、到最終落地成爲現實，蘋果跨越了現實和理想之間的那條線。

更重要的是，蘋果非常清楚，「全面屏」並不是終點，iPhone X 所處的位置也不是其要到達的目的地。Cook 要開啓的未來 10 年，其實才剛剛開始。

這讓我的興奮遠遠不只停留在 iPhone X 本身，更是在於其引申出的「What next?」這一未來時命題，最終將我帶向屬於未來的遠方。
