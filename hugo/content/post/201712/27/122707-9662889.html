2017 年哪些品牌請的代言人顛覆了你的認知？

— 以下內容均爲個人觀點，如有不同意見歡迎理性討論 —

-

爭議可惜組

Dior X 趙麗穎

如果說 AB 成爲迪奧中國區品牌大使還算是有黃曉明夫人、美貌以及一貫以來的好資源加持的話，趙麗穎這次合作只能用「萬萬沒想到」來形容。在 Dior 之前，趙麗穎是沒什麼好時尚資源的，也很少在傳統老牌奢侈品品牌的活動上看到她的身影（對比歐陽娜娜和 Chanel 感受下），這次可以說是一步登天了。拿下 Dior 就開了掛，捎帶腳還成了浪琴的優雅形象大使。講道理，在不談作品演技的情況下，以這位穎寶時至今日的硬照、街拍、雜誌戰績和人設，離時尚優雅似乎都還有老遠的第一段距離。受爭議也就在所難免。至於後續是否能夠就勢飛昇，明年拭目以待。

p.s. 順帶心疼王珞丹一分鐘，16 年就開始頻繁合作，幾乎每一次公開出席活動都演繹的非常好……哎。可惜。

p.p.s. 天仙辣麼美，可功可柔，才只是 Dior 花蜜系列形象大使。拿下花木蘭後，應該會有更多品牌踏破門檻吧，爲天仙打 call。

Michael Kors X 楊冪

2017 對楊冪可以說是非常好了，先後（順序可能不對）拿下 Michel Kors 全球代言人（注意是全球不是中國哦，是代言人不是手袋 / 香水 / 美妝代言人哦）、雅詩蘭黛亞太區代言人、伯爵 Possession 系列推廣大使、Adidas Originals 大中華區品牌形象代言人等至少四個大單，應該是一線小花目前的最好成績了。放在這組的原因是，MK 這個代言有些尷尬，說品牌 level ，似乎真的是不太配得上大冪冪現在的人氣和 level 了，但是從名頭上來看，卻又當真是超級重磅。兩廂平衡可能纔是最終選了這個代言的原因吧。（不是故意給冪冪大圖的，想縮小但是不得感覺是系統問題。

萬寶龍 X 楊洋

失去林丹後的萬寶龍選擇了楊洋，並且還是品牌大使的身份（不分產品線哦，腕錶皮具書寫工具全包哦），實在是讓人難以相信這是一個用了桂綸鎂多年，還找了俞飛鴻來給 campaign 站臺的品牌。不吐槽楊洋了，但願這只是品牌一時失血的絕望之舉。（對比下萬寶龍的全球品牌形象大使 - 休傑克曼，這個吐槽就不難理解了吧？）

DeBeers X 范冰冰

（這也可能是我對 DeBeers 的偏見）眼看着冰冰從肖邦到卡地亞再到 DeBeers ，無論是摯友還是代言，持續時間都很短。DeBeers 雖然發明了鑽石恆久遠，但是從目前的品牌知名度，產品火熱度，即便是「全球代言人」似乎都配不上冰冰姐。期待看到 BB 在珠寶腕錶領域找到像 LV 這樣勢均力敵的長久合作伙伴。

GUCCI X 倪妮

這一組，其實是蠻可惜倪妮。硬照素質是一方面，業務素質又是另一方面呀。好的作品，是人氣的基礎呀。亞洲區眼鏡形象大使，還是蠻可惜的。已經手握 Tiffany 中國區品牌代言人、女演員中傑出的硬照素質和鮎魚臉，這麼好的底子，怎麼可以不在一衆小花中大殺四方呢！

p.s. 想到李宇春是 Gucci 的亞洲區腕錶首飾形象大使，原來倪妮和春春還有這樣的交集~有趣~

-

穩準狠組

SKII X 竇靖童

多年來，SKII 選代言人的路子，從湯唯倪妮到霍建華，甚至是年輕的李沁，都只能說是從未失手指哪兒打哪兒。宣佈竇靖童成爲 SK-II 護膚精華露全新形象代言人後，妥妥的抓住了年輕的少男少女，也讓更年長的用戶也心服口服。傑出，厲害。

Burberry X 吳亦凡

Kris Wu 先生這一年新聞不斷，代言也沒閒着，（至少）去年已經簽了寶格麗品牌代言人和 Burberry 品牌代言人，今年在此基礎上設計了 Burberry X 吳亦凡系列，如果我是粉絲（且有錢），那還說什麼呢，當然是立刻買啊。爲品牌的思路清晰（抓住有錢的年輕消費者）和財大氣粗打 call。（坦白講，我也是看了吳亦凡穿 B 家才覺得，原來這個格子還可以這麼潮的。

說到寶格麗想再說一句舒淇（雖然這倆相知多年已經是摯友中的摯友了，從腕錶代言到品牌代言）。寶格麗跟舒淇真的是配一臉，Serpenti 系列，真的想不到哪一位一線女星還能比舒淇更 hold 得住，氣質搭，合作默契，真是好緣分。更不要說今年寶格麗酒店（北京）開幕，舒淇至少出席了兩次在該酒店的活動（對於一個在北京的，非 destination hotel 的酒店來說真的是非常大陣仗了）。希望可以延續下去，希望年年都能看到舒淇戴 Serpenti 出現在廣告牌上！

Chanel X 劉雯

穩紮穩打的大表姐和香家，可以說是鐵打的劉雯，流水的歐陽娜娜。大表姐今年先後拿下了 Chanel 中國腕錶形象大使和 Chanel 香水與美容形象大使（跟胡歌一起），對於香家這種細水長流多年才終能簽下一紙合約的品牌來說，實屬不易。且大表姐人美氣質佳，不作不用力過猛，跟 C 一直打的「法式風情」實在是太配了。希望看到大表姐 18 年繼續高歌猛進。

p.s. 不心疼白百何，雖然通稿少了，但是實際參加品牌活動和曝光次數並沒有；且，在一衆合作多次的女星中，劉詩詩、江疏影都好太多了。（桂綸鎂是臺灣地區形象大使，所以不放在這裏比

Omega X 劉詩詩

Omega 剛剛簽了劉詩詩成爲名人大使，這步算詩詩贏比較大，還在她一直以來都是優雅古典掛的，跟優雅靈動這樣的詞一起出現在新聞稿裏，也是毫不違和。說到 Omega 想感慨一句，Kaia Gerber 紅了之後，火速簽了媽媽女兒兒子仨人成爲名人大使，不僅是品牌眼光毒，也證明星二代也要有實力纔好使啊。

說回詩詩，今年還拿下了 TOD'S 的品牌大使，雖然有點意外，但真的是驚喜。都低調、有質感、有品位，選得非常正。

寶珀和他的大使們

文化大使樑文道、藝術大使馮遠征 、文化大使吳曉波 、風格大使吳秀波。再加上一位摯友李健，集齊了四大領域（最有魅力的）中年男子的寶珀，這張「文化」牌打得十分給勁兒。這幾個人配上「堅持，讓 282 年更有價值」，就特別立得住腳。

-

最後放一個私心裏愛到爆炸，但不是代言的：Prada X 杜鵑

Prada 的靈氣和媚氣，杜鵑都太有了，不敢想象這套衣服到其他人身上會變成怎樣的災難。

一個不成熟的個人總結：

所有合作說到底都是博弈。

明星要看品牌是什麼 level（法國的還是非法國的，LVMH 的還是 Kering 的），給的什麼 title 和資源（產品線還是全品牌；摯友還是大使 - 一道排列組合題）；品牌要看明星氣質調性合不合適，有沒有黑歷史，紅不紅，紅得有沒有可持續性等等。

兩廂博弈，能夠互相成全，自是最好，也最難得，大部分情況下，要麼是品牌成就了明星（Dior X 趙麗穎），要麼是明星遷就了品牌（De Beers X 冰冰）。這還沒有考慮競品（品牌間的，明星間的）排他、品牌是否有找強綁定明星的傳統和磨合期（例如 PP，Valextra）等等因素。

這就好像談戀愛，勢均力敵，總是最合襯也最難得。

以上。
